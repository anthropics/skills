## 📋 翻译方式说明

### ✅ 采用的方式：**直接添加中文翻译**（推荐）
```markdown
## Overview
## 概述

Description text in English
英文描述文本的中文翻译
```

**为什么选择这种方式：**
1. ✓ 模型能完整看到英文和中文
2. ✓ 模型能同时理解两种语言的指令
3. ✓ 中文翻译不会被忽略
4. ✓ 保持文档可读性和格式一致
5. ✓ 符合提供的模板风格

### ❌ 不采用的方式：HTML 注释方式
```markdown
## Overview
<!-- 概述 -->
```
**原因：** 模型在处理 HTML 注释时通常会忽略其中的内容，导致中文指令无法被理解和执行。
