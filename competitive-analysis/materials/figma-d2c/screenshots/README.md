# 截图与页面内图 — Figma D2C 图文证据

- **优先级**：**页面内图 > 区域截图 > 整页截图**。正文/产品区内的 `<img src>` 先提取并下载，再考虑截图（见 [research-methodology §6](../../references/research-methodology.md)）。
- **命名**：`{竞品}-{页面或功能}-{序号}.png`；页面内图可用 `-inspect-panel` 等后缀区分。
- **来源**：按 `../links-to-drill.md` 下钻；每页至少 1 张**产品关键图**（优先正文内图，避免只截首屏）。
- **登记**：在 `../sources.md` 区分「页面内图」与「截图」；整页截图须备注「首屏/整页，非正文关键图」。
