# Consulting Contract Generator

## Overview

生成AIアドバイザリー・コンサルティング業務委託契約書を自動生成するスキル。月額顧問型の継続契約を想定し、知的財産権の共有、再委託の許可、損害賠償の上限設定等、コンサルティング業務に適したバランス型の契約書をWord形式（.docx）で作成する。

**重要**: 本スキルは受託者・委託者双方にとって公平な内容を目指しています。損害賠償の上限設定（300万円）、知的財産権の共有、再委託の許可により、柔軟で実務的な契約関係を構築できます。

## 使用方法

### 基本ワークフロー

1. **情報収集**: ユーザーに以下の情報を確認
   - 契約当事者（委託者・受託者の企業名）
   - 業務内容（生成AIコンサル、DXコンサル、システム開発等）
   - 契約形態（月額顧問 or プロジェクト型 or 時間単価型）
   - 報酬額・支払条件
   - 成果物の有無と知的財産権の帰属
   - 再委託の可否
   - 損害賠償の上限額

2. **スクリプト実行**: `scripts/create_contract.js`をカスタマイズして実行

3. **出力**: `/mnt/user-data/outputs/`にWord文書を出力

### スクリプトのカスタマイズ

`scripts/create_contract.js`を編集して以下を調整：

- **企業名**: 委託者・受託者の商号を変更
- **業務内容**: 第2条の業務内容リストを具体的に修正
- **報酬額**: 第4条の月額金額を変更
- **知的財産権**: 第6条の帰属（共有 or 委託者 or 受託者）を選択
- **損害賠償上限**: 第13条の上限額を調整
- **管轄裁判所**: 第19条の管轄裁判所を適切な地域に変更

### 実行手順

```bash
# スクリプトを編集後、実行
node scripts/create_contract.js
```

## 標準的な契約構成

生成される業務委託契約書には以下の条項が含まれる：

1. **目的**: 契約の目的と背景
2. **業務内容**: 受託者が提供する業務の明確化
3. **契約期間**: 有効期間1年・自動更新、3ヶ月前通知で解約可能
4. **報酬及び支払条件**: 月額報酬制、月末締め翌月末払い
5. **成果物**: 提案書・レポート・研修資料等の納品規定
6. **知的財産権** ⭐: 成果物の著作権は共有（カスタマイズ可能）
7. **再委託** ⭐: 事前書面承諾により再委託可能
8. **機密保持**: 別途NDAに従う
9. **委託者の義務**: 情報提供・担当者選任等の協力義務
10. **受託者の義務**: 善管注意義務・誠実な業務遂行
11. **契約解除**: 義務違反・破産等の場合の即時解除
12. **損害賠償** ⭐⭐: 上限300万円、故意・重過失は除外、直接損害のみ
13. **権利義務の譲渡禁止**: 契約上の地位譲渡の制限
14. **反社会的勢力の排除**: 反社条項（企業間契約として必須）
15. **契約の変更**: 書面による合意のみ有効
16. **分離可能性**: 一部無効時の取扱い
17. **協議事項**: 未定事項の解決方法
18. **管轄裁判所**: 紛争時の裁判管轄

⭐⭐ = 受託者のリスクを適切に管理した重要条項
⭐ = コンサルティング業務の柔軟性を確保する条項

## カスタマイズオプション

詳細は`references/clauses.md`を参照。

### 業務内容のカスタマイズ

デフォルトは生成AIコンサルティング。以下に変更可能：
- システム開発コンサルティング
- DX推進コンサルティング
- 経営コンサルティング
- マーケティングコンサルティング

### 報酬体系の変更

- **月額顧問型**（デフォルト）: 月額200,000円
- **時間単価型**: 時間単価×実績稼働時間
- **成果報酬型**: 基本報酬+成果達成時の追加報酬
- **プロジェクト型**: 総額を着手金・中間金・完了時に分割

スクリプト内の第4条を編集して変更。

### 知的財産権の帰属

- **共有型**（デフォルト）: 委託者・受託者の共有
- **委託者帰属型**: 委託者に帰属、受託者は実績公表権のみ
- **受託者帰属型**: 受託者に帰属、委託者に使用許諾

スクリプト内の第6条を編集して変更。

### 損害賠償の上限額調整

プロジェクト規模に応じて調整可能：
- 小規模プロジェクト: 100万円
- 標準（デフォルト）: 300万円
- 大規模プロジェクト: 500万円〜1,000万円

月額報酬の3〜6ヶ月分程度が目安。

スクリプト内の該当箇所を編集：
```javascript
children: [new TextRun("1. 両当事者は、本契約に違反し、相手方に損害を与えた場合、当該損害を賠償する責任を負うものとする。ただし、賠償額の上限は金300万円とする。")]
```

### 再委託の扱い

- **事前承諾型**（デフォルト）: 委託者の事前書面承諾が必要
- **原則禁止型**: 特別な理由がない限り禁止
- **一部自由型**: 定型業務は通知のみで可、コア業務は承諾必要

## レイアウトの調整

署名欄がページをまたぐ場合：
- 署名欄の直前に`pageBreakBefore: true`を設定
- これにより署名欄全体が新しいページから始まる

## 技術仕様

- **ライブラリ**: docx.js（Node.js）
- **フォント**: MS Mincho（日本語契約書の標準）
- **出力形式**: .docx（Microsoft Word形式）
- **ページ設定**: A4サイズ、余白各25.4mm（1インチ）

## 業種別の応用

本スキルは生成AIコンサルティング以外にも応用可能：
- ITコンサルティング
- DX推進コンサルティング
- 経営コンサルティング
- マーケティングコンサルティング
- 人材育成・研修サービス

業務内容（第2条）を編集することで幅広い業種に対応。
