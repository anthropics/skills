# 生成AIアドバイザリー業務委託契約書 - 条項リファレンス

## 標準条項（バランス型）

本スキルが生成する業務委託契約書は、コンサルティング業務に適したバランス型の内容です。

### 第1条 (目的)
契約の目的を明記。生成AI活用に関するアドバイザリー及びコンサルティング業務。

### 第2条 (業務内容)
受託者が提供する業務を明確化。
- 生成AI活用戦略の策定支援
- 生成AIツールの選定・導入支援
- 従業員向け生成AI研修・教育の実施
- プロンプトエンジニアリング指導
- AI活用の効果測定及びレポーティング
- その他、委託者が必要とする生成AI関連のアドバイザリー業務

### 第3条 (契約期間)
- 有効期間: 1年間（自動更新）
- 解約通知: 3ヶ月前の書面通知で可能
- 継続的な顧問契約に適した設計

### 第4条 (報酬及び支払条件)
- 月額報酬制（デフォルト: 200,000円）
- 支払サイト: 月末締め翌月末払い
- 振込手数料は委託者負担

### 第5条 (成果物)
- 提案書、レポート、研修資料等の納品
- 納品方法・納期は別途協議

### 第6条 (知的財産権) ⭐重要ポイント
**知的財産権の共有:**
- 成果物の著作権は委託者・受託者の共有
- 行使については別途協議
- 受託者の既存知的財産は受託者に帰属

**メリット:**
- 受託者: 成果物を自社のノウハウとして蓄積可能
- 委託者: 成果物を自由に利用可能
- 双方の利益をバランス良く保護

### 第7条 (再委託) ⭐柔軟性のポイント
- 委託者の事前書面承諾により再委託可能
- 再委託先の行為については受託者が責任を負う
- 外部専門家との協業が可能

### 第8条 (機密保持)
- 別途締結する機密保持契約（NDA）に従う
- 機密情報の取扱いは専用契約で詳細管理

### 第9条 (委託者の義務)
- 必要な情報提供
- 担当者の選任
- 円滑な業務遂行への協力

### 第10条 (受託者の義務)
- 善良なる管理者の注意義務
- 誠実な業務遂行

### 第11条 (契約解除)
以下の場合に即時解除可能:
- 契約義務違反（催告後も是正しない場合）
- 支払停止・破産手続開始等
- 差押・仮差押等
- その他重大事由

### 第12条 (損害賠償) ⭐⭐リスク管理ポイント
**改善内容:**
1. 賠償額の上限: 300万円（デフォルト）
2. 故意・重過失の場合は上限なし
3. 直接損害のみ対象（逸失利益・間接損害は除外）

**効果:**
- 受託者のリスクを予測可能に
- 通常のミスと故意・重過失を区別
- コンサルティング業務に適した責任範囲

### 第13条 (権利義務の譲渡禁止)
契約上の地位を第三者に譲渡する際は事前承諾が必要。

### 第14条 (反社会的勢力の排除)
- 反社会的勢力との関係を排除
- 該当した場合は即時解除
- 企業間契約として必須の条項

### 第15条 (契約の変更)
書面による合意のみ有効。

### 第16条 (分離可能性)
一部条項が無効でも他の条項は有効。

### 第17条 (協議事項)
未定事項や解釈の疑義は協議で解決。

### 第18条 (管轄裁判所)
紛争時の専属管轄裁判所を指定。

```
(1) 生成AI活用戦略の策定支援
(2) 生成AIツールの選定・導入支援
(3) 従業員向け生成AI研修・教育の実施
(4) プロンプトエンジニアリング指導
(5) AI活用の効果測定及びレポーティング
```

```
(3) プロジェクトマネジメント支援

```

```
(1) デジタル化戦略の策定

(4) 従業員のデジタルスキル向上支援
```

## 報酬体系のカスタマイズ

### 月額顧問型（デフォルト）
```
月額金200,000円(消費税別)
```
**適用場面:** 継続的なアドバイザリー、定期的な相談対応

### 時間単価型
```
時間単価: 金10,000円(消費税別)
月間稼働時間の実績に応じて請求
```
**適用場面:** 稼働時間が月によって変動する場合

### 成果報酬型
```
基本報酬: 月額金100,000円(消費税別)
成果報酬: 目標達成時に別途協議した金額
```
**適用場面:** 明確な成果指標がある場合

### プロジェクト型
```
総額: 金2,000,000円(消費税別)
支払: 着手金30%、中間金40%、完了時30%
```
**適用場面:** 期間限定のプロジェクト

## 知的財産権のカスタマイズ

### 共有型（デフォルト）
```
著作権その他の知的財産権は、委託者及び受託者の共有とする。
```
**メリット:** 双方が成果物を活用可能

### 委託者帰属型
```
著作権その他の知的財産権は、委託者に帰属する。
ただし、受託者は自社の実績として公表する権利を有する。
```
**メリット:** 委託者が成果物を独占的に利用

### 受託者帰属・使用許諾型
```
著作権その他の知的財産権は、受託者に帰属する。
受託者は、委託者に対し、無償で非独占的な使用権を許諾する。
```
**メリット:** 受託者がノウハウを保護しつつ委託者も利用可能

## 損害賠償の上限額調整

プロジェクト規模・リスクに応じて調整:
- 小規模: 100万円
- 中規模: 300万円（デフォルト）
- 大規模: 500万円〜1,000万円

月額報酬の3〜6ヶ月分程度が目安。

## 再委託の扱い

### 事前承諾型（デフォルト）
```
委託者の事前の書面による承諾を得た場合に限り再委託可能
```

### 原則禁止型
```
受託者は、委託者の事前の書面による承諾を得た場合を除き、
本契約に基づく業務を第三者に再委託してはならない。
```

### 一部業務は自由型
```
受託者は、定型的な作業については委託者への通知により
第三者に再委託することができる。
ただし、コンサルティング業務の中核部分については
委託者の事前承諾が必要とする。
```

## 契約期間の調整

### 継続型（デフォルト）
- 1年間（自動更新）
- 3ヶ月前通知で解約可能

### プロジェクト型
- 特定期間（例: 6ヶ月間）
- 自動更新なし

### 短期型
- 3ヶ月間（自動更新）
- 1ヶ月前通知で解約可能

## 業種別の適用例

本契約書は生成AIコンサルティング以外にも、以下の業務に応用可能:
- ITコンサルティング
- DX推進コンサルティング
- 経営コンサルティング
- マーケティングコンサルティング
- 人材育成・研修サービス
