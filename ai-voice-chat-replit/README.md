# AI Voice Chat - Replit Version

Simple, private AI voice chat optimized for iPhone. Confident, honest AI personality.

## Deploy to Replit (2 Minutes)

### Step 1: Create New Repl
1. Go to [replit.com](https://replit.com)
2. Click **"+ Create Repl"**
3. Choose **"Import from GitHub"**
4. Or click **"Python"** template

### Step 2: Upload Files
If you didn't import from GitHub:
1. Delete any default files
2. Click **"+ New file"**
3. Upload these files:
   - `main.py`
   - `index.html`
   - `requirements.txt`
   - `.replit`

### Step 3: Run It
1. Click the big green **"Run"** button at the top
2. Wait for it to install dependencies (first time only)
3. A web preview will open

### Step 4: Get Your Link
1. Look for the URL at the top of the preview window
2. It will be something like: `https://your-repl-name.your-username.repl.co`
3. Open that link on your iPhone

### Step 5: Use It
1. Enter your Anthropic API key (one time only)
2. Start talking!

- **Blue mic** = Ready to listen
- **Red pulsing** = Listening
- **Green** = AI speaking
- Tap mic while speaking to stop

## Get API Key

1. Go to [console.anthropic.com](https://console.anthropic.com/)
2. Sign up (free tier available)
3. Create API key
4. Copy and paste into the app

## Troubleshooting

**"Module not found" error?**
- Make sure `requirements.txt` is uploaded
- Click "Run" again

**Can't access from iPhone?**
- Use the `.repl.co` URL, not localhost
- Make sure Repl is running (green button should say "Stop")

**API key errors?**
- Double-check the key is correct
- Verify you have credits in Anthropic account

## Privacy

- API key stored in browser only
- Backend doesn't log or store anything
- Conversation history in memory only
- Refresh to clear history
