# Parallel Skill Generator

## 概要
Claude Codeの並列処理機能を活用して、CSVファイルから複数のAgent Skillsを高速で一括生成するスキルです。

## 特徴
- **高速処理**: 従来の5-10倍の速度でスキルを生成
- **並列実行**: 最大10個のスキルを同時作成
- **バッチ処理**: 大量のスキルも効率的に処理
- **エラー耐性**: 個別の失敗が全体に影響しない設計

## ファイル構成
```
parallel-skill-generator/
├── SKILL.md              # スキルの詳細定義
├── README.md            # このファイル

├── parallel-workflow.md # 並列処理ワークフローの説明
└── examples/            # 実行例（今後追加予定）
```

## 使用方法

### 1. 基本的な使用
```
「professional_skill_requests.csvを使って並列でスキルを作成してください」
```

### 2. 優先度フィルタリング
```
「優先度が高のスキルだけを並列で作成してください」
```

### 3. 職業別フィルタリング
```
「行政書士向けのスキルを並列で作成してください」
```

## CSVフォーマット
```csv
記入者名,部門/チーム,職業/役職,ペインポイント,解決したいタスク,希望するスキル名,出力形式,優先度,備考
```

必須項目:
- 記入者名
- 職業/役職
- ペインポイント
- 解決したいタスク

## 実行例

### 例1: 9個のスキルを並列作成
```
入力: professional_skill_requests.csv (9行のスキルリクエスト)

処理フロー:
1. CSV読み込み
2. 9個のディレクトリを同時作成（1バッチで処理）
3. 9個のSKILL.mdを同時生成
4. 検証とレポート出力

所要時間: 約30-45秒（従来: 3-5分）
```

### 例2: 30個のスキルをバッチ処理
```
入力: large_skill_requests.csv (30行のスキルリクエスト)

処理フロー:
1. CSV読み込み
2. 3バッチに分割（10個×3）
3. 各バッチを並列処理
4. 全体の検証とレポート

所要時間: 約1.5-2分（従来: 15-20分）
```
