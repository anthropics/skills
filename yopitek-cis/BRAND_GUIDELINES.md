# Yopitek Corporate Identity System (CIS)

## Company Overview

**Yopitek** is a technology company specializing in:
- Wireless Routers
- Graphic Cards
- NFC Card Readers

Our brand represents innovation, connectivity, and cutting-edge technology solutions.

---

## 1. Brand Colors

### Primary Colors

**Tech Blue** - Main brand color representing connectivity and innovation
- HEX: `#0066CC`
- RGB: `0, 102, 204`
- CMYK: `100, 50, 0, 20`
- Usage: Primary brand elements, headers, CTAs

**Deep Navy** - Professional and stable
- HEX: `#1A2332`
- RGB: `26, 35, 50`
- CMYK: `100, 75, 50, 70`
- Usage: Text, backgrounds, professional contexts

**Pure White**
- HEX: `#FFFFFF`
- RGB: `255, 255, 255`
- CMYK: `0, 0, 0, 0`
- Usage: Backgrounds, negative space

### Secondary Colors

**Electric Cyan** - Digital innovation
- HEX: `#00D4FF`
- RGB: `0, 212, 255`
- CMYK: `70, 0, 0, 0`
- Usage: Accents, highlights, digital interfaces

**Signal Green** - Connectivity and status
- HEX: `#00E676`
- RGB: `0, 230, 118`
- CMYK: `70, 0, 70, 0`
- Usage: Active states, success indicators

**Circuit Orange** - Energy and performance
- HEX: `#FF6B35`
- RGB: `255, 107, 53`
- CMYK: `0, 70, 80, 0`
- Usage: Highlights, performance indicators

### Neutral Colors

**Cool Gray 90**
- HEX: `#2D3142`
- RGB: `45, 49, 66`
- Usage: Secondary text

**Cool Gray 50**
- HEX: `#9DA3B4`
- RGB: `157, 163, 180`
- Usage: Disabled states, borders

**Cool Gray 10**
- HEX: `#F5F6F8`
- RGB: `245, 246, 248`
- Usage: Subtle backgrounds

## 2. Typography

### Primary Typeface: **Inter**

**Headings & Titles**
- Font: Inter Bold / Semi-Bold
- Weights: 600, 700
- Usage: Headlines, product names, section headers
- Fallback: Arial, Helvetica, sans-serif

**Body Text**
- Font: Inter Regular / Medium
- Weights: 400, 500
- Usage: Paragraphs, descriptions, UI text
- Fallback: Arial, Helvetica, sans-serif

### Secondary Typeface: **JetBrains Mono**

**Technical Content**
- Font: JetBrains Mono
- Weight: 400, 500
- Usage: Specifications, model numbers, technical data
- Fallback: Courier New, monospace

### Typography Scale

- H1: 48pt / 3rem (Inter Bold)
- H2: 36pt / 2.25rem (Inter Semi-Bold)
- H3: 28pt / 1.75rem (Inter Semi-Bold)
- H4: 24pt / 1.5rem (Inter Medium)
- Body: 16pt / 1rem (Inter Regular)
- Small: 14pt / 0.875rem (Inter Regular)
- Caption: 12pt / 0.75rem (Inter Regular)

### Typography Guidelines

- Line height: 1.5 for body text, 1.2 for headings
- Letter spacing: -0.02em for large headings
- Paragraph spacing: 1em between paragraphs
- Maximum line length: 65-75 characters

## 3. Logo System

### Logo Concept

**Yopitek** logo combines:
- Modern, geometric wordmark
- Circuit-inspired icon representing connectivity
- Clean, tech-forward aesthetic

### Logo Variations

**Primary Logo**
- Full color on white background
- Full color on dark background (inverted)

**Monochrome Versions**
- Black on white
- White on dark

**Icon Only**
- For small applications (app icons, favicons)
- Minimum size: 32x32px

### Clear Space

- Maintain clear space around logo equal to the height of the letter "Y"
- No other elements should enter this protected area

### Minimum Sizes

- Full logo: 120px width (digital), 30mm (print)
- Icon only: 32px (digital), 8mm (print)

### Incorrect Usage

❌ Do not stretch or distort
❌ Do not rotate
❌ Do not use unapproved colors
❌ Do not add effects (shadows, gradients, outlines)
❌ Do not place on busy backgrounds without proper contrast

---

## 4. Visual Identity Elements

### Grid System

- 12-column grid for layouts
- 8px base unit for spacing
- Spacing scale: 8, 16, 24, 32, 48, 64, 96px

### Border Radius

- Small elements: 4px (buttons, inputs)
- Cards and panels: 8px
- Large containers: 12px
- Circular: 50% (profile images, icons)

### Shadows

**Elevation 1** (Subtle)
```
box-shadow: 0 1px 3px rgba(26, 35, 50, 0.12)
```

**Elevation 2** (Medium)
```
box-shadow: 0 4px 12px rgba(26, 35, 50, 0.15)
```

**Elevation 3** (Prominent)
```
box-shadow: 0 8px 24px rgba(26, 35, 50, 0.18)
```

### Iconography

**Style**: Line icons with 2px stroke
**Set**: Use Material Design Icons or Feather Icons
**Color**: Inherit from parent or use Tech Blue
**Size**: 16px, 24px, 32px, 48px

## 6. Photography Style

### General Guidelines

- Clean, modern, high-tech aesthetic
- Minimal backgrounds (white or subtle gradients)
- Dramatic lighting with tech-blue tints
- Focus on product details and craftsmanship

### Composition

- Use rule of thirds
- Maintain negative space
- Show products in use when appropriate
- Consistent lighting and color temperature

### Color Grading

- Cool tones (blue-shifted)
- High contrast
- Crisp, sharp focus
- Subtle blue/cyan color overlay (10-15% opacity)

## 8. Digital Guidelines

### Website

**Header:**
- White background
- Tech Blue logo
- Deep Navy navigation text
- Electric Cyan hover states

**Hero Section:**
- Large product imagery
- Tech Blue or gradient background
- White text with clear CTAs

**Content Areas:**
- White backgrounds
- Cool Gray 10 for alternate sections
- Tech Blue for links and interactive elements

**Footer:**
- Deep Navy background
- White text
- Organized in grid layout

### UI Components

**Buttons**

*Primary:*
```
Background: #0066CC (Tech Blue)
Text: #FFFFFF
Border-radius: 4px
Padding: 12px 24px
Font: Inter Medium, 16px
```

*Secondary:*
```
Background: Transparent
Border: 2px solid #0066CC
Text: #0066CC
Border-radius: 4px
Padding: 12px 24px
```

*Hover State:*
- Darken background by 10%
- Add subtle shadow (Elevation 1)

**Form Inputs**
```
Border: 1px solid #9DA3B4
Border-radius: 4px
Padding: 12px 16px
Font: Inter Regular, 16px
Focus: Border color #0066CC, add shadow
```

**Cards**
```
Background: #FFFFFF
Border: 1px solid #F5F6F8
Border-radius: 8px
Shadow: Elevation 1
Padding: 24px
```

### Social Media

**Profile Images:**
- Use icon-only logo version
- Consistent across all platforms

**Cover Images:**
- 1200x630px recommended
- Feature products with Tech Blue background
- Include tagline

**Post Templates:**
- Square (1080x1080px) for Instagram
- Landscape (1200x628px) for Facebook/LinkedIn
- Maintain brand colors and typography
- Include small logo in corner

## 10. Brand Voice & Messaging

### Tone of Voice

**Professional yet approachable**
- Clear and concise
- Technical but not jargon-heavy
- Confident and innovative
- Helpful and customer-focused

### Key Messages

**Tagline Options:**
- "Connecting Innovation"
- "Performance Without Limits"
- "Technology That Connects"

**Value Propositions:**
- Cutting-edge technology
- Reliable performance
- User-friendly solutions
- Innovation-driven design

### Messaging Guidelines

**Do:**
- Focus on benefits, not just features
- Use active voice
- Be specific with technical claims
- Emphasize quality and reliability

**Don't:**
- Use excessive jargon
- Make unsubstantiated claims
- Use informal slang
- Overcomplicate simple concepts
