# Yopitek CIS Quick Reference Guide

## Brand Essentials At-A-Glance

### Primary Brand Color
**Tech Blue:** `#0066CC`

### Primary Typography
**Headings:** Inter Bold/Semi-Bold
**Body:** Inter Regular/Medium
**Technical:** JetBrains Mono

### Logo Usage
- Minimum size: 120px (digital) / 30mm (print)
- Clear space: Equal to height of letter "Y"
- Never stretch, rotate, or add effects

## Common Use Cases

### Email Signature

### Social Media Profile
- Use icon-only logo version
- Profile image: Square format
- Cover image: 1200x630px with Tech Blue background

### Presentation Title Slide
- Background: Tech Blue (#0066CC)
- Title: Inter Bold, 48pt, White
- Subtitle: Inter Regular, 24pt, White
- Logo: White version, top left

### Button Styles

**Primary Button:**
- Background: #0066CC
- Text: #FFFFFF, Inter Medium, 16px
- Border-radius: 4px
- Padding: 12px 24px

**Secondary Button:**
- Background: Transparent
- Border: 2px solid #0066CC
- Text: #0066CC, Inter Medium, 16px

## Typography Scale

- H1: 48pt (Inter Bold)
- H2: 36pt (Inter Semi-Bold)
- H3: 28pt (Inter Semi-Bold)
- Body: 16pt (Inter Regular)
- Small: 14pt (Inter Regular)

## Common Mistakes to Avoid

❌ Using unapproved colors
❌ Stretching or distorting the logo
❌ Using logo on busy backgrounds
❌ Mixing incompatible fonts
❌ Insufficient contrast for text
❌ Ignoring clear space requirements

✓ Use approved color palette
✓ Maintain logo proportions
✓ Ensure clean backgrounds
✓ Stick to Inter and JetBrains Mono
✓ Check WCAG contrast ratios
✓ Respect clear space guidelines
