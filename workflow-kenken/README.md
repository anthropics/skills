# 株式会社けんけん 業務ワークフロー

## 概要

株式会社けんけんの業務プロセスを自動化するワークフローシステムです。契約フェーズから実行フェーズまでの一連の業務を、マスタートリガーで制御します。

## 特徴

- 🚀 **マスタートリガー**: 「起きろ」「月締め」「フル実行」の3つのトリガーで制御
- 🔄 **パラレル実行**: 複数企業の処理を同時実行（最大20社）
- 🎨 **統一デザイン**: 全文書でJony Iveデザイン哲学を適用
- 💰 **補助金連携**: Jグランツと連携して補助金情報を自動取得

## クイックスタート

### 単一企業の場合

```bash
# 契約フェーズのみ
python master_trigger.py 起きろ

# 実行フェーズのみ
python master_trigger.py 月締め

# 全フェーズ
python master_trigger.py フル実行
```

### 複数企業の場合（パラレル実行）

```bash
# 3社分の契約フェーズを並列実行
python master_trigger.py 起きろ×3

# 5社分の実行フェーズを並列実行
python master_trigger.py 月締め×5

# 2社分の全フェーズを並列実行
python master_trigger.py フル実行×2
```

## ワークフロー構成

### 契約フェーズ（起きろ）
1. **補助金確認** - Jグランツで利用可能な補助金を検索
2. **見積書作成** - 補助金情報を反映した見積書（Excel/PDF）
3. **NDA作成** - 機密保持契約書（Word）
4. **業務委託契約書作成** - コンサルティング契約書（Word）
5. **WBS/ガントチャート作成** - プロジェクト計画（Excel）

### 実行フェーズ（月締め）
1. **日報集計** - 月間の作業内容を集計
2. **月次レポート作成** - 活動報告書（Word）
3. **請求書作成** - インボイス対応請求書（Excel/PDF）

## 必要なスキル

以下のスキルがインストールされている必要があります：

- billing-documents（請求書・見積書）
- nda-generator（NDA作成）
- consulting-contract-generator（業務委託契約書）
- gantt-chart-generator（ガントチャート）
- daily-report-voice（日報）
- monthly-report-generator（月次レポート）
- jony-ive-design-system（デザインシステム）

| フェーズ | 最大並列数 |
|---------|----------|

| 実行フェーズ | 20社 |
| フル実行 | 5社 |

## 注意事項

1. **補助金情報**: 参考情報であり、採択を保証するものではありません
2. **契約書類**: 法務専門家のレビューを推奨します
3. **データ保護**: 機密情報は適切に管理してください

## トラブルシューティング

### よくある質問

**Q: パラレル実行時にエラーが出る**
A: 同時実行数を減らしてみてください。システムリソースの制限がある場合があります。

**Q: 補助金情報が取得できない**
A: インターネット接続を確認し、Jグランツのアクセス権限を確認してください。

**Q: ファイルが生成されない**
A: outputsフォルダの書き込み権限を確認してください。
