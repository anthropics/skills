# 株式会社けんけん 業務ワークフロー v3.3

IT/DXコンサルティング企業「株式会社けんけん」の業務プロセスを自動化する統合ワークフロースキルです。

## マスタートリガー

### 基本トリガー
- **「起きろ」** - 契約フェーズ全体を実行（補助金確認→見積書→NDA→契約書→WBS）
- **「月締め」** - 実行フェーズ全体を実行（日報集計→月次レポート→請求書）
- **「フル実行」** - 契約フェーズ＋実行フェーズを連続実行

### パラレル実行トリガー
- **「起きろ×3」** - 3社分の契約フェーズを並列実行
- **「月締め×5」** - 5社分の実行フェーズを並列実行
- **「フル実行×2」** - 2社分の全フェーズを並列実行

## 実行フロー

### 契約フェーズ（STEP 0-4）
```
0. 補助金確認（Jグランツ検索）
   ↓
1. 見積書作成（補助金情報反映、Excel/PDF）
   ↓
2. NDA作成（機密保持契約書、Word）
   ↓
3. 業務委託契約書作成（Word）
   ↓
4. WBS/ガントチャート作成（Excel）
```

### 実行フェーズ（STEP 5-7）
```
5. 日報データ確認・集計
   ↓
6. 月次レポート作成（Word）
   ↓
7. 請求書作成（Excel/PDF）
```

## 使用方法

### 単一企業の場合

1. **契約フェーズ開始**
   ```
   起きろ
   ```
   必要情報を入力：
   - クライアント名
   - 業務内容
   - 月額金額
   - 契約期間
   - 補助金検索用情報（業種、従業員数、地域）

2. **月次処理**
   ```
   月締め
   ```
   必要情報を入力：
   - クライアント名
   - 対象月
   - 請求金額
   - 支払期限

### 複数企業の場合（パラレル実行）

1. **3社分の契約フェーズ**
   ```
   起きろ×3
   ```
   3社分の情報を一括入力

2. **5社分の月次処理**
   ```
   月締め×5
   ```
   5社分の情報を一括入力

## デザインガイドライン

全ての生成文書は**Jony Iveデザイン哲学**に準拠：
- フォント：メイリオ統一
- カラー：グレースケール基調 + アクセントカラー（#5B7B94）
- 原則：シンプル、余白重視、完璧な整列

## 出力ファイル

### ファイル命名規則
```
{文書タイプ}_{クライアント名}_{日付}.{拡張子}

例：
- quotation_大吉_20251201.xlsx
- nda_大吉_20251201.docx
- invoice_大吉_202512.pdf
```

outputs/

## パラレル実行の利点

1. **時間短縮**: 5社個別処理（25ステップ）→ パラレル（5ステップ）
2. **一括管理**: 複数企業の進捗を同時確認
3. **効率化**: 補助金検索など共通処理を1回で実行
4. **統一性**: 同じフォーマット・デザインで一括作成

## 補助金確認の詳細

### 検索パラメータ
- keyword: IT、DX、生成AI、デジタル化
- target_area_search: 香川県、四国地方、全国
- industry: サービス業、製造業、小売業など
- target_number_of_employees: 5名以下、20名以下など

### 条件緩和ロジック
1. 全条件で検索
2. 地域を「全国」に変更
3. 業種条件を除外
4. キーワードを汎用語に変更
5. Web検索で補完

## 注意事項

1. **スキル境界**: 各スキルは定義範囲内でのみ動作
2. **デザイン一貫性**: 全文書でJony Iveデザインを適用
3. **補助金免責**: 採択を保証するものではない旨を明記
4. **法的免責**: 契約書は参考文書、法務レビュー推奨
5. **出典明記**: 補助金情報にはJグランツ出典を記載

## エラー処理

- スキルが見つからない場合：該当スキルのインストールを促す
- 並列数超過：自動的に上限値に調整
- ネットワークエラー：リトライまたはオフライン処理
