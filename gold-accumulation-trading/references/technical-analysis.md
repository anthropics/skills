## 均线系统(MA)

### 黄金常用参数
- **短期**: MA5(5日均线)、MA10(10日均线)
- **中期**: MA20(20日均线)、MA60(60日均线)
- **长期**: MA120(半年线)、MA250(年线)
- **小时线**: 适用MA5、MA10、MA20

### 压力/支撑判断
- **多头排列**(MA5 > MA10 > MA20): 趋势向上，均线为支撑
- **空头排列**(MA5 < MA10 < MA20): 趋势向下，均线为压力
- **交叉信号**: 金叉(短上穿长)看多，死叉(短下穿长)看空
- **均线粘合**: 变盘信号，关注突破方向

### 在压力/支撑位中的作用
- 价格回踩MA20不破 → 确认为有效支撑
- 价格反弹触及MA60受阻 → 确认为有效压力
- 多根均线汇聚区域 → 关键多空分界

## 布林带(BOLL)

### 参数设置
- 周期: 20，标准差倍数: 2
- 小时线/日线均可使用

### 三轨含义
- **上轨(UP)**: 动态压力位
- **中轨(MID)**: 趋势方向参考（即MA20）
- **下轨(DN)**: 动态支撑位

### 交易信号
- 价格触及上轨 → 短期超买，可能回调（压力）
- 价格触及下轨 → 短期超卖，可能反弹（支撑）
- **布林带收口** → 变盘前兆，关注突破方向
- **布林带开口** → 趋势确认，顺势操作
- 价格沿上轨运行 → 强势上涨，上轨变为支撑参考
- 价格沿下轨运行 → 强势下跌，下轨变为压力参考

## MACD

### 参数设置
- 快线EMA: 12，慢线EMA: 26，信号线: 9
- 适用: 日线、4小时线、1小时线

### 信号判断
| 信号 | 含义 | 点位 |
|------|------|------|
| DIF上穿DEA(金叉) | 多头信号 | 交叉时的价位为支撑参考 |
| DIF下穿DEA(死叉) | 空头信号 | 交叉时的价位为压力参考 |
| 零轴上方金叉 | 强势做多 | - |
| 零轴下方死叉 | 强势做空 | - |
| 顶背离(价格新高MACD未新高) | 见顶风险 | 前高为压力位 |
| 底背离(价格新低MACD未新低) | 见底信号 | 前低为支撑位 |

### 柱状图(BAR)
- 红柱缩短 → 多头动能减弱
- 绿柱缩短 → 空头动能减弱
- 柱体翻转 → 多空转换关键点

## KDJ随机指标

### 参数设置
- 周期: 9，信号: 3，平滑: 3
- 适用: 短线交易(1小时线/4小时线)

### 区域划分
| K值区间 | 状态 | 操作参考 |
|---------|------|----------|
| K > 80 | 超买区 | 注意压力位减仓 |
| K < 20 | 超卖区 | 注意支撑位入场 |
| 20 < K < 80 | 中性区 | 看趋势操作 |

### 交叉信号
- K线上穿D线(金叉) + 超卖区 → 较强买入信号
- K线下穿D线(死叉) + 超买区 → 较强卖出信号
- J线钝化(极端高位/低位停留) → 趋势可能延续

## RSI相对强弱

### 参数设置
- 周期: 14(默认)，辅助: RSI(6)和RSI(24)
- 适用: 日线、4小时线

### 区域判断
| RSI区间 | 状态 | 信号 |
|---------|------|------|
| RSI > 70 | 超买 | 上方压力增大，注意回调 |
| RSI < 30 | 超卖 | 下方支撑显现，注意反弹 |
| RSI 40-60 | 中性 | 方向不明，等待突破 |
| RSI > 80 | 极度超买 | 强压力信号 |
| RSI < 20 | 极度超卖 | 强支撑信号 |

### 背离信号
- 价格创新高但RSI未创新高 → 顶背离(压力确认)
- 价格创新低但RSI未创新低 → 底背离(支撑确认)

## 多指标共振判断

**核心原则**: 单一指标信号可靠性有限，多指标共振才是有效的压力/支撑确认。

### 共振强度评级

**强共振(高置信度) — 3个及以上指标同向确认**:
```
示例: MA20支撑 + BOLL下轨支撑 + RSI超卖 + KDJ金叉
→ 该价位为强支撑，可作为S1
```

**中等共振(中等置信度) — 2个指标同向确认**:
```
示例: MA60压力 + MACD顶背离
→ 该价位为有效压力，可作为R1或R2
```

**弱共振(低置信度) — 仅1个指标信号**:
```
示例: 仅RSI超买
→ 参考价值有限，需等待更多确认
```

### 确定R1/R2/R3和S1/S2/S3

1. 列出所有通过技术指标识别的压力位/支撑位
2. 按共振强度排序
3. 选取最关键的3个压力位和3个支撑位
4. 按价格从低到高排列:
   - S3(最远支撑) < S2 < S1(最近支撑) < 当前价 < R1(最近压力) < R2 < R3(最远压力)
5. 所有点位要精简实用，只给关键有效位置

## 轴心位算法对比

### 1. 自定义轴心位（本Skill默认使用）
```
P = (R1 + S1) ÷ 2
```
- R1: 第一压力位(通过技术分析确定)
- S1: 第一支撑位(通过技术分析确定)
- **优点**: 直接基于当下市场结构，逻辑清晰、易懂好用

**举例**:
- R1 = 3370 (XAUUSD)
- S1 = 3350 (XAUUSD)
- P = (3370 + 3350) ÷ 2 = 3360

### 2. 经典轴心位(Classic Pivot Point)
```
P = (前日最高 + 前日最低 + 前日收盘) ÷ 3
R1 = 2P − 前日最低
S1 = 2P − 前日最高
R2 = P + (前日最高 − 前日最低)
S2 = P − (前日最高 − 前日最低)
R3 = 前日最高 + 2 × (P − 前日最低)
S3 = 前日最低 − 2 × (前日最高 − P)
```

### 3. Woodie轴心位
```
P = (前日最高 + 前日最低 + 2 × 前日收盘) ÷ 4
```
- 更看重收盘价，对趋势延续性更敏感

### 4. Camarilla轴心位
```
P = (前日最高 + 前日最低 + 前日收盘) ÷ 3
```
- 适合日内交易，阻力/支撑分级更精细

### 配合使用建议
- 以自定义轴心位为主，经典轴心位为辅
- 配合压力位/支撑位一起看，多空强弱更清晰
- 金价在轴心上方 = 短线看多；金价在轴心下方 = 短线看空

## 订单流与资金分析

### 关注要点
- **COMEX黄金期货持仓报告**: 商业/非商业持仓变化
- **ETF持仓**(SPDR Gold Trust GLD): 增持=看多，减持=看空
- **上海金交所**: 交易量、持仓量变化
- **央行购金数据**: 长期趋势参考

### 资金博弈信号
- 大量买单堆积在某价位 → 该价位为支撑
- 大量卖单堆积在某价位 → 该价位为压力
- 成交量突然放大 + 价格突破 → 有效突破信号
- 成交量萎缩 + 价格到达关键位 → 假突破风险
