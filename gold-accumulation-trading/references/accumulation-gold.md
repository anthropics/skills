# 中国银行积存金产品知识库

积存金是中国各商业银行推出的黄金投资产品，以人民币计价、按克交易，与国际现货黄金存在换算关系。

## 核心换算公式

### 国际金价 → 理论人民币金价

```
理论人民币金价(元/克) = XAUUSD(美元/盎司) × USD/CNY汇率 ÷ 31.1035

其中:
  1金衡盎司 = 31.1035克
  XAUUSD = 伦敦现货黄金美元报价
  USD/CNY = 美元兑人民币即期汇率
  理论换算比 = 汇率 ÷ 31.1035
```

> **注**: 汇率实时变动，每次换算需使用最新汇率。用 `web_search` 搜索 "USD CNY exchange rate" 获取。

## 换算比与溢价概念

### 什么是换算比

换算比 = 积存金实际报价 ÷ XAUUSD伦敦金报价

它是一个综合系数，内含了**汇率换算 + 国内溢价/折价**两个因素。

### 三种换算比的关系

```
理论换算比 = 汇率 ÷ 31.1035
                  （纯数学换算，不含溢价）

实际换算比 = 银行积存金报价 ÷ XAUUSD
                  （包含了溢价/折价）

溢价 = 银行实际报价 − 理论换算价
       = (实际换算比 − 理论换算比) × XAUUSD
```

### 溢价说明

国内金价（上海金交所/银行积存金）与国际金价理论换算值之间存在**浮动溢价**：
- **正溢价**: 国内金价高于国际换算价 → 国内买盘强于国际
- **负溢价(折价)**: 国内金价低于国际换算价 → 国内供给充裕或需求偏弱
- 溢价通常在 **-15 ~ +15 元/克** 之间波动
- 溢价非固定值，随市场供需实时变化

### 实例（2026年2月28日数据）

```
XAUUSD = 5279.95 美元/盎司
USD/CNY汇率 = 6.8602

理论换算比 = 6.8602 ÷ 31.1035 = 0.22058
理论换算价 = 5279.95 × 0.22058 = 1164.6 元/克

浙商黄金实际报价 = 1154.91 元/克
浙商换算比 = 1154.91 ÷ 5279.95 = 0.21874
浙商溢价 = 1154.91 − 1164.6 = −9.7 元/克（负溢价/折价）

民生黄金实际报价 = 1156.28 元/克
民生换算比 = 1156.28 ÷ 5279.95 = 0.21899
民生溢价 = 1156.28 − 1164.6 = −8.3 元/克（负溢价/折价）
```

## 换算实操步骤

在分析报告中输出积存金点位时，按以下顺序操作:

### 步骤1: 获取实时数据（4组搜索）

```
搜索A — XAUUSD:
  web_search: "XAUUSD spot price today"
  获取: 当前报价(美元/盎司)

搜索B — 汇率:
  web_search: "USD CNY exchange rate today"
  获取: 美元兑人民币即期汇率

搜索C — 积存金/国内金价（关键！）:
  优先: "浙商积存金 民生积存金 今日金价" 或 "积存金实时报价"
  备选: "上海黄金交易所 Au99.99 最新价"
  兜底: 访问 sge.com.cn 延时行情页面

搜索D — 黄金TD（辅助验证）:
  web_search: "黄金TD Au(T+D) 实时价格"
```

### 步骤2: 计算理论换算比
```
理论换算比 = 汇率 ÷ 31.1035
理论换算价 = XAUUSD × 理论换算比
```

### 步骤3: 按优先级确定实际换算比

**🥇 优先 — 搜索到银行积存金实际报价**:
```
浙商换算比 = 浙商报价 ÷ XAUUSD
民生换算比 = 民生报价 ÷ XAUUSD
溢价 = 银行报价 − 理论换算价
→ 标注: "基于银行实时报价换算"
```

**🥈 备选 — 搜索到SGE Au99.99价格**:
```
SGE换算比 = Au99.99价格 ÷ XAUUSD
溢价 = Au99.99 − 理论换算价
浙商/民生均使用SGE换算比
→ 标注: "基于Au99.99换算，实际可能有±2元/克偏差"
```

**🥉 兜底 — 仅理论换算**:
```
使用理论换算比
→ 标注: "⚠️ 理论汇率换算，未含溢价，实际偏差可能达10-15元/克"
→ 建议用户查看银行APP
```

### 步骤4: 换算各关键点位
```
各点位积存金价 = 该XAUUSD价 × 实际换算比
```

### 步骤5: 在报告中呈现
```
点位表格: XAUUSD | 浙商积存金(元/克) | 民生积存金(元/克)
底部标注: 使用了哪级策略 + 换算比数值 + 溢价值 + 数据时间
```

## 注意事项

1. **溢价波动风险**: 国内外价差(溢价)非固定值，会随市场供需变化，影响实际收益
2. **汇率波动**: 人民币汇率变动直接影响换算比
3. **交易时间限制**: 积存金仅在工作日银行营业时间可交易，非24h
4. **非实时联动**: 积存金价格与国际金价存在滞后，尤其在剧烈波动时
5. **起购限制**: 虽然1克起购，但建议单笔≥5克以降低成本影响
6. **节假日风险**: 节假日国际金价波动无法通过积存金操作，需提前管理仓位
7. **报价以银行APP为准**: AI换算值为参考，实际交易价以银行端实时报价为准
