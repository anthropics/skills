# 黄金积存金短线交易智能分析

面向中国积存金短线交易者的AI分析工具，覆盖消息面汇总→技术分析→交易策略→积存金点位换算全流程。

## 工作流程

用户触发后按以下4个阶段顺序执行：

### 阶段1: 消息面搜集与市场情绪判断

**目标**: 小时级别消息面汇总，输出市场情绪标签

1. 使用 `web_search` 搜集近1-4小时黄金相关消息：
   - **宏观经济**: 美联储利率决议、非农数据、CPI/PPI、GDP、失业率
   - **地缘政治**: 军事冲突、贸易摩擦、制裁动态
   - **央行动态**: 各国央行购金/售金、外汇储备变动
   - **美元指数**: DXY走势、美债收益率变化
   - **市场资金**: ETF持仓变动(SPDR GLD)、COMEX持仓报告
   - **国内市场**: 上海金交所动态、央行黄金储备、人民币汇率

2. 对每条消息判定情感倾向：
   - **利多黄金**(+): 美联储降息预期、地缘冲突升级、美元走弱、避险情绪升温、央行增持
   - **利空黄金**(−): 美联储加息/缩表、风险偏好回升、美元走强、通胀数据降温
   - **中性**(0): 数据符合预期、无重大变化

3. 综合判定市场情绪标签：

| 条件 | 情绪标签 | 操作倾向 |
|------|----------|----------|
| 利多信号 ≥ 60% | 🟢 **看好黄金** | 逢低做多为主 |
| 利空信号 ≥ 60% | 🔴 **看跌黄金** | 逢高减仓/观望 |
| 利多利空均 < 60% | 🟡 **中性震荡** | 区间操作/轻仓 |

### 阶段2: 技术分析 — 压力位·支撑位·轴心位

**目标**: 基于实时行情计算关键点位

1. **获取实时数据**（必须按顺序搜索以下4组数据）：

**搜索A — 国际金价 XAUUSD**:
```
web_search: "XAUUSD spot price today" 或 "现货黄金实时行情"
获取: 当前价、日开盘、日最高、日最低、前日收盘
```

**搜索B — USD/CNY汇率**:
```
web_search: "USD CNY exchange rate today"
获取: 美元兑人民币即期汇率
```

**搜索C — 积存金实时报价（关键！用于计算准确换算比和溢价）**:
```
优先搜索: "浙商积存金 民生积存金 今日金价" 或 "积存金实时报价"
备选搜索: "上海黄金交易所 Au99.99 最新价" 或访问 sge.com.cn 延时行情
获取: 浙商黄金报价、民生黄金报价（或Au99.99价格作为代理）
```

**搜索D — 黄金TD价格（辅助验证溢价）**:
```
web_search: "黄金TD Au(T+D) 实时价格"
获取: 黄金TD价格，用于交叉验证国内金价水平
```

2. 计算压力位(R1/R2/R3)和支撑位(S1/S2/S3)，依据详见 [technical-analysis.md](references/technical-analysis.md)：

**压力位判断依据（5维）**:
- 近期高点、前期成交密集区
- 均线MA、布林带BOLL上轨压力
- MACD、KDJ、RSI指标共振位
- 实盘订单流与资金博弈阻力区
- 短期趋势线与形态颈线位

**支撑位判断依据（5维）**:
- 近期回调低点、前低支撑区域
- 关键均线、布林带下轨支撑
- 指标超卖区域与多头承接位
- 实盘买盘集中进场价位
- 趋势线、箱体下沿技术结构位

**统一原则**: 所有点位结合盘面实时走势，不硬套固定公式、跟着当下资金情绪走，点位精简实用只给关键有效位置。

3. 计算轴心位：

采用自定义算法（更贴合当下市场结构）：

```
P(轴心) = (R1 + S1) ÷ 2

其中:
  R1 = 第一压力位（最小数值压力位）
  S1 = 第一支撑位（最大数值支撑位）
```

**轴心位用途**:
- **判断强弱**: 金价在P之上 → 短期多头占优；金价在P之下 → 短期空头占优
- **交易参考**: 价格靠近轴心位 → 观察多空博弈；有效突破/跌破 → 新趋势确立
- **策略锚点**: 逢高减仓和逢低补仓的参考坐标，规划交易区间和风险边界

同时提供经典轴心位供对照（参见 [technical-analysis.md](references/technical-analysis.md)）。

### 阶段3: 交易策略生成与积存金点位换算

**目标**: 输出买卖策略并转换为积存金点位

1. 结合阶段1(情绪)和阶段2(点位)生成交易策略：

| 场景 | 策略建议 |
|------|----------|
| 情绪看好 + 价格靠近S1/S2 | 💰 **积极做多**，在支撑位附近分批买入 |
| 情绪看好 + 价格在轴心上方 | ⏳ **等回调**，等价格回到轴心附近再入场 |
| 情绪看跌 + 价格靠近R1/R2 | 🔻 **逢高减仓**，在压力位附近卖出 |
| 情绪中性 + 价格区间震荡 | 🔄 **区间操作**，支撑买/压力卖，轻仓 |

2. **积存金点位换算 — 三级获取策略**（详见 [accumulation-gold.md](references/accumulation-gold.md)）：

首先计算理论换算比作为基准:
```
理论换算比 = USD/CNY汇率 ÷ 31.1035
理论换算价 = XAUUSD × 理论换算比
```

然后按优先级获取**实际换算比**:

**🥇 优先: 获取到银行积存金实际报价**
```
浙商换算比 = 浙商积存金报价 ÷ 当前XAUUSD
民生换算比 = 民生积存金报价 ÷ 当前XAUUSD
溢价 = 银行报价 − 理论换算价
各点位积存金价 = XAUUSD关键点位 × 对应换算比
→ 标注: "基于银行实时报价换算"
```

**🥈 备选: 获取到上海金交所Au99.99价格（搜索SGE延时行情）**
```
SGE换算比 = Au99.99价格 ÷ 当前XAUUSD
溢价 = Au99.99价格 − 理论换算价
各点位积存金价 ≈ XAUUSD关键点位 × SGE换算比
→ 标注: "基于上海金交所Au99.99换算，积存金实际价格可能有±2元/克偏差"
```

**🥉 兜底: 仅有理论换算（未获取到国内价格）**
```
各点位积存金价 = XAUUSD关键点位 × 理论换算比
→ 标注: "⚠️ 基于理论汇率换算，未包含溢价，实际价格可能偏差10-15元/克"
→ 建议用户自行查看银行APP确认
```

输出时同时展示:
- **国际金价** (XAUUSD): 美元/盎司
- **浙商积存金参考**: 人民币元/克
- **民生积存金参考**: 人民币元/克
- **换算依据**: 标注使用了哪级获取策略 + 当前换算比 + 溢价值

> **注意**: 积存金实际成交价由各银行实时报价决定，AI给出的是换算参考值。
> 国内金价与国际金价之间存在浮动溢价（通常 -15 ~ +15 元/克），非固定价差。

## 执行原则

### Do（要做）
- ✅ 搜索最新实时数据，标注数据时间确保时效性
- ✅ 所有点位同时展示国际金价和积存金换算价
- ✅ 压力位/支撑位要结合多指标共振，给出关键有效位置
- ✅ 轴心位用 P=(R1+S1)÷2 公式计算
- ✅ 风险提示要充分，尤其是积存金买卖价差风险
- ✅ 区分消息面的时效性（小时级别 vs 日级别）

### Don't（不要做）
- ❌ 不给出绝对化建议（如"必须买入"、"一定涨"）
- ❌ 不忽略积存金与现货黄金的价差和交易成本
- ❌ 不使用过时数据，所有数据需标注时间
- ❌ 不硬套固定公式，要跟着当下资金情绪走
- ❌ 不忽略黑天鹅事件风险
