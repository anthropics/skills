# 黄金积存金短线交易智能分析

面向中国积存金短线交易者的AI分析工具，覆盖消息面汇总→技术分析→交易策略→积存金点位换算全流程。

## 核心定位与目标

- **交易节奏**: 小时/天级别短线，快进快出，利用黄金波动赚差价
- **数据原则**: 越实时越好，数据延迟=判断失效，必须标注每条数据的获取时间
- **收益目标**: 通过多次高胜率操作复利积累，目标资产翻倍
- **积存金特点**: 无杠杆，每次涨跌约1-3%，需选择波动大的窗口（地缘事件、重大数据发布日）入场，买卖价差约0.5-1元/克需计入成本

## 工作流程


用户触发后按以下4个阶段顺序执行：

### 阶段1: 消息面搜集与市场情绪判断

**目标**: 小时级别消息面汇总，输出市场情绪标签

1. 使用 `web_search` 搜集最近1-4小时黄金相关消息：
   - **宏观经济**: 美联储利率决议、非农数据、CPI/PPI、GDP、失业率
   - **地缘政治**: 军事冲突、贸易摩擦、制裁动态
   - **央行动态**: 各国央行购金/售金、外汇储备变动
   - **美元指数**: DXY走势、美债收益率变化
   - **市场资金**: ETF持仓变动(SPDR GLD)、COMEX持仓报告
   - **国内市场**: 上海金交所动态、央行黄金储备、人民币汇率

2. 对每条消息判定情感倾向：
   - **利多黄金**(+): 美联储降息预期、地缘冲突升级、美元走弱、避险情绪升温、央行增持
   - **利空黄金**(−): 美联储加息/缩表、风险偏好回升、美元走强、通胀数据降温
   - **中性**(0): 数据符合预期、无重大变化

3. 综合判定市场情绪标签：

| 条件 | 情绪标签 | 操作倾向 |
|------|----------|----------|
| 利多信号 ≥ 60% | 🟢 **看好黄金** | 逢低做多为主 |
| 利空信号 ≥ 60% | 🔴 **看跌黄金** | 逢高减仓/观望 |
| 利多利空均 < 60% | 🟡 **中性震荡** | 区间操作/轻仓 |

### 阶段2: 技术分析 — 压力位·支撑位·轴心位

**目标**: 基于实时行情计算关键点位

1. **获取实时数据**（按优先级顺序）：

**🥇 首选 — 直接 WebFetch Investing.com（一次获取全部关键数据）**:
```
WebFetch: https://www.investing.com/currencies/xau-usd
一次获取:
  - XAUUSD 实时报价（Bid/Ask，精确到分钟）
  - 今日 Open / High / Low
  - 多时间框架技术信号（30min/1H/5H/Daily/Weekly/Monthly 的 买入/卖出/强烈买入/强烈卖出）
  - 当日经济日历（已发布数据实际值 vs 预期值）
  - 相关市场数据（美债收益率、美元指数DXY、原油价格）
```

> ⚡ 这一步是短线交易的核心，必须执行，获取 Bid/Ask 实时价和多时间框架信号。

**搜索A — USD/CNY汇率（实时即期汇率）**:
```
web_search: "美元兑人民币 今日汇率" 或 "USD CNY spot rate today"
获取: 银行间即期汇率（非中间价）
注意: 当美元指数DXY大幅波动时（>0.5%），须使用即期汇率而非中间价，否则积存金换算偏差可达10-20元/克
```

**搜索B — 积存金实时报价（关键！用于计算准确换算比和溢价）**:
```
优先搜索: "浙商积存金 民生积存金 今日金价" 或 "积存金实时报价"
备选搜索: "上海黄金交易所 Au99.99 最新价" 或访问 sge.com.cn 延时行情
获取: 浙商黄金报价、民生黄金报价（或Au99.99价格作为代理）
```

**搜索C — 黄金TD价格（辅助验证溢价）**:
```
web_search: "黄金TD Au(T+D) 实时价格"
获取: 黄金TD价格，用于交叉验证国内金价水平
```

2. **解读多时间框架技术信号**（从 Investing.com 获取，为短线入场提供关键依据）：

| 信号级别 | 用途 | 操作含义 |
|----------|------|----------|
| **30分钟信号** | 判断当前入场时机 | 强烈卖出/卖出 → 不入场，等待；买入/强烈买入 → 可考虑入场 |
| **1小时信号** | 确认短线方向 | 与30分钟共振时信号最强 |
| **5小时信号** | 判断日内趋势 | 决定今天整体做多还是做空 |
| **日线/周线** | 确认大方向 | 顺大趋势操作，逆势要轻仓 |

**多时间框架信号解读规则**:
- 30min + 1H 均为卖出/强烈卖出 → **⛔ 不入场，等支撑**
- 30min + 1H 均为买入/强烈买入 → **✅ 可以入场**
- 30min 卖出 但 5H/日线 买入 → **📊 中线看多，短线等回调至支撑后买**
- 各级别信号矛盾 → **🔄 轻仓或观望**

3. 计算压力位(R1/R2/R3)和支撑位(S1/S2/S3)，依据详见 [technical-analysis.md](references/technical-analysis.md)：

**压力位判断依据（5维）**:
- 近期高点、前期成交密集区
- 均线MA、布林带BOLL上轨压力
- MACD、KDJ、RSI指标共振位
- 实盘订单流与资金博弈阻力区
- 短期趋势线与形态颈线位

**支撑位判断依据（5维）**:
- 近期回调低点、前低支撑区域
- 关键均线、布林带下轨支撑
- 指标超卖区域与多头承接位
- 实盘买盘集中进场价位
- 趋势线、箱体下沿技术结构位

**统一原则**: 所有点位结合盘面实时走势，不硬套固定公式、跟着当下资金情绪走，点位精简实用只给关键有效位置。

3. 计算轴心位：

采用自定义算法（更贴合当下市场结构）：

```
P(轴心) = (R1 + S1) ÷ 2

其中:
  R1 = 第一压力位（最小数值压力位）
  S1 = 第一支撑位（最大数值支撑位）

盘中补充算法（使用今日已确认的实际高低点）:
  P(盘中轴心) = (今日实际最高价 + 今日实际最低价) ÷ 2
  → 盘中两种轴心都展示，互相印证
```

**轴心位用途**:
- **判断强弱**: 金价在P之上 → 短期多头占优；金价在P之下 → 短期空头占优
- **交易参考**: 价格靠近轴心位 → 观察多空博弈；有效突破/跌破 → 新趋势确立
- **策略锚点**: 逢高减仓和逢低补仓的参考坐标，规划交易区间和风险边界

同时提供经典轴心位供对照（参见 [technical-analysis.md](references/technical-analysis.md)）。

### 阶段3: 交易策略生成与积存金点位换算

**目标**: 输出买卖策略并转换为积存金点位

1. 结合阶段1(情绪)、阶段2(点位)、**多时间框架信号**生成交易策略：

| 场景 | 策略建议 |
|------|----------|
| 情绪看好 + 价格靠近S1/S2 + **30min/1H 买入** | 💰 **立即分批买入**，在支撑位确认后入场 |
| 情绪看好 + 价格在轴心上方 + **30min/1H 买入** | ✅ **可以追多**，但注意设好止损 |
| 情绪看好 + 价格在轴心上方 + **30min/1H 卖出** | ⏳ **等回调**，不追高，等价格回调至S1再入场 |
| 情绪看好 + 价格从高点大幅回落 + **30min 强烈卖出** | ⛔ **不抢反弹**，等回调到支撑区止稳后入场 |
| 情绪看跌 + 价格靠近R1/R2 | 🔻 **逢高减仓**，在压力位附近卖出 |
| 情绪中性 + 价格区间震荡 | 🔄 **区间操作**，支撑买/压力卖，轻仓 |

**入场条件（不只看价位，还要有信号确认）**:
- 价格触及支撑区后，需要以下至少一个确认信号再入场：
  - 小时K线收阳（非假突破）
  - 成交量明显萎缩（卖盘枯竭）
  - RSI在支撑附近出现底背离
  - 30分钟信号由卖出转为买入

**风险收益比要求（每笔操作）**:
```
R:R = (目标收益) ÷ (止损距离) ≥ 1.5:1
例: 入场 1,162元/克，止损 1,153元/克（-9元），目标 1,179元/克（+17元）→ R:R = 1.9:1 ✅
```

2. **积存金点位换算 — 三级获取策略**（详见 [accumulation-gold.md](references/accumulation-gold.md)）：

首先计算理论换算比作为基准:
```
理论换算比 = USD/CNY汇率 ÷ 31.1035
理论换算价 = XAUUSD × 理论换算比
```

然后按优先级获取**实际换算比**:

**🥇 优先: 获取到银行积存金实际报价**
```
浙商换算比 = 浙商积存金报价 ÷ 当前XAUUSD
民生换算比 = 民生积存金报价 ÷ 当前XAUUSD
溢价 = 银行报价 − 理论换算价
各点位积存金价 = XAUUSD关键点位 × 对应换算比
→ 标注: "基于银行实时报价换算"
```

**🥈 备选: 获取到上海金交所Au99.99价格（搜索SGE延时行情）**
```
SGE换算比 = Au99.99价格 ÷ XAUUSD（取SGE报价时对应的同期XAUUSD，避免时间错位）
溢价 = Au99.99价格 − 理论换算价
各点位积存金价 ≈ XAUUSD关键点位 × SGE换算比
→ 标注: "基于上海金交所Au99.99换算，积存金实际价格可能有±5元/克偏差"

⚠️ SGE时间错位警告: SGE日间盘15:30收盘，若当前时间在中国夜盘或美盘，
SGE数据已是小时前的数据。若XAUUSD较SGE收盘后又大幅变动（>$20），
则用SGE换算比偏差较大，应改用理论换算法并注明。
```

**🥉 兜底: 仅有理论换算（未获取到国内价格）**
```
各点位积存金价 = XAUUSD关键点位 × 理论换算比
→ 标注: "⚠️ 基于理论汇率换算，未包含溢价，实际价格可能偏差10-15元/克"
→ 建议用户自行查看银行APP确认
```

输出时同时展示:
- **国际金价** (XAUUSD): 美元/盎司
- **浙商积存金参考**: 人民币元/克
- **民生积存金参考**: 人民币元/克
- **换算依据**: 标注使用了哪级获取策略 + 当前换算比 + 溢价值

> **注意**: 积存金实际成交价由各银行实时报价决定，AI给出的是换算参考值。
> 国内金价与国际金价之间存在浮动溢价（通常 -15 ~ +15 元/克），非固定价差。

- ✅ 每笔交易必须给出止损位和风险收益比（R:R ≥ 1.5:1）
- ✅ 盘中使用 P=(今日高+今日低)÷2 计算实时轴心
- ✅ 当美元指数DXY单日波动 >0.5% 时，必须说明对国内积存金价格的影响（美元涨→积存金涨幅缩小；美元跌→积存金涨幅放大）
- ✅ 区分消息面的时效性（小时级别 vs 日级别）
- ✅ 报告末尾必须包含「五、数据来源」章节，逐一列出每条数据的来源平台和获取时间

### Don't（不要做）
- ❌ **不在30min/1H信号为卖出时推荐入场**，必须先等待短线信号转好或等到支撑位确认
- ❌ **不用延时数据（SGE等）直接换算当前积存金价**，若XAUUSD已大幅变动，SGE数据已失效，改用理论换算法并注明
- ❌ 不给出绝对化建议（如"必须买入"、"一定涨"）
- ❌ 不忽略积存金的买卖价差成本（通常0.5-1元/克），每笔分析都要计入
- ❌ 不硬套固定公式，要跟着当下资金情绪和实时信号走
- ❌ 不忽略"买预期卖事实"规律：重大利多消息落地后（如数据实际发布时），金价可能反向回调
