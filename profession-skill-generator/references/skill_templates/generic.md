# 汎用スキルテンプレート

その他の職業や、特定カテゴリに当てはまらない場合に使用。

## 5軸評価の実施方法

対象職業について、以下の観点で評価:

### 時間軸
- **過去志向**: 記録、分析、監査、品質管理系
- **現在志向**: 処理、対応、運用、実行系
- **未来志向**: 企画、戦略、創造、イノベーション系

### 対象軸
- **内部特化**: 社内業務中心（経理、人事、IT管理など）
- **外部特化**: 対外業務中心（営業、CS、広報など）
- **内外統合**: 両方をつなぐ（マネージャー、PM、コンサルなど）

### 処理軸
- **ルーチン重視**: 定型・反復業務が多い
- **判断重視**: 専門知識に基づく判断が多い
- **創造重視**: 新しいものを生み出す業務が多い

### 価値軸
- **コンプライアンス**: 法令遵守、規則順守が重要
- **オペレーション**: 効率、品質、安定性が重要
- **イノベーション**: 革新、成長、差別化が重要

### 専門軸
- **汎用型**: 特定領域に限定されないスキル
- **専門型**: 深い専門知識が必要
- **統合型**: 複数領域を横断的に扱う

## 7カテゴリー設計の原則

1. **コア業務（2-3）**: その職業の本質的価値を生む業務
2. **サポート業務（2-3）**: コアを支える周辺業務
3. **成長業務（1-2）**: スキルアップ、改善活動

カテゴリー命名: 「○○・○○業務」形式

## 汎用的なスキルパターン

### ドキュメント系
- `{doc-type}-generator` - 文書生成
- `{doc-type}-reviewer` - 文書レビュー
- `{doc-type}-template-filler` - テンプレート入力

### コミュニケーション系
- `email-drafter` - メール作成
- `meeting-summarizer` - 会議要約
- `presentation-outliner` - プレゼン構成

### 分析系
- `data-summarizer` - データ要約
- `report-builder` - レポート作成
- `insight-extractor` - 洞察抽出

### 管理系
- `task-prioritizer` - タスク優先順位付け
- `schedule-optimizer` - スケジュール最適化
- `checklist-generator` - チェックリスト生成

## ペインからスキルへの変換パターン

| ペインのパターン | スキル化の方向性 |
|---|---|
| 「〇〇に時間がかかる」 | 自動化・効率化スキル |
| 「〇〇が面倒」 | テンプレート・自動生成スキル |
| 「〇〇の判断が難しい」 | 分析・提案スキル |
| 「〇〇を忘れる」 | チェックリスト・リマインダースキル |
| 「〇〇が複雑」 | 整理・可視化スキル |
| 「〇〇のネタがない」 | アイデア生成スキル |
| 「〇〇の品質が不安」 | レビュー・検証スキル |

## スキル命名規則

基本形: `{対象}-{動詞}` の kebab-case

動詞の例:
- generator, builder, creator（生成系）
- analyzer, checker, reviewer（分析系）
- summarizer, extractor（要約系）
- optimizer, enhancer（改善系）
- finder, suggester（提案系）
