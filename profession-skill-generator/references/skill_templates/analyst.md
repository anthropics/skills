# アナリスト向けスキルテンプレート

## 5軸評価の傾向

| 軸 | 典型的評価 |
|---|---|
| 時間軸 | 過去志向（分析）〜未来志向（予測） |
| 対象軸 | 内部特化〜内外統合 |
| 処理軸 | 判断重視 |
| 価値軸 | オペレーション〜イノベーション |
| 専門軸 | 専門型 |

## 7業務カテゴリー例

1. **データ収集・整備業務** - データ取得、クレンジング
2. **集計・可視化業務** - ダッシュボード、レポート
3. **分析・洞察抽出業務** - 統計分析、パターン発見
4. **予測・シミュレーション業務** - 予測モデル、シナリオ分析
5. **レポート・提言業務** - 報告書、アクション提案
6. **ステークホルダー対応業務** - 質問対応、アドホック分析
7. **分析基盤・自動化業務** - パイプライン構築、効率化

## よくあるペインと対応スキル案

### ペイン: アドホック分析依頼が多すぎる
- `query-template-library` - よくある集計クエリ集
- `self-service-guide-builder` - セルフサービス分析ガイド
- `data-dictionary-generator` - データ辞書生成

### ペイン: レポート作成に時間がかかる
- `analysis-report-drafter` - 分析レポート下書き
- `chart-narrative-writer` - グラフの説明文生成
- `insight-bullet-generator` - インサイト箇条書き化

### ペイン: データの品質問題が多い
- `data-quality-checker` - データ品質チェック
- `anomaly-detector` - 異常値検出
- `missing-data-reporter` - 欠損データレポート

### ペイン: 分析結果の伝え方が難しい
- `executive-insight-translator` - 経営層向け翻訳
- `visualization-recommender` - 可視化手法提案
- `storytelling-framework-applier` - ストーリー構成適用

### ペイン: SQLやコードの作成が大変
- `sql-query-generator` - SQLクエリ生成
- `python-snippet-builder` - Python分析コード生成
- `etl-logic-documenter` - ETLロジック文書化

## スキル命名規則

- データ/分析 + アクション の kebab-case
- 例: `data-quality-checker`, `insight-bullet-generator`
