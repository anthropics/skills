# 営業職向けスキルテンプレート

## 5軸評価の傾向

| 軸 | 典型的評価 |
|---|---|
| 時間軸 | 現在志向〜未来志向 |
| 対象軸 | 外部特化 |
| 処理軸 | 判断重視 |
| 価値軸 | オペレーション |
| 専門軸 | 汎用型〜専門型 |

## 7業務カテゴリー例

1. **見込客開拓・リード獲得業務** - リスト作成、アプローチ
2. **商談・プレゼン業務** - 提案書作成、デモ
3. **顧客対応・フォロー業務** - 問い合わせ対応、関係構築
4. **契約・クロージング業務** - 見積作成、契約締結
5. **CRM・案件管理業務** - 商談管理、パイプライン更新
6. **市場・競合分析業務** - 競合調査、市場動向把握
7. **営業企画・改善業務** - 営業戦略、ナレッジ共有

## よくあるペインと対応スキル案

### ペイン: 提案書作成に時間がかかる
- `proposal-drafter` - 提案書ドラフト作成
- `case-study-finder` - 類似事例検索
- `roi-calculator-builder` - ROI計算シート作成

### ペイン: 商談メモ・CRM更新が面倒
- `meeting-notes-to-crm` - 商談メモからCRM更新
- `next-action-extractor` - ネクストアクション抽出
- `deal-summary-generator` - 案件サマリー生成

### ペイン: フォローメール作成が大変
- `follow-up-email-drafter` - フォローメール作成
- `thank-you-note-generator` - お礼メール生成
- `nurture-sequence-planner` - ナーチャリング計画

### ペイン: 見込客リストの整理が大変
- `lead-qualifier` - リード評価・スコアリング
- `company-research-summarizer` - 企業情報要約
- `contact-enricher` - コンタクト情報補完

### ペイン: 競合対策の情報整理が大変
- `competitor-battlecard-builder` - 競合バトルカード作成
- `objection-handler-library` - 反論対応集
- `win-loss-analyzer` - 勝敗分析

## スキル命名規則

- 営業プロセス + アクション の kebab-case
- 例: `proposal-drafter`, `lead-qualifier`
