# マネージャー向けスキルテンプレート

## 5軸評価の傾向

| 軸 | 典型的評価 |
|---|---|
| 時間軸 | 現在志向〜未来志向 |
| 対象軸 | 内外統合 |
| 処理軸 | 判断重視 |
| 価値軸 | オペレーション〜イノベーション |
| 専門軸 | 統合型 |

## 7業務カテゴリー例

1. **戦略立案・目標設定業務** - OKR設定、ロードマップ
2. **プロジェクト管理業務** - 進捗管理、リソース配分
3. **チームマネジメント業務** - 1on1、評価、育成
4. **会議・コミュニケーション業務** - 会議運営、情報共有
5. **意思決定・問題解決業務** - 判断、エスカレーション対応
6. **ステークホルダー連携業務** - 経営報告、部門間調整
7. **組織開発・改善業務** - プロセス改善、文化醸成

## よくあるペインと対応スキル案

### ペイン: 会議が多すぎて時間がない
- `meeting-agenda-optimizer` - アジェンダ最適化
- `meeting-notes-summarizer` - 議事録自動要約
- `async-decision-facilitator` - 非同期意思決定支援

### ペイン: 進捗報告の作成が面倒
- `status-report-generator` - ステータスレポート生成
- `project-health-checker` - プロジェクト健全性チェック
- `blocker-identifier` - ブロッカー特定・可視化

### ペイン: 1on1の準備・記録が大変
- `one-on-one-prep-helper` - 1on1準備シート生成
- `feedback-drafter` - フィードバック文作成
- `growth-tracker` - 成長記録管理

### ペイン: 意思決定に時間がかかる
- `decision-matrix-builder` - 意思決定マトリクス作成
- `pros-cons-analyzer` - メリデメ分析
- `risk-assessment-helper` - リスク評価支援

### ペイン: 経営層への報告資料作成が大変
- `executive-summary-builder` - エグゼクティブサマリー生成
- `kpi-narrative-writer` - KPIストーリー作成
- `presentation-outliner` - プレゼン構成作成

## スキル命名規則

- 業務 + 支援動詞 の kebab-case
- 例: `meeting-notes-summarizer`, `decision-matrix-builder`
