# デザイナー向けスキルテンプレート

## 5軸評価の傾向

| 軸 | 典型的評価 |
|---|---|
| 時間軸 | 未来志向（創造） |
| 対象軸 | 内外統合 |
| 処理軸 | 創造重視 |
| 価値軸 | イノベーション |
| 専門軸 | 専門型〜統合型 |

## 7業務カテゴリー例

1. **リサーチ・ユーザー理解業務** - ユーザー調査、ペルソナ作成
2. **コンセプト・企画業務** - コンセプト設計、ムードボード
3. **UI/UX設計業務** - ワイヤーフレーム、プロトタイプ
4. **ビジュアル制作業務** - グラフィック、イラスト
5. **デザインシステム管理業務** - コンポーネント管理、ガイドライン
6. **フィードバック・改善業務** - レビュー対応、A/Bテスト分析
7. **アセット管理・納品業務** - ファイル管理、書き出し

## よくあるペインと対応スキル案

### ペイン: デザインシステムの管理が複雑
- `design-token-documenter` - デザイントークン文書化
- `component-inventory-tracker` - コンポーネント棚卸し
- `style-guide-generator` - スタイルガイド生成

### ペイン: フィードバック対応に追われる
- `feedback-categorizer` - フィードバック分類・整理
- `revision-tracker` - 修正履歴管理
- `stakeholder-summary-builder` - ステークホルダー向け要約

### ペイン: 仕様書・ハンドオフ資料作成が面倒
- `design-spec-generator` - デザイン仕様書生成
- `handoff-checklist-creator` - ハンドオフチェックリスト
- `dev-annotation-helper` - 開発者向け注釈追加

### ペイン: アイデア出しに詰まる
- `design-inspiration-finder` - インスピレーション検索
- `layout-pattern-suggester` - レイアウトパターン提案
- `color-palette-generator` - カラーパレット生成

### ペイン: ユーザー調査の整理が大変
- `interview-insight-extractor` - インタビュー洞察抽出
- `persona-builder` - ペルソナ作成支援
- `journey-map-outliner` - ジャーニーマップ骨子作成

## スキル命名規則

- 成果物 + アクション の kebab-case
- 例: `design-spec-generator`, `component-inventory-tracker`
