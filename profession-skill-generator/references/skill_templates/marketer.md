# マーケター向けスキルテンプレート

## 5軸評価の傾向

| 軸 | 典型的評価 |
|---|---|
| 時間軸 | 過去志向〜未来志向（全般） |
| 対象軸 | 外部特化〜内外統合 |
| 処理軸 | 判断重視〜創造重視 |
| 価値軸 | イノベーション |
| 専門軸 | 統合型 |

## 7業務カテゴリー例

1. **市場調査・競合分析業務** - リサーチ、トレンド分析
2. **戦略立案・企画業務** - マーケ戦略、キャンペーン企画
3. **コンテンツ制作業務** - 記事、SNS投稿、広告クリエイティブ
4. **広告運用・配信管理業務** - 広告設定、入札調整
5. **データ分析・レポート業務** - KPI分析、週次/月次レポート
6. **CRM・顧客コミュニケーション業務** - メール配信、セグメント管理
7. **ブランディング・PR業務** - ブランド管理、プレスリリース

## よくあるペインと対応スキル案

### ペイン: レポート作成が毎週面倒
- `weekly-report-builder` - 週次レポート自動生成
- `kpi-dashboard-narrator` - KPIを文章で解説
- `data-insight-extractor` - データからインサイト抽出

### ペイン: コンテンツのネタ切れ
- `content-idea-generator` - コンテンツアイデア生成
- `trending-topic-finder` - トレンドトピック発見
- `competitor-content-analyzer` - 競合コンテンツ分析

### ペイン: SNS投稿作成に時間がかかる
- `social-post-drafter` - SNS投稿文作成
- `hashtag-optimizer` - ハッシュタグ最適化
- `post-scheduler-planner` - 投稿スケジュール提案

### ペイン: 広告文の作成が大変
- `ad-copy-generator` - 広告コピー生成
- `ab-test-variant-creator` - A/Bテストバリエーション作成
- `cta-optimizer` - CTAボタン文言最適化

### ペイン: 競合情報の収集が大変
- `competitor-tracker` - 競合動向追跡
- `market-news-summarizer` - 業界ニュース要約
- `swot-analyzer` - SWOT分析支援

## スキル命名規則

- 対象 + アクション の kebab-case
- 例: `weekly-report-builder`, `content-idea-generator`
