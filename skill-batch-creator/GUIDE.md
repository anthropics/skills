# スプレッドシート記入ガイド

## 概要

このスプレッドシートは、職業別Agent Skillsを一括作成するためのリクエストシートです。
複数人で同時に記入し、Claude Codeで参照することで効率的にスキルを作成できます。

---

## 使い方

### 1. スプレッドシートを共有

`skill_request_template.csv` をGoogle Sheetsにインポートするか、Excelで開いて共有してください。

**Google Sheetsの場合：**
1. Google Driveで「新規」→「ファイルのアップロード」
2. CSVファイルをアップロード
3. 「共有」ボタンで編集権限を付与

### 2. 各自が記入

各列の説明に従って、自分の業務課題を記入してください。

### 3. CSVでエクスポート

記入完了後、CSVファイルとしてダウンロードしてください。

**Google Sheetsの場合：**
「ファイル」→「ダウンロード」→「カンマ区切り形式（.csv）」

### 4. Claude Codeで一括作成

```bash
# Claude Codeで以下のように依頼
"skill_requests.csv を参照して、記載されている全員分のAgent Skillsを作成してください"
```

## 列の説明

| 列名 | 必須 | 説明 | 記入例 |
|------|------|------|--------|
| **記入者名** | ○ | あなたの名前 | 田中太郎 |
| **部門/チーム** | ○ | 所属部門やチーム名 | 営業部、開発チームA |
| **職業/役職** | ○ | 職種や役職 | 営業マネージャー、バックエンドエンジニア |
| **ペインポイント** | ○ | 日常業務で困っていること、時間がかかっていること | 毎週の営業レポート作成に3時間かかる |
| **解決したいタスク** | ○ | 具体的に自動化・効率化したい作業 | 週次営業レポートの自動生成 |
| **希望するスキル名** | △ | スキルの名前（空欄でもOK） | sales-report-generator |
| **出力形式の希望** | △ | 成果物のファイル形式 | Excel, PDF, Markdown, CSV |
| **優先度** | △ | 高/中/低 から選択 | 高 |
| **備考** | - | 補足情報、特別な要望など | グラフも含めてほしい |

---

## 記入のコツ

### 良いペインポイントの書き方

**具体的に書く：**
- ✅ 「毎週月曜に先週の売上データを集計してレポートを作成するのに3時間かかる」
- ❌ 「レポート作成が大変」

**数値を含める：**
- ✅ 「月に20件の契約書をレビューしている」
- ❌ 「契約書のレビューが多い」

### 良いタスクの書き方

**アクションを明確に：**
- ✅ 「売上CSVから週次レポート（グラフ付き）をPDFで生成」
- ❌ 「レポートを作る」

**インプットとアウトプットを意識：**
- ✅ 「議事録テキストから、タスクリストをMarkdownで抽出」
- ❌ 「議事録を整理」

## サンプル記入例

| 職業 | ペインポイント | 解決したいタスク |
|------|---------------|-----------------|
| 経理担当 | 請求書の確認と仕訳入力に毎月丸1日かかる | 請求書PDFから仕訳データをCSVで出力 |
| プロジェクトマネージャー | 進捗報告メールの作成に毎日30分 | タスク管理ツールの情報から進捗報告を自動生成 |
| カスタマーサポート | FAQ更新が追いつかない | 問い合わせ履歴からFAQ候補を抽出 |
| デザイナー | デザインレビュー依頼の文面作成 | デザイン画像からレビュー依頼文を生成 |
| 法務担当 | 契約書のリスク箇所チェックに時間がかかる | 契約書PDFからリスク条項をハイライト |
| 採用担当 | 面接日程調整のメール往復が多い | 候補日から調整メール文面を生成 |
| マーケター | 競合分析レポートの作成 | Webページから競合情報を抽出・比較表作成 |
| 研究者 | 論文サーベイのまとめ作成 | 複数論文PDFから要約と比較表を生成 |

---

## 注意事項

- 機密情報は記載しないでください
- 1人で複数のスキルリクエストを記入してもOKです
- 空行は残しておいて構いません（処理時にスキップされます）
- スキル名は英語・ハイフン区切りが推奨です（例: `sales-report-generator`）
