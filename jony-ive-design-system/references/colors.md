# カラーパレット詳細

## 設計思想

**「色で騒がない」**

Jony Iveのデザインは色に頼らない。情報の優先度はグレーの濃淡で表現し、本当に特別な意味を持つものにだけアクセント色を使う。

## 基本カラー（グレースケール）

| 用途 | 色名 | HEX | RGB | 使用場面 |
|------|------|-----|-----|----------|
| タイトル | ブラック | #000000 | 0,0,0 | 最重要テキスト |
| 本文/ヘッダー背景 | ダークグレー | #333333 | 51,51,51 | 本文、確定した情報 |
| 見出し/進行中 | ミディアムグレー | #666666 | 102,102,102 | 注目すべき情報 |
| 補足 | ライトグレー | #999999 | 153,153,153 | 補足、注釈 |
| 未着手 | ペールグレー | #CCCCCC | 204,204,204 | 存在するが未処理 |
| 区切り線 | セパレーター | #E0E0E0 | 224,224,224 | 罫線、境界 |
| フェーズ背景 | 背景グレー | #F5F5F5 | 245,245,245 | グループ区分 |
| 背景 | ホワイト | #FFFFFF | 255,255,255 | 基本背景 |

## アクセントカラー（1色のみ）

| 用途 | 色名 | HEX | RGB | 使用場面 |
|------|------|-----|-----|----------|
| マイルストーン | アクセントブルー | #5B7B94 | 91,123,148 | 特別な節目のみ |

**注意**: アクセント色は「マイルストーン」など、本当に特別な意味を持つ要素にのみ使用する。それ以外は全てグレースケールで表現する。

## ステータス表現

### グレー濃淡による意味付け

```
濃い ← ─────────────────────────── → 薄い
確定     注目      存在      未定義

#333333  #666666  #CCCCCC   #FFFFFF
 完了    進行中    未着手     保留
```

| ステータス | HEX | 意味 | 視覚的印象 |
|-----------|-----|------|-----------|
| 完了 | #333333 | 確定・安定 | 重く、確かな存在感 |
| 進行中 | #666666 | 注目すべき | 目に入る、活動的 |
| 未着手 | #CCCCCC | 存在するが未処理 | 控えめ、待機中 |
| 保留 | #FFFFFF + 枠線 | 空・停止 | 「無い」状態 |
| マイルストーン | #5B7B94 | 特別な節目 | 唯一の色、際立つ |

## 使用原則

### 色の使用頻度

```
ホワイト     ████████████████████ 60%（背景）
グレー系     ████████████████ 38%（テキスト、ステータス）
アクセント   █ 2%（マイルストーンのみ）
```

### ルール

1. **98%はグレースケール** - 色は例外
2. **アクセント色は1色のみ** - #5B7B94（くすんだブルー）
3. **濃淡で優先度を表現** - 濃い＝重要/確定、薄い＝補足/未定
4. **派手な色は使わない** - 赤、オレンジ、緑などは禁止
5. **Officeデフォルト色を避ける** - #4472C4等は使わない

## Python定数

```python
COLORS = {
    # 基本グレースケール
    "black": "000000",
    "dark_gray": "333333",
    "medium_gray": "666666",
    "light_gray": "999999",
    "pale_gray": "CCCCCC",
    "separator": "E0E0E0",
    "bg_gray": "F5F5F5",
    "white": "FFFFFF",
    
    # アクセント（1色のみ）
    "accent": "5B7B94",
    
    # ステータス（グレー濃淡）
    "status_pending": "CCCCCC",    # 未着手（薄い＝控えめ）
    "status_progress": "666666",   # 進行中（中間＝注目）
    "status_done": "333333",       # 完了（濃い＝確定）
    "status_hold": "FFFFFF",       # 保留（白＝空）
    "milestone": "5B7B94",         # マイルストーン（唯一のアクセント）
    
    # 構造
    "header_bg": "333333",         # ヘッダー背景（ダークグレー）
    "phase_bg": "F5F5F5",          # フェーズ背景（薄いグレー）
}
```

## 廃止した色（使用禁止）

以下の色は「派手すぎる」ため使用しない：

| 旧色 | HEX | 理由 |
|------|-----|------|
| ネイビー | #366092 | Officeデフォルト、主張が強い |
| ライトブルー | #B4C7E7 | Officeデフォルト |
| 鮮やかブルー | #4472C4 | Officeデフォルト、派手 |
| グリーン | #70AD47 | 派手、色数増加 |
| オレンジ | #ED7D31 | 派手、色数増加 |
| レッド | #FF6B6B | 派手、警告色は不要 |
| イエロー | #FFFACD | 色数増加 |
