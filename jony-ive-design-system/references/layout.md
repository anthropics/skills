# レイアウト原則

## 余白（Whitespace）

### 基本思想
**余白は無駄ではなく、デザイン要素**

余白を恐れない。情報を詰め込むより、呼吸する空間を与える。

### 余白の種類

| 種類 | 説明 | 目安 |
|------|------|------|
| マージン | ページ端からの距離 | 20-30mm |
| パディング | 要素内の余白 | 8-16px |
| ガター | 列間の余白 | 16-24px |
| セクション間 | ブロック間の余白 | 24-32px |

### 余白の比率

```
小 : 中 : 大 = 1 : 2 : 3
例: 8px : 16px : 24px
```

## グリッドシステム

### 原則
- 見えないグリッドに沿って配置
- 要素の開始位置を揃える
- 不規則な配置を避ける

### 整列パターン

```
┌─────────────────────────────┐
│  ┌───────────────────────┐  │
│  │ タイトル              │  │
│  └───────────────────────┘  │
│                             │
│  ┌───────────────────────┐  │
│  │ セクション1           │  │
│  │   コンテンツ          │  │
│  └───────────────────────┘  │
│                             │
│  ┌───────────────────────┐  │
│  │ セクション2           │  │
│  │   コンテンツ          │  │
│  └───────────────────────┘  │
└─────────────────────────────┘
```

## 視覚的階層

### 優先度の表現方法

1. **サイズ** - 大きいほど重要
2. **太さ** - 太いほど重要
3. **色** - 濃いほど重要
4. **位置** - 上・左ほど重要
5. **余白** - 周囲に余白があるほど目立つ

### 階層の例（請求書）

```
[最重要] 請求金額 ─────────── 36pt, Bold, #000
[重要]   タイトル「請求書」── 28pt, Bold, #000
[通常]   明細内容 ─────────── 10pt, Regular, #333
[補足]   発行者情報 ────────── 9pt, Regular, #999
```

## 罫線・区切り

### 使用する罫線
- **細いグレー線**（#E0E0E0、1px）
- 下線のみ（上下左右の枠は避ける）

### 避ける罫線
- 太い線
- 二重線
- 装飾的な線
- 全面グリッド

### 区切りの代替手段
罫線の代わりに**余白**で区切ることを優先

```
❌ ─────────────────
   セクションA
   ─────────────────
   セクションB
   ─────────────────

✅ 
   セクションA
   
   
   セクションB
```

## 情報密度

### ルール
- 1画面に入れる情報量を制限
- スクロールを恐れない
- 空白を恐れない

### 目安
- A4ページ: 本文は60-70%以下
- 表: 行間に余裕を持たせる（行高18-22px）
- リスト: 項目間に8px以上の余白
