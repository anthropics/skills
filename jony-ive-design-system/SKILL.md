# Jony Ive Design System

## 概要

Apple元チーフデザインオフィサー、ジョナサン・アイブのデザイン哲学に基づくビジネス文書向けスタイルガイドライン。

**"Simplicity is the ultimate sophistication"**（シンプルさこそ究極の洗練）

## デザイン原則

1. **極限までシンプルに** - 不要な装飾を徹底的に排除
2. **余白を贅沢に使う** - 情報に呼吸する空間を与える
3. **完璧な整列** - すべての要素を精密に配置
4. **色で騒がない** - グレースケール基調、アクセント色は1色のみ
5. **階層の明確化** - 濃淡と太さで情報の重要度を表現

## クイックリファレンス

### カラーパレット（グレースケール＋1アクセント）

**基本カラー（グレースケール）**

| 用途 | HEX | 使用場面 |
|------|-----|----------|
| ブラック | #000000 | タイトル、最重要テキスト |
| ダークグレー | #333333 | 本文、ヘッダー背景 |
| ミディアムグレー | #666666 | 見出し、進行中ステータス |
| ライトグレー | #999999 | 補足情報 |
| ペールグレー | #CCCCCC | 未着手ステータス |
| セパレーター | #E0E0E0 | 区切り線 |
| 背景グレー | #F5F5F5 | フェーズ背景 |
| ホワイト | #FFFFFF | 基本背景 |

**アクセントカラー（1色のみ）**

| 用途 | HEX | 使用場面 |
|------|-----|----------|
| アクセントブルー | #5B7B94 | マイルストーンのみ |

### ステータス表現（グレー濃淡）

| ステータス | 色 | 考え方 |
|-----------|-----|--------|
| 未着手 | #CCCCCC | 存在するが目立たない |
| 進行中 | #666666 | 注目すべき |
| 完了 | #333333 | 確定・安定 |
| 保留 | #FFFFFF + 枠線 | 「空」の状態 |
| マイルストーン | #5B7B94 | 唯一の色、特別な意味 |

### フォント

- **統一フォント**: メイリオ（日本語環境）
- 英語環境の場合: Helvetica Neue または San Francisco

### フォントサイズ階層

| レベル | サイズ | 太さ | 用途 |
|--------|--------|------|------|
| H1 | 28-36pt | Bold | ドキュメントタイトル |
| H2 | 14-16pt | Bold | セクション見出し |
| H3 | 11-12pt | Bold | サブ見出し |
| 本文 | 10pt | Regular | 通常テキスト |
| 補足 | 9pt | Regular | 注釈、フッター |

## アンチパターン（避けるべきこと）

❌ 派手な色使い（赤、オレンジ、緑など）
❌ 複数のアクセント色
❌ 装飾的な罫線・枠
❌ 過度な太字・下線
❌ 中央揃えの多用
❌ グラデーション・影
❌ 複数フォントの混在
❌ 情報の詰め込みすぎ

## Python定数

```python
COLORS = {
    # 基本グレースケール
    "black": "000000",
    "dark_gray": "333333",
    "medium_gray": "666666",
    "light_gray": "999999",
    "pale_gray": "CCCCCC",
    "separator": "E0E0E0",
    "bg_gray": "F5F5F5",
    "white": "FFFFFF",
    
    # アクセント（1色のみ）
    "accent": "5B7B94",
    
    # ステータス（グレー濃淡）
    "status_pending": "CCCCCC",    # 未着手
    "status_progress": "666666",   # 進行中
    "status_done": "333333",       # 完了
    "status_hold": "FFFFFF",       # 保留（枠線付き）
    "milestone": "5B7B94",         # マイルストーン（唯一のアクセント）
    
    # 構造
    "header_bg": "333333",         # ヘッダー背景
    "phase_bg": "F5F5F5",          # フェーズ背景
}

FONT_NAME = "メイリオ"
```
