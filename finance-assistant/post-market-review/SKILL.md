## 📊 盘后数据采集清单（15:00-18:00）

### 模块1：市场总览数据

- 📈 **两市成交额**：今日总量 vs 近5日均值（判断情绪热度）
- 📊 **涨跌停统计**：涨停板数量 / 跌停板数量 / 涨跌停比
- 🏭 **成交量情绪指标**：量比（今日量/近5日均量），>1.5=活跃，<0.8=萎缩
- 📉 **指数收盘情况**：沪深300/中证500/创业板/科创50各自涨跌幅

### 模块2：ETF份额变化（15:00后陆续公告）

**数据源**：上交所/深交所ETF份额公告（每日15:00-17:00发布）

| 信号 | 含义 | 权重调整 |
|------|------|---------|
| 份额净增加 > 5% | 机构大幅申购，看多 | +10~20分 |
| 份额净增加 1-5% | 机构小幅买入 | +5~10分 |
| 份额基本不变 | 机构中性 | 0 |
| 份额净减少 1-5% | 机构小幅赎回 | -5~10分 |
| 份额净减少 > 5% | 机构大幅赎回，看空 | -10~20分 |

### 模块3：主力资金净流向（东方财富/同花顺）

**板块资金排行**（重点关注前5名净流入/流出板块）：

```
净流入最多的板块 → 明日主线候选
净流出最多的板块 → 明日回避方向
净流入持续3日+  → 主线确立，可加仓相关ETF
净流出持续3日+  → 主线结束，应减仓相关ETF
```

### 模块4：龙虎榜数据（若有）

- 哪些ETF/成分股上榜？
- 机构净买入 vs 游资净买入：机构信号更稳定
- 北向资金（季度披露）：龙虎榜上是否出现北向资金集中操作？

### 模块5：政策与研报更新（18:00-22:00）

**政策扫描**：
- 今日央行/发改委/工信部/证监会有无新文件？
- 人民日报/新华社/央视财经有无新定调或新提法？
- 重要部委官员有无公开发言？

**研报扫描**：
- 头部券商（中信/中金/华泰）今日有无相关行业研报？
- 是否有上调/下调目标价的动作？（3个月内≥10家上调=加分+10）

## 🔄 评分更新规则

基于今日新信息，更新各持仓ETF的综合评分：

```
政策评分更新触发条件:
  + 新政策利好文件       → 政策分 +10~30
  + 监管约谈/利空政策    → 政策分 -20~40
  + 无新政策             → 政策分不变

资金评分更新触发条件:
  + ETF份额净增 > 5%     → 资金分 +15~20
  + 主力净流入连续3日+   → 资金分 +10~15
  + ETF份额净减 > 5%     → 资金分 -15~20
  + 主力净流出连续3日+   → 资金分 -10~15

技术评分更新触发条件:
  + 放量突破关键压力位   → 技术分 +15~25
  + 均线金叉确立         → 技术分 +10~15
  + 跌破关键支撑/均线    → 技术分 -15~25
  + 放量下跌破位         → 技术分 -20~30
```

## ⚙️ 执行原则

### ✅ Do
- **15:30开始复盘**，趁记忆清晰记录今日操作原因
- **数据优先**：用数据更新评分，不用"感觉"覆盖判断
- **记录错误**：每次未执行止损的原因必须写下来
- **评分及时更新**：有重大新信号时立即修正评分

### ❌ Don't
- 不在盘后"后悔"没追的股票（幸存者偏差）
- 不因今日亏损而明日冲动报复性交易
- 不忽略ETF份额变化数据（机构行为最客观）
- 不把每日小波动当成"信号变化"来频繁更新评分

## 🚀 开始执行

当用户触发此Skill时：

1. 采集今日市场总览数据（成交量/涨跌停/指数涨跌）
2. 查询持仓ETF今日份额变化和主力资金数据
3. 生成操作复盘表格（计划 vs 实际）
4. 更新各ETF综合评分（有变化才更新，无变化不动）
5. 输出明日操作计划清单

> **核心心法**：盘后复盘不是为了后悔，是为了让系统变得更聪明。每一次记录都是在给未来的自己留下「战场情报」。
