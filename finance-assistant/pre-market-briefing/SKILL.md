## 🔎 盘前信息扫描清单（5大模块 · 定序执行）

### 模块1：隔夜消息扫描（权重最高）⏱️ 8:00-8:20

#### 1.1 A股政策面（最高优先级）
- 🏛️ **凌晨/昨夜政策文件**：新华社客户端、国务院客户端、证监会官网
  - 是否有新会议通知/政策文件？
  - 是否有监管约谈/暂停/新规？
  - 是否有「新提法」首次出现？
- 🏦 **央行操作**：今日逆回购投放/回笼规模 → 净投放=偏宽松，净回笼=偏紧
- 📋 **今日重要日程**：CPI/PPI/PMI发布？特殊会议？国务院新闻发布会？

**⚡ 判断规则**：
```
出现【重大利好政策】→ 今日做多偏强，评分+20分
出现【监管利空信号】→ 今日降仓，评分-25分
央行净投放 > 1000亿  → 流动性宽松加分+5
无特殊政策           → 政策面中性，不影响方向
```

#### 1.2 隔夜美股/港股夜盘
- **纳斯达克涨跌幅**：>+1% 科技情绪正向传导；<-2% 科技板块需降权
- **恒生指数夜盘**（若有）：港股夜盘方向 → 港股ETF及外资偏好信号
- **VIX恐慌指数**：VIX > 25 = 全球避险情绪升温，A股科技/成长承压
- **美元指数/离岸人民币**：美元大涨 → 北向资金流出压力；人民币升值 → 外资流入利好

#### 1.3 大宗商品与宏观
- **原油价格变化**：影响能源ETF、化工ETF方向
- **黄金价格**：黄金大涨 = 避险情绪，防御品种受益
- **铜价/铝价**：中国需求敏感品，涨价 → 有色金属ETF利好
- **美债10年期收益率**：>4.5%且上行 → 成长股估值承压

### 模块2：期货与衍生品信号（市场方向预判）⏱️ 8:20-8:40

#### 2.1 股指期货升贴水（核心指标）

| 合约 | 关注点 | 信号含义 |
|------|--------|---------|
| **IF（沪深300期货）** | 主力合约升贴水 | 升水>15点=机构看多；贴水>15点=机构对冲/看空 |
| **IC（中证500期货）** | 主力合约升贴水 | IC贴水收窄=中小盘预期改善 |
| **IM（中证1000期货）** | 升贴水变化 | 反映小盘股/量化策略方向 |

**判断规则**：
```
IF升水持续扩大   → 今日大盘偏强，沪深300ETF加分
IF贴水扩大      → 机构做空对冲，大盘承压，降仓
IC贴水 > 30点   → 小盘成本高，量化资金观望
```

#### 2.2 ETF期权隐含波动率（选做）
- 50ETF期权/300ETF期权IV水平
- IV大幅上升 = 市场预期今日波动放大，减小仓位或等待稳定

---

### 模块3：成分股与ETF盘前状态 ⏱️ 8:40-9:15

#### 3.1 持仓ETF核心成分股动态
- 昨夜/今早持仓ETF核心成分股有无重大公告：
  - [ ] 业绩预告/年报/半年报
  - [ ] 大股东增减持公告
  - [ ] 重大合同/诉讼/处罚
  - [ ] 停牌/复牌公告
- **龙头股港股/ADR表现**（若有对应标的）

#### 3.2 重要解禁日核查
- 今日是否有ETF成分股大额解禁？（交易所公告）
- 解禁量 > 流通市值5% → 相关ETF降权

#### 3.3 ETF折溢价预判
- 前日收盘ETF折溢价数据（上交所/深交所公告）
- 折价 > 1% → 抑制买入冲动；溢价 > 1% → 警惕追高
- 跨境ETF（港股/美股ETF）：海外市场已收盘，折溢价可提前精确计算

### 模块4：竞价阶段实时观察（9:15-9:25）⏱️ 实时

> ⚠️ 竞价阶段是盘前最后确认窗口，此时不做新决策，只做最终确认

#### 4.1 指数竞价方向
- 沪深300ETF竞价：高开 > 0.5% / 平开 / 低开 < -0.5%？
- 与股指期货方向是否一致？（一致=信号可信度↑）

#### 4.2 成交量判断
```
竞价成交量 > 平日均值120%  → 情绪活跃，今日波动可能放大
竞价成交量 < 平日均值80%   → 观望情绪重，震荡概率高
高开+竞价量大              → 追高风险高，等回调再入
低开+竞价量萎缩            → 观望，等开盘方向确认后再行动
```

#### 4.3 ETF竞价折溢价
- 重点观察今日计划操作的ETF竞价价格vs上日收盘NAV
- 溢价 > 0.5% = 溢价买入成本高，考虑延迟建仓

## 🔴 盘前紧急预案

### 开盘前发现以下情况，立即调整计划：

| 情况 | 触发条件 | 立即行动 |
|------|---------|---------|
| **政策急转弯** | 凌晨重大利空政策文件 | 删除全部买入计划，检查止损位 |
| **美股暴跌>2%** | 纳指单日跌幅>2% | 科技类ETF买入计划降仓50% |
| **期货大幅贴水** | IF贴水>30点 | 不追高，等开盘稳定后观察 |
| **核心成分股停牌** | 重仓ETF核心股停牌>10% | ETF流动性风险，降低该ETF仓位 |
| **人民币大幅贬值** | 离岸人民币单日贬>0.5% | 北向资金流出压力，大盘ETF降权 |

## ⚙️ 执行原则

### ✅ Do
- **8:00准时扫描**，不在开盘后才开始收集信息
- **决策要在9:25前完成**，9:30后只执行，不临时改变方向
- **信息优先级从高到低**：政策 → 期货信号 → 全球市场 → 技术面
- **高开不追**：超过前日收盘1.5%以上的高开，等重新评估再入场

### ❌ Don't
- 不在9:30后还在收集信息做决策（开盘后信息会带来情绪干扰）
- 不因昨日涨停/大涨就在盘前决定追高
- 不忽视期货升贴水信号（这是机构今日方向的"投票"）
- 不把竞价15分钟当作"赚钱窗口"，竞价只用于确认，不用于猜测

## 🚀 开始执行

当用户触发此Skill时：

1. **告知当前时间**，确认是否在合理的盘前时间窗口
2. **按模块1→5顺序执行信息扫描**（优先政策和期货信号）

5. **标注今日最大风险**（一条最关键的风险提示）

> **核心心法**：盘前的功夫，决定盘中的从容。如果盘前没想清楚，就不要在盘中仓促决定。—— 宁可错过，不可做错。
