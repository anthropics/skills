**顶层政策雷达（每次分析必扫）：**

- 🏦 **货币政策**：央行降准/降息、MLF/LPR调整、汇率政策、流动性投放（逆回购）
- 💰 **财政政策**：专项债发行节奏、财政支出方向、减税降费新政

- 🔍 **监管政策**：证监会表态（重要！）、北交所/科创板政策、分红监管要求
- 🌐 **外部政策**：中美关系、关税政策、地缘政治风险等级

**政策解读框架（四象限）：**
```
              政策强度
                 强
                 |
    持续性强 ——— • ——— 持续性弱
                 |
                 弱

• 强+强: 政策大牛（最高权重，可重仓）
• 强+弱: 政策催化（短线机会，快进快出）
• 弱+强: 静待政策（等待加码信号）
• 弱+弱: 政策真空（规避或小仓位试探）
```

**关键政策识别清单（A股独有信号）：**
- 「 **新提法首次出现** 」→ 极强信号，立即关联ETF
- 「 **一把手表态** 」→ 落地概率>80%，高确定性机会
- 「 **会议召开频率变化** 」→ 非常规高频会议 = 非常规动作
- 「 **万亿级资金部署** 」→ 长达6-18个月主线机会
- 「 **约谈/暂停/叫停** 」→ 行业利空，相关ETF立即降权

### 2.2 技术形态分析（A股实用系统）

**A股高效技术指标组合：**

**A股独特技术要素：**
- **龙头效应**：ETF核心成分股龙头的技术形态权重 > ETF指数本身

- **涨停溢价**：高位涨停≠强势，低位涨停（缩量堆板）=主力吸筹信号
- **缺口意义**：向上跳空缺口是支撑，向下跳空缺口是压力
- **节前节后规律**：节前3日资金撤离，节后3日情绪回暖

### 2.3 基本面数据（A股定价锚点）

**估值锚（优先使用相对估值）：**
- **PE分位数**：当前PE在近5年/10年的百分位（<20%分位=低估区，>80%=高估区）
- **PB分位数**：金融/资源类行业更参考PB
- **股息率**：>3%具备安全边际，>4%防御价值显著
- **ERP（股权风险溢价）**：ERP = 1/PE - 10年国债收益率，ERP>3.5%=配置价值高

**A股基本面独有分析点：**

- **商誉/应收账款**：高商誉行业ETF风险折价
- **财报季节奏**：1月（三季报末）、4月（年报）、8月（半年报）是重要时间窗口
- **ROE趋势**：ROE改善中的行业 > 高ROE但下滑的行业

### 4.3 止盈止损纪律（铁则）

**止损规则（不得突破）：**
```
硬止损:
  - 建仓后跌破设定止损位 → 当日收盘前无条件清仓（T+1可执行）
  - 综合评分从正转负 → 次日减仓50%
  - 政策发生180°逆转 → 即日清仓，不等后续确认

动态止损（止盈后回撤保护）:
  - 盈利>15%: 将止损线移至成本价（保本）
  - 盈利>30%: 将止损线移至+10%（锁定部分收益）
  - 盈利>50%: 分批止盈30%，将止损线移至+25%
```

**止盈规则（避免贪婪）：**
```
分批止盈策略（3段式）:
  - 目标价第一阶段（+15%~20%）: 止盈持仓的30%
  - 目标价第二阶段（+30%~40%）: 再止盈持仓的30%
  - 剩余40%: 跟随趋势，动态止损保护，直到止损触发或政策逆转

事件驱动止盈:
  - 政策利好正式落地 → 「买预期卖事实」，止盈50%
  - 板块情绪过热（换手率超历史90%分位）→ 减仓30-50%
  - 核心逻辑发生根本性改变 → 无条件全部止盈
```
