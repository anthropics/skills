# Skills Collection

这是 bufan 个人收集的各种好用的 skill 集合。

## 📚 关于

本仓库包含了我在日常开发和使用中收集整理的各类实用技能（skills），涵盖了前端设计、文档处理、图像生成、数据分析、自动化工具等多个领域。

## 🎯 技能分类

### 前端与设计
- **frontend-design** - 前端设计相关技能
- **d2c-figma** - Figma 设计转代码
- **canvas-design** - Canvas 设计工具
- **web-asset-generator** - Web 资源生成器
- **theme-factory** - 主题工厂

### 文档处理
- **docx** - Word 文档处理
- **pptx** - PowerPoint 演示文稿处理
- **pdf** - PDF 文档处理
- **doc-coauthoring** - 文档协作

### 内容创作
- **blog-writer** - 博客写作助手
- **internal-comms** - 内部通讯工具
- **art-style-creator** - 艺术风格创建器
- **algorithmic-art** - 算法艺术生成

### 数据分析与爬虫
- **smallnest-crawl4ai** - AI 爬虫工具
- **xlsx** - Excel 表格处理
- **ux-competitive-analysis** - UX 竞品分析

### 图像与媒体
- **gemini-imagegen** - Gemini 图像生成
- **slack-gif-creator** - Slack GIF 创建器
- **scientific-schematics** - 科学图表生成
- **image-template-extraction** - 图像模板提取

### 开发工具
- **loki-mode** - Loki 模式开发工具
- **mcp-builder** - MCP 构建器
- **skill-creator** - 技能创建器
- **webapp-testing** - Web 应用测试
- **web-artifacts-builder** - Web 工件构建器

### 其他
- **obsidian-skills** - Obsidian 技能集合
- **brand-guidelines** - 品牌指南

## 🚀 使用说明

每个技能目录下通常包含：
- `SKILL.md` - 技能说明文档
- `scripts/` - 相关脚本文件
- `references/` - 参考文档
- `README.md` - 项目说明（部分技能）
