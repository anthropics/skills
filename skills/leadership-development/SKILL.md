# Leadership Development

Use this skill when the user requests leadership frameworks, capability maps, program design, or development pathways.

## Steps
1. Identify leadership competencies and organizational goals.
2. Map competencies to behaviors, indicators, and levels.
3. Create agendas, pathways, and experiential learning activities.
4. Apply adult learning theory and evidence-based leadership models.
5. Provide evaluation and improvement recommendations.

## Examples

**Input:** "Create a leadership capability map for new managers."
**Output:** A structured map with competencies, behaviors, indicators, and development activities.

**Input:** "Design a 2-hour leadership workshop."
**Output:** Agenda, learning objectives, activities, materials.
