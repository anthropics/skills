# AgentLux

## Installation
```bash
npm install -g agentlux
```

## Description
An open-source AgentSkill that imbues autonomous AI vision models with the soul of a Leica and the geometric discipline of Henri Cartier-Bresson. It re-crops user images using mathematical precision based on `gpt-4o` Vision and applies an unmistakable M10 color science—all with zero disk footprint.
