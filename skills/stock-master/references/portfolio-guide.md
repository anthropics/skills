# Excel 持仓管理指南

Stock Master 提供完整的 Excel 持仓管理功能，帮助你跟踪投资组合。

## 快速开始

### 创建持仓表格

在 Claude 中说：
```
创建持仓表格
```

系统会自动创建包含以下 Sheet 的 Excel 文件：

## 表格结构

### Sheet 1: 持仓

| 列名 | 说明 | 填写方式 |
|------|------|----------|
| 股票代码 | 股票代码 | 手动填写 |
| 股票名称 | 公司名称 | 手动填写 |
| 持仓数量 | 当前持有股数 | 手动填写 |
| 买入均价 | 平均成本价 | 手动填写 |
| 当前价格 | 实时价格 | **自动更新** |
| 市值 | 持仓数量 × 当前价格 | **自动计算** |
| 盈亏金额 | 市值 - 成本 | **自动计算** |
| 盈亏比例 | 盈亏百分比 | **自动计算** |
| 买入日期 | 建仓日期 | 手动填写 |
| 备注 | 备注信息 | 手动填写 |

**股票代码格式**：
- 美股：`AAPL`, `TSLA`, `GOOGL`
- 港股：`0700.HK`, `9988.HK`
- A股：`600519.SS`（上海）, `000001.SZ`（深圳）

### Sheet 2: 交易记录

| 列名 | 说明 |
|------|------|
| 日期 | 交易日期 |
| 股票代码 | 交易的股票 |
| 操作 | 买入 / 卖出 |
| 数量 | 交易股数 |
| 价格 | 成交价格 |
| 金额 | 交易总额 |
| 手续费 | 交易费用 |
| 备注 | 交易备注 |

### Sheet 3: 账户汇总

自动汇总的账户信息：
- 总投入
- 当前市值
- 总盈亏
- 收益率
- 最后更新时间

### Sheet 4: 使用说明

内置的使用指南。

## 使用命令

### 分析持仓
```
分析我的持仓
```
Claude 会：
1. 读取你的持仓数据
2. 获取每只股票的实时价格
3. 更新 Excel 中的价格和盈亏
4. 给出每只股票的技术分析建议

### 更新价格
```
更新持仓价格
```
只更新价格，不做分析。

### 持仓建议
```
我的持仓哪些该卖了？
```
根据技术指标给出持仓调整建议。

## 做T操作说明

持仓表只记录"结果"，不需要记录每次做T操作。

**示例**：
- TSLA 做了几轮T，最终持仓 100股，均价 $380
- 直接填：数量=100，均价=380
- 均价可从券商 App 查看

如需追踪每次操作，可在"交易记录"表记录。

## 配置文件

### 默认路径

系统按以下顺序查找持仓文件：
1. 环境变量 `PORTFOLIO_PATH`
2. `config.json` 中的 `portfolio_path`
3. 当前目录的 `my_portfolio.xlsx`

### 自定义路径

创建 `config.json`：
```json
{
  "portfolio_path": "/your/custom/path/portfolio.xlsx"
}
```

或设置环境变量：
```bash
export PORTFOLIO_PATH="/your/custom/path/portfolio.xlsx"
```

## 常见问题

### Q: 如何处理分红？
A: 分红后股价会除权，建议在备注中记录分红信息。

### Q: 多个账户怎么管理？
A: 可以创建多个 Excel 文件，分析时指定路径：
```
分析 /path/to/账户1.xlsx 的持仓
```

### Q: 港股如何处理汇率？
A: 目前按港币计算，建议在备注中记录汇率。
