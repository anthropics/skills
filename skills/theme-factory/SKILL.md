# 主题工厂技能

此技能提供精心策划的专业字体和颜色主题集合，每个主题都有精心挑选的调色板和字体配对。一旦选择了主题，就可以将其应用于任何工件。

## 目的

为演示幻灯片组应用一致的专业样式，请使用此技能。每个主题包括：
- 带有十六进制代码的连贯调色板
- 用于标题和正文的互补字体配对
- 适合不同上下文和受众的独特视觉标识

## 使用说明

要为幻灯片组或其他工件应用样式：

1. **显示主题展示**：显示 `theme-showcase.pdf` 文件，允许用户直观地查看所有可用主题。不要对其进行任何修改；仅显示文件以供查看。
2. **询问他们的选择**：询问要将哪个主题应用于幻灯片组
3. **等待选择**：获得对所选主题的明确确认
4. **应用主题**：选择主题后，将所选主题的颜色和字体应用于幻灯片组/工件

以下 10 个主题在 `theme-showcase.pdf` 中展示：

1. **Ocean Depths** - 专业而宁静的海洋主题
2. **Sunset Boulevard** - 温暖而充满活力的日落色彩
3. **Forest Canopy** - 自然而接地的大地色调
4. **Modern Minimalist** - 干净而现代的灰度
5. **Golden Hour** - 丰富而温暖的秋季调色板
6. **Arctic Frost** - 凉爽而清脆的冬季灵感主题
7. **Desert Rose** - 柔和而精致的尘土色调
8. **Tech Innovation** - 大胆而现代的技术美学
9. **Botanical Garden** - 清新而有机的花园色彩
10. **Midnight Galaxy** - 戏剧性而宇宙的深色调

## 主题详细信息

每个主题都在 `themes/` 目录中定义，包含完整的规范，包括：
- 带有十六进制代码的连贯调色板
- 用于标题和正文的互补字体配对
- 适合不同上下文和受众的独特视觉标识

## 应用流程

选择首选主题后：
1. 从 `themes/` 目录读取相应的主题文件
2. 在整个幻灯片组中一致地应用指定的颜色和字体
3. 确保适当的对比度和可读性
4. 在所有幻灯片上保持主题的视觉标识

## 创建您自己的主题
为了处理现有主题都不适用于工件的情况，请创建自定义主题。根据提供的输入，生成类似于上述主题的新主题。给出一个描述字体/颜色组合代表内容的类似主题名称。使用提供的任何基本描述来选择适当的颜色/字体。生成主题后，显示以供审查和验证。之后，按照上述描述应用主题。
