# Universal Voice Rules

These rules apply across all creators regardless of their specific voice profile.
They represent the baseline of good, human-sounding writing that this skill enforces.

---

## Clarity Rules

- **Say it once, say it well.** Don't repeat the same point in different words within the same paragraph.
- **Cut throat-clearing.** Remove openers like "In today's post I want to talk about..." — start with the actual thing.
- **Prefer active voice.** "We built this" not "This was built by us."
- **Short sentences win.** If a sentence has more than two commas, split it.
- **Avoid nominalisations.** "We decided" not "We made a decision." "She leads" not "She provides leadership."

---

## Authenticity Rules

- **No empty affirmations.** Remove "Great question!", "Absolutely!", "Certainly!" from the start of responses.
- **No hedge stacking.** Pick one hedge if needed — not "It might potentially seem like it could possibly be..."
- **No filler compliments.** "This is a thought-provoking topic" adds nothing. Cut it.
- **Write like a human.** Avoid phrases AI models default to: "dive into", "delve into", "it's worth noting", "in conclusion", "in summary", "I hope this helps."
- **Specific beats vague.** "3 years building SaaS tools" beats "years of experience in the tech industry."

---

## Tone Consistency Rules

- **Match the medium.** A tweet should sound like a tweet. A newsletter should sound like a newsletter. Don't write LinkedIn prose for Instagram.
- **Match the energy.** If the topic is urgent or exciting, the writing should reflect that. If it's reflective, slow down the sentence rhythm.
- **Consistency across pieces.** A creator who sounds warm and direct in their newsletter should not sound cold and distant in their bio.

---

## Banned Words (Universal)

These words are banned across all creator voices unless they are part of a creator's documented signature style:

| Word/Phrase | Replace with |
|---|---|
| Leverage (as a verb) | Use / apply / tap into |
| Synergy | Collaboration / teamwork |
| Ecosystem | Community / network / world |
| Unlock | Access / open up / discover |
| Empower | Help / enable / give people the tools to |
| Dive into | Explore / look at / walk through |
| Delve into | Explore / dig into |
| Game-changer | Be specific about what changed and how |
| Holistic | Complete / full / end-to-end |
| Robust | Strong / reliable / solid |
| Seamless | Smooth / effortless — or cut entirely |
| Transformative | Be specific about the transformation |
| It's worth noting | Just note it. Cut the preamble. |
| In conclusion | Just conclude. Cut the label. |
| In today's fast-paced world | Cut entirely. Start with the real point. |

---

## Structural Rules

- **Front-load value.** The most important point goes first — not buried in paragraph three.
- **Break up walls of text.** No paragraph longer than 4 lines for online content.
- **Use white space intentionally.** A one-line paragraph can be powerful. Use it sparingly.
- **End with direction.** Every piece of content should leave the reader knowing what to think, feel, or do next.
