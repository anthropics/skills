# Catppuccin Mocha

The wildly popular pastel dark theme — soothing mauve, pink, and blue.

## Color Palette

- **Base**: `#1E1E2E` - Base / background
- **Blue**: `#89B4FA` - Surface
- **Mauve**: `#CBA6F7` - Muted accent
- **Pink**: `#F5C2E7` - Accent
- **Green**: `#A6E3A1` - Bright accent
- **Text**: `#CDD6F4` - Foreground / text

## Typography

- **Headers**: JetBrains Mono (use JetBrains Mono Bold for emphasis)
- **Body Text**: Inter

## Best Used For

Cozy developer UIs, community tools, friendly technical presentations.
