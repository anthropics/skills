# Monokai

The classic Sublime palette: warm charcoal with hot pink, lime, and cyan.

## Color Palette

- **Background**: `#272822` - Base / background
- **Pink**: `#F92672` - Surface
- **Orange**: `#FD971F` - Muted accent
- **Cyan**: `#66D9EF` - Accent
- **Green**: `#A6E22E` - Bright accent
- **Foreground**: `#F8F8F2` - Foreground / text

## Typography

- **Headers**: JetBrains Mono (use JetBrains Mono Bold for emphasis)
- **Body Text**: Inter

Developer brands, vibrant code demos, energetic technical talks.
