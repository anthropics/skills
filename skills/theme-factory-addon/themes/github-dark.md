# GitHub Dark

GitHub's own dark mode — the syntax colors powering github.com.

## Color Palette

- **Canvas**: `#0D1117` - Base / background
- **Red**: `#F85149` - Surface
- **Green**: `#3FB950` - Muted accent
- **Blue**: `#58A6FF` - Accent
- **Purple**: `#BC8CFF` - Bright accent
- **Foreground**: `#C9D1D9` - Foreground / text

## Typography

- **Headers**: JetBrains Mono (use JetBrains Mono Bold for emphasis)
- **Body Text**: Inter

## Best Used For

Open-source projects, developer platforms, code-review and docs UIs.

> Category: Editor Classics (Dark)  ·  Mode: dark
