# Perfectly Peach

Soft peach warmed by deep charcoal grounding.

## Color Palette

- **Peach Cream**: `#FFF2EE` - Base / background
- **Lilac Gray**: `#D9CED6` - Light neutral
- **Apricot**: `#F9C7A1` - Soft accent
- **Terracotta**: `#D1835C` - Mid-tone accent
- **Mauve Slate**: `#60555D` - Deep accent
- **Charcoal**: `#303437` - Anchor / text

## Typography

- **Headers**: Marcellus (use Marcellus Bold for emphasis)
- **Body Text**: Mulish

## Best Used For

Lifestyle brands, boutique services, warm editorial layouts.
