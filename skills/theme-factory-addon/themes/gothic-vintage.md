# Gothic Vintage

Dramatic vintage contrast — oxblood and forest on bone.

## Color Palette

- **Bone White**: `#FFFFFF` - Base / background
- **Silver**: `#D9D9D9` - Light neutral
- **Parchment**: `#DFC9B4` - Soft accent
- **Forest**: `#263F2C` - Mid-tone accent
- **Oxblood**: `#4D0011` - Deep accent
- **Black**: `#000000` - Anchor / text

## Typography

- **Headers**: Playfair Display (use Playfair Display Bold for emphasis)
- **Body Text**: EB Garamond

## Best Used For

Heritage and luxury brands, moody editorials, dramatic event decks.
