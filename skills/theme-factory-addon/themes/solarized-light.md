# Solarized Light

The warm-paper counterpart to Solarized Dark — cream base, calm accents.

## Color Palette

- **Base3**: `#FDF6E3` - Base / background
- **Green**: `#859900` - Light neutral
- **Cyan**: `#2AA198` - Soft accent
- **Blue**: `#268BD2` - Mid-tone accent
- **Body**: `#586E75` - Deep accent
- **Red**: `#DC322F` - Anchor / text

## Typography

- **Headers**: Fira Code (use Fira Code Bold for emphasis)
- **Body Text**: Inter

## Best Used For

Readable docs, academic writing, easy-on-the-eyes light presentations.
