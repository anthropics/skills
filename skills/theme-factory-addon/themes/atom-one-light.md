# Atom One Light

Atom's light theme — bright neutral canvas with friendly saturated accents.

## Color Palette

- **Canvas**: `#FAFAFA` - Base / background
- **Green**: `#50A14F` - Light neutral
- **Blue**: `#4078F2` - Soft accent
- **Red**: `#E45649` - Mid-tone accent
- **Purple**: `#A626A4` - Deep accent
- **Ink**: `#383A42` - Anchor / text

## Typography

- **Headers**: Fira Code (use Fira Code Bold for emphasis)
- **Body Text**: Inter

## Best Used For

Approachable developer docs, tutorials, clean product UIs.
