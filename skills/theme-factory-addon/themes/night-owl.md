# Night Owl

Sarah Drasner's theme tuned for low light — deep navy with coral and teal.

## Color Palette

- **Background**: `#011627` - Base / background
- **Coral**: `#F78C6C` - Surface
- **Purple**: `#C792EA` - Muted accent
- **Blue**: `#82AAFF` - Accent
- **Green**: `#C5E478` - Bright accent
- **Foreground**: `#D6DEEB` - Foreground / text

## Typography

- **Headers**: JetBrains Mono (use JetBrains Mono Bold for emphasis)
- **Body Text**: Inter

## Best Used For

Accessible dark UIs, late-night dev tools, low-light presentations.

> Category: Editor Classics (Dark)  ·  Mode: dark
