# One Dark Pro

Atom's beloved One Dark: balanced contrast with soft blue, red, and green.

## Color Palette

- **Background**: `#282C34` - Base / background
- **Red**: `#E06C75` - Surface
- **Purple**: `#C678DD` - Muted accent
- **Blue**: `#61AFEF` - Accent
- **Foreground**: `#ABB2BF` - Bright accent
- **Green**: `#98C379` - Foreground / text

## Typography

- **Headers**: JetBrains Mono (use JetBrains Mono Bold for emphasis)
- **Body Text**: Inter

## Best Used For

Developer products, code-forward presentations, engineering docs.
