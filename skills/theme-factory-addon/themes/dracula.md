# Dracula

The iconic dark theme: deep slate background with vivid purple, pink, and cyan.

## Color Palette

- **Background**: `#282A36` - Base / background
- **Pink**: `#FF79C6` - Surface
- **Purple**: `#BD93F9` - Muted accent
- **Green**: `#50FA7B` - Accent
- **Cyan**: `#8BE9FD` - Bright accent
- **Foreground**: `#F8F8F2` - Foreground / text

## Typography

- **Headers**: JetBrains Mono (use JetBrains Mono Bold for emphasis)
- **Body Text**: Inter

## Best Used For

Developer tools, dark dashboards, technical decks, hacker-aesthetic slides.

> Category: Editor Classics (Dark)  ·  Mode: dark
