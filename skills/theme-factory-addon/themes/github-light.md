# GitHub Light

GitHub's default light mode — clean white with crisp functional accents.

## Color Palette

- **Canvas**: `#FFFFFF` - Base / background
- **Purple**: `#8250DF` - Light neutral
- **Green**: `#1A7F37` - Soft accent
- **Blue**: `#0969DA` - Mid-tone accent
- **Red**: `#CF222E` - Deep accent
- **Ink**: `#24292F` - Anchor / text

## Typography

- **Headers**: Fira Code (use Fira Code Bold for emphasis)
- **Body Text**: Inter

## Best Used For

Documentation sites, developer platforms, clean technical decks.

> Category: Editor Light  ·  Mode: light
