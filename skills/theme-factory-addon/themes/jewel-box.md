# Jewel Box

Jewel tones — rose, teal, and olive on cream.

## Color Palette

- **Cream**: `#E7DECD` - Base / background
- **Sea Blue**: `#62929E` - Light neutral
- **Raspberry**: `#A6445D` - Accent
- **Olive**: `#393A10` - Deep accent
- **Deep Teal**: `#142F40` - Anchor / text

## Typography

- **Headers**: Bodoni Moda (use Bodoni Moda Bold for emphasis)
- **Body Text**: Montserrat

## Best Used For

Boutique luxury, artful brands, rich colourful presentations.

> Category: Luxury & Sophisticated  ·  Mode: light
