# Coca-Cola

The timeless Coca-Cola red with crisp black and white.

## Color Palette

- **White**: `#FFFFFF` - Base / background
- **Coca-Cola Red**: `#F40009` - Light accent
- **Crimson**: `#B30000` - Deep accent
- **Black**: `#000000` - Anchor / text

## Typography

- **Headers**: Montserrat (use Montserrat Bold for emphasis)
- **Body Text**: Work Sans

## Best Used For

Beverage and FMCG, classic mass-market campaigns, bold retail.
