# Twitter X

Twitter's bright blue and the X-era near-black, with cool UI grays.

## Color Palette

- **White**: `#FFFFFF` - Base / background
- **Light Gray**: `#E1E8ED` - Light neutral
- **Twitter Blue**: `#1DA1F2` - Accent
- **Slate**: `#657786` - Deep accent
- **Ink Black**: `#14171A` - Anchor / text

## Typography

- **Headers**: Montserrat (use Montserrat Bold for emphasis)
- **Body Text**: Work Sans

## Best Used For

News and media, real-time products, clean tech presentations.
