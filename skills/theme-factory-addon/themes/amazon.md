# Amazon

Amazon orange and squid-ink navy with a trusted link blue.

## Color Palette

- **White**: `#FFFFFF` - Base / background
- **Light Gray**: `#F2F3F3` - Light neutral
- **Amazon Orange**: `#FF9900` - Accent
- **Link Blue**: `#146EB4` - Deep accent
- **Squid Ink**: `#232F3E` - Anchor / text

## Typography

- **Headers**: Montserrat (use Montserrat Bold for emphasis)
- **Body Text**: Work Sans

## Best Used For

E-commerce and logistics, trusted marketplace UIs, retail decks.
