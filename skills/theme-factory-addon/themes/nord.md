# Nord

Arctic, north-bluish palette — calm frost blues over polar night.

## Color Palette

- **Polar Night**: `#2E3440` - Base / background
- **Aurora Red**: `#BF616A` - Surface
- **Deep Frost**: `#5E81AC` - Muted accent
- **Aurora Green**: `#A3BE8C` - Accent
- **Frost**: `#88C0D0` - Bright accent
- **Snow Storm**: `#ECEFF4` - Foreground / text

## Typography

- **Headers**: JetBrains Mono (use JetBrains Mono Bold for emphasis)
- **Body Text**: Inter

## Best Used For

Calm developer UIs, minimalist tech decks, design-system documentation.
