## Usage Instructions
使用说明

### How to Use This Glossary
如何使用此术语表

1. **Consistency**: Always use the translations listed here
   一致性：始终使用此处列出的翻译

2. **Lookup**: Check here before translating technical terms
   查找：翻译技术术语前在此处查找

3. **Update**: Add new terms as you encounter them
   更新：在遇到新术语时添加

4. **Share**: Reference this glossary in team conversations
   分享：在团队对话中参考此术语表

### When Translating
翻译时

- ✓ Use standardized terms from this glossary
  使用此术语表中的标准化术语
- ✓ Maintain consistency with previous translations
  保持与之前翻译的一致性
- ✓ Document any new technical terms here
  在此处记录任何新的技术术语
- ✗ Don't deviate from established translations
  不要偏离已建立的翻译
- ✗ Don't invent new translations for existing terms
  不要为现有术语发明新翻译

## Adding New Terms
添加新术语

When you encounter a technical term that's not in this glossary:

遇到此术语表中没有的技术术语时：

1. Research the standard translation in technical communities
   在技术社区中研究标准翻译

2. Consider context and common usage
   考虑背景和常见用法

3. Choose the most appropriate translation
   选择最合适的翻译

4. Add it to this glossary with usage context
   将其添加到此术语表中，并说明使用背景

5. Share with team members for approval
   与团队成员分享以获得批准
