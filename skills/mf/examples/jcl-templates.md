# JCL Templates

Ready-to-use JCL templates for common tasks.

## Compile COBOL

```jcl
//COMPILE  JOB (ACCT),'COBOL COMPILE',CLASS=A,MSGCLASS=X,
//             MSGLEVEL=(1,1),NOTIFY=&SYSUID
//*
//* Compile and link a COBOL program
//*
//STEP1    EXEC IGYWCL
//COBOL.SYSIN DD DSN=&SYSUID..SOURCE(MYPROG),DISP=SHR
//LKED.SYSLMOD DD DSN=&SYSUID..LOAD(MYPROG),DISP=SHR
```

## Copy PDS Members

```jcl
//COPYPDS  JOB (ACCT),'COPY PDS',CLASS=A,MSGCLASS=X,
//             NOTIFY=&SYSUID
//*
//STEP1    EXEC PGM=IEBCOPY
//SYSPRINT DD SYSOUT=*
//INDD     DD DSN=INPUT.PDS,DISP=SHR
//OUTDD    DD DSN=OUTPUT.PDS,DISP=SHR
//SYSIN    DD *
  COPY INDD=INDD,OUTDD=OUTDD
  SELECT MEMBER=(MEM1,MEM2,MEM3)
/*
```

## Define VSAM Cluster

```jcl
//DEFVSAM  JOB (ACCT),'DEF VSAM',CLASS=A,MSGCLASS=X,
//             NOTIFY=&SYSUID
//*
//STEP1    EXEC PGM=IDCAMS
//SYSPRINT DD SYSOUT=*
//SYSIN    DD *
  DEFINE CLUSTER (NAME(MY.VSAM.CLUSTER) -
         INDEXED -
         KEYS(10 0) -
         RECORDSIZE(100 200) -
         SHAREOPTIONS(2 3) -
         CYLINDERS(10 5)) -
    DATA (NAME(MY.VSAM.CLUSTER.DATA)) -
    INDEX (NAME(MY.VSAM.CLUSTER.INDEX))
/*
```

## Multi-Step Job with Condition Codes

```jcl
//MULTI    JOB (ACCT),'MULTI STEP',CLASS=A,MSGCLASS=X,
//             NOTIFY=&SYSUID
//*
//STEP1    EXEC PGM=PROG1
//...
//*
//STEP2    EXEC PGM=PROG2,COND=(4,LT)
//*        Run if STEP1 CC < 4
//...
//*
//STEP3    EXEC PGM=PROG3,COND=(0,NE,STEP1)
//*        Run only if STEP1 CC = 0
//...
```
