# Question Bank — Meta PM Mock Interview

Reference this file for interview questions organized by type and difficulty tier.

---

## Product Sense Questions

### Tier 1 — Warm-Up
- Design a feature to help Facebook Groups become more active
- How would you improve Instagram DMs?
- Design a product to help elderly users get more value from WhatsApp

### Tier 2 — Standard
- Design a new Facebook feature for small business owners
- How would you improve Facebook Marketplace?
- Design a feature to help new users find their community on Instagram faster
- How would you redesign Facebook Events?

### Tier 3 — Stretch
- Design a Meta product for the next billion internet users
- How would you use Meta's social graph to improve healthcare outcomes?
- Design a VR product for remote collaboration using Meta Quest
- How would you build a new monetization surface for WhatsApp?

---

## Analytical Thinking Questions

### Goal-Setting / Metrics
- What metrics would you use to measure the health of Facebook Groups?
- How would you set goals for the Facebook Marketplace team?
- What is the North Star Metric for Instagram Reels?
- How would you measure the success of WhatsApp Business?
- What metrics matter most for the Meta Quest platform?

### Debugging / Root Cause Analysis
- Facebook DAU dropped 10% last week — walk me through how you'd investigate
- Instagram Story views are down 15% — what would you do?
- WhatsApp message send rate dropped significantly in India — what's your process?
- Facebook ad click-through rate declined 20% this quarter — how do you diagnose this?

### Tradeoff Questions
- Should Facebook prioritize short-form video (Reels) or long-form (Watch)? How do you decide?
- Meta is deciding between monetizing WhatsApp via ads vs. a business API — how do you evaluate?
- Instagram is considering removing public like counts permanently — walk me through the tradeoffs
- Should Meta invest more in AI features or VR/AR for the next 3 years?

## Difficulty Guide

| Tier | Characteristics | Candidate Profile |
|------|----------------|-------------------|
| Tier 1 | Familiar product, clear user base, obvious pain points | Early prep / first mock |
| Tier 2 | Requires scoping decisions, less obvious user segments | Active prep, intermediate |
| Tier 3 | Ambiguous scope, cross-platform thinking, business model complexity | Final prep, advanced |
