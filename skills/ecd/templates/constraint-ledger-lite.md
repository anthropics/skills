# Constraint Ledger Lite Template `[L1]`

## Goal

Capture minimal shared truth for L1 tasks. Lighter than the Full constraint ledger.

## Must Capture

- `retained_goal`: what the workflow currently believes the user actually wants
- `confirmed_facts`: only facts supported by user answers, repo reality, or source documents
- `frozen_constraints`: constraints that `/code` must not silently reinterpret
- `risks`: top 1-3 high-impact uncertainties

## Optional (L1 may skip)

- `challenged_claims`
- `dropped_options`
- `dependency_chain`
- `verification_semantics`
