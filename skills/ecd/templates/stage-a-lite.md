# Stage A-Lite Template `[L1]`

## Goal

Capture the minimum frozen semantics needed for L1 execution. Max 3 clarification questions.

## Must Capture

- `user_stated_request`: the original request verbatim
- `reframed_request`: the user's confirmed intent after clarification (max 3 questions)
- `retained_scope`: what will be delivered in this pass
- `excluded_scope`: what will NOT be delivered
- `acceptance_checks`: 3-5 concrete, verifiable checks
- `key_assumptions`: assumptions that affect implementation
- `frozen_for_code`: decisions code must treat as fixed truth

## Exit Gate

- The reframed goal and scope boundary are clear enough for execution
- The user has explicitly approved the compact approval pack
- Remaining unknowns are genuinely low-impact implementation details
