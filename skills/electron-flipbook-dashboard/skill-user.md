# Flipbook Dashboard — 사용 & 배포 가이드

## 앱 설치

### 처음 설치할 때
1. `Flipbook Dashboard Setup 1.0.0.exe` 더블클릭
2. 설치 경로 선택 후 설치 완료
3. 바탕화면 + 시작메뉴에 아이콘 자동 생성
4. `.flipbook` 파일 연결 자동 등록 (더블클릭으로 바로 열림)

### 설치 파일 위치
```
C:\dev\flipbook-dashboard\release\Flipbook Dashboard Setup 1.0.0.exe
```

---

## 앱 실행

| 방법 | 설명 |
|------|------|
| 바탕화면 아이콘 더블클릭 | 가장 빠름 |
| 시작메뉴 → "Flipbook" 검색 | 아이콘 없을 때 |
| `.flipbook` 파일 더블클릭 | 앱 실행 + 데이터 가져오기 동시에 |

---

## 뷰어 모드 (배포받은 사람)

**비밀번호 없이 바로 책을 볼 수 있습니다.**

```
.flipbook 파일 더블클릭
→ 앱 자동 실행
→ 책 목록 즉시 표시 ✅ (비밀번호 입력 불필요)
```

책을 클릭하면 책장 넘기는 애니메이션으로 읽을 수 있습니다.

---

## 어드민 로그인 (관리자만)

앱 우상단 🔐 아이콘 클릭 → 비밀번호 입력.
앱을 처음 켜면 비밀번호 설정 화면이 나옵니다.

**비밀번호 분실 시:** 설정 시 발급된 복구 코드 사용 (`XXXX-XXXX-XXXX-XXXX` 형식)

## PDF 책 추가 (어드민 전용)

**방법 1 — 버튼으로 추가**
```
어드민 로그인 → 상단 "+ 책 추가" 클릭 → PDF 파일 선택 (여러 개 동시 선택 가능)
```

**방법 2 — 드래그&드롭**
```
PDF 파일을 앱 위에 드래그해서 놓기 (여러 개 동시 가능)
```

- 같은 PDF를 다시 추가해도 중복 등록 안 됨 (자동 감지)
- 추가 후 7일간 책 카드에 **NEW** 뱃지 자동 표시
- 표지 이미지는 자동 생성됨

---

## 화면 꾸미기 (어드민 전용)

어드민 로그인 후 상단 패널에서 설정:

| 항목 | 방법 |
|------|------|
| 배경 색상 | 컬러 피커 클릭 |
| 배경 이미지 | "이미지" 버튼 → 이미지 파일 선택 |
| 캐릭터 | "업로드" 버튼 → 이미지 파일 선택 |
| 제목 | 헤더 텍스트 클릭 후 편집 |
| 열 수 | 그리드 컬럼 수 조절 |

---

## 프리셋 (설정 저장/불러오기, 어드민 전용)

현재 배경·캐릭터·제목 등 전체 설정을 이름 붙여 저장.
**교실마다 다른 설정을 빠르게 전환할 때 유용.**

```
📋 프리셋 버튼 클릭
→ 이름 입력 후 "현재 상태 저장"
→ 나중에 목록에서 "불러오기" 클릭
```

## 다른 PC에 배포하기

### 전달할 파일 (2개만)

```
📁 USB 또는 공유폴더
├── Flipbook Dashboard Setup 1.0.0.exe   ← 앱 설치용
└── flipbook-2025-01-15.flipbook          ← 책 + 설정 패키지
```

### 내보내기 (보내는 쪽, 어드민)
```
어드민 로그인 → 📦 내보내기 버튼 → 저장 위치 선택
→ .flipbook 파일 생성됨 (PDF + 이미지 + 설정 모두 포함)
```

### 가져오기 (받는 쪽) — 방법 2가지

**방법 A: 더블클릭 (권장)**
```
1. Setup.exe 설치
2. .flipbook 파일 더블클릭
→ 앱 자동 실행
→ 책 목록 즉시 표시 ✅ (비밀번호 불필요)
```

**방법 B: 앱에서 직접 (어드민)**
```
앱 실행 → 어드민 로그인 → 📂 가져오기 버튼 → .flipbook 파일 선택
→ 책 목록 즉시 표시 ✅
```

### 중복 처리
이미 있는 책은 자동으로 스킵됩니다.
완료 메시지: `✅ 가져오기 완료 (책 5권 추가, 2권 중복 스킵)`

## 자주 묻는 질문

**Q. .flipbook 파일을 더블클릭해도 아무것도 안 열려요**
→ Setup.exe로 앱을 먼저 설치해야 파일 연결이 등록됩니다.
→ 포터블 EXE 사용 중이라면: `.flipbook` 파일 우클릭 → 연결 프로그램 → Flipbook Dashboard 선택

**Q. 책이 안 보여요 (가져오기 후)**
→ 앱을 완전히 껐다가 다시 켜보세요.
→ 그래도 안 되면 어드민으로 로그인 → 📂 가져오기 버튼으로 다시 가져오기.

**Q. 책 표지가 안 보여요**
→ 책을 한 번 클릭해서 열면 표지가 자동 생성됩니다.

**Q. 비밀번호를 잊었어요**
→ 설치 시 발급받은 복구 코드(`XXXX-XXXX-XXXX-XXXX`)를 로그인 화면에 입력하세요.

**Q. 같은 PDF를 두 번 추가했는데 왜 하나만 있나요?**
→ 정상입니다. 중복 파일은 자동으로 걸러집니다.

**Q. 배포받은 PC에서 어드민 로그인이 되나요?**
→ 네. .flipbook 파일에 비밀번호 정보가 포함되어 있어 보내는 쪽과 같은 비밀번호로 로그인됩니다.

**Q. 포터블 EXE와 Setup.exe 차이가 뭔가요?**

| | Setup.exe (설치형) | Portable.exe (포터블) |
|--|--|--|
| 설치 필요 | ✅ | ❌ |
| 바탕화면 아이콘 | ✅ | ❌ |
| .flipbook 더블클릭 | ✅ 자동 | ❌ 수동 연결 필요 |
| 데이터 저장 위치 | C:\Users\...\AppData | EXE 옆 FlipbookData\ |
