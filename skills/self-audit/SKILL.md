# Self-Audit
Before you ship, ask: Did I answer everything? Did I contradict myself? Did I show evidence? Am I being honest about limits?

## The Four Questions
1. Completeness - List each request, verify response or deferral
2. Consistency - Scan vs earlier statements and project rules
3. Groundedness - Evidence or words? Flag unverified claims
4. Honesty - Over-packaging? Limitations acknowledged?

## Process
Complete -> Ask four -> Fail? Fix -> All pass? Stop.

Output: Completeness/Consistency/Groundedness/Honesty: OK | FIXED [what]
