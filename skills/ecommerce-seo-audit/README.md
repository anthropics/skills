# Ecommerce SEO Audit Skill

A comprehensive SEO audit skill specifically designed for ecommerce websites, covering product pages, collection pages, technical SEO, crawl optimization, and competitive analysis.

## Features

This skill provides 7 specialized audit types:

1. **Quick Technical Audit** - Crawlability, indexability, and schema validation
2. **Product Page Audit** - Deep optimization analysis for product listings
3. **Collection Page Audit** - Category and collection page SEO review
4. **Log File Analysis** - Crawl budget and Googlebot behavior insights
5. **Competitor Analysis** - Analyze top 5 ranking competitors for keywords
6. **Keyword Research & Mapping** - Discover opportunities and map keywords to pages
7. **Full Comprehensive Audit** - Complete audit covering all areas

## Usage

Use this skill when you need to:
- Audit ecommerce website SEO performance
- Optimize product or collection pages
- Analyze crawl budget and technical issues
- Research competitor SEO strategies
- Improve organic search rankings
