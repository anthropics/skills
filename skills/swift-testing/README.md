# swift-testing

A production-grade Agent Skill for generating iOS UI tests using **XCUITest (Swift)**.

## What it produces
- A test plan (steps + assertions)
- XCUITest Swift code
- Screen Objects (Page Object style)
- Required accessibility identifiers
- Flake-hardening guidance (waits/scroll/alerts)

## How to use (examples)
Ask:
- “Write an XCUITest for login success flow”
- “Refactor this flaky XCUITest to be stable”
- “Create screen objects for these UI tests”
