# MLOps / LLMOps 详解

> 从实验到生产的桥梁，决定 AI 项目能否持续迭代和创造价值。

## 📚 本章概述

本章深入探讨 MLOps（Machine Learning Operations）和 LLMOps（Large Language Model Operations）的核心概念、工具和最佳实践，帮助你构建可持续、可扩展的机器学习系统。

## 🎯 学习目标

完成本章学习后，你将能够：

- [ ] 理解 MLOps 的核心概念和价值
- [ ] 评估团队的 MLOps 成熟度等级
- [ ] 搭建完整的实验跟踪系统
- [ ] 实现模型版本管理和 Model Registry
- [ ] 设计和构建特征工程平台
- [ ] 构建 ML CI/CD Pipeline
- [ ] 实现模型监控和漂移检测
- [ ] 理解 LLMOps 的特殊挑战和解决方案
- [ ] 选择合适的 MLOps 工具栈

## 📝 快速开始

### 环境准备

```bash
# 安装核心工具
pip install mlflow wandb feast evidently

# 安装 LLMOps 工具
pip install langsmith langchain ragas
```

### 最小化 MLOps 实践

```python
import mlflow

# 1. 设置实验
mlflow.set_experiment("quickstart")

# 2. 记录实验
with mlflow.start_run():
    mlflow.log_param("learning_rate", 0.01)
    mlflow.log_metric("accuracy", 0.95)
    mlflow.log_artifact("model.pkl")

# 3. 查看结果
# mlflow ui --port 5000
```
