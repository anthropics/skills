## 路径 A：后端/系统工程师转型

### 你的优势

- ✅ 熟悉分布式系统、网络、存储
- ✅ 了解 K8s、容器化部署
- ✅ 有性能优化经验
- ✅ 理解高可用、容错设计

### 你需要补充

- ❌ ML/DL 基础概念
- ❌ GPU 架构和 CUDA
- ❌ 训练/推理流程

## 面试准备指南

### 常见面试题

#### GPU/CUDA

```
Q1: 解释 GPU 和 CPU 的主要区别？为什么 GPU 适合深度学习？
Q2: 什么是 Warp Divergence？如何避免？
Q3: 解释 Coalesced Memory Access 的重要性
Q4: Tensor Core 是什么？如何使用？
Q5: 比较 NVLink 和 PCIe 的区别和适用场景
```

#### 分布式训练

```
Q1: 解释数据并行和模型并行的区别
Q2: ZeRO 的三个阶段分别优化了什么？
Q3: 什么是 Ring AllReduce？画出通信流程
Q4: 分布式训练中如何处理 Gradient Accumulation？
Q5: 如何选择 DP/TP/PP 的并行度？
```

#### 推理优化

```
Q1: LLM 推理中 KV Cache 的作用是什么？
Q2: 解释 PagedAttention 的原理
Q3: Continuous Batching 如何提升吞吐？
Q4: INT8 量化会带来什么影响？如何评估？
Q5: Speculative Decoding 的原理是什么？
```

#### 系统设计

```
Q1: 设计一个支持多租户的 GPU 集群
Q2: 设计一个 LLM 推理服务的高可用架构
Q3: 如何实现模型的 A/B 测试？
Q4: 设计一个 ML Pipeline 系统
Q5: 如何优化 GPU 集群的利用率？
```

### 面试准备 Checklist

```
□ 能手写 DDP 训练代码

□ 能画出 Transformer 的并行切分方式
□ 能计算模型训练的显存需求
□ 能解释 vLLM 的核心优化
□ 能设计基本的 ML 平台架构
□ 做过至少一个完整的 AI Infra 项目
□ 读过至少 3 篇核心论文
```
