## 学习习惯养成

### 有效学习的 5 个原则

```
1. 主动学习
   - 带着问题学习
   - 边学边实践
   - 及时总结

2. 间隔重复
   - 定期复习
   - 建立知识关联
   - 使用 Anki 等工具

3. 输出倒逼输入
   - 写博客
   - 做分享
   - 教他人

4. 深度优先
   - 在一个领域深耕
   - 避免浅尝辄止
   - 建立专长

5. 保持好奇
   - 追踪前沿
   - 尝试新事物
   - 保持热情
```

### 知识管理工具

| 工具 | 用途 |
|------|------|
| Notion/Obsidian | 笔记管理 |
| Zotero | 论文管理 |
| Anki | 间隔重复 |
| GitHub | 代码管理 |

### 克服学习障碍

```
常见障碍和解决方法:

1. "没有时间"
   → 固定时间段，哪怕每天 30 分钟

2. "不知道学什么"
   → 选择一个路径，按计划执行

3. "学了就忘"
   → 实践+输出，间隔重复

4. "遇到困难就放弃"
   → 分解问题，寻求社区帮助

5. "感觉学不完"
   → 深度优先，先精通一个领域
```

## 职业发展建议

### AI Infra 职业路径

```
入门 (0-2年)
├── 学习基础知识
├── 完成入门项目
└── 建立技能基础

中级 (2-5年)
├── 深入专业领域
├── 参与大型项目
└── 建立行业影响力

高级 (5年+)
├── 技术专家/架构师
├── 技术管理
└── 创业/独立顾问
```

### 持续成长

```
技术成长:
- 跟踪前沿技术
- 参与开源项目
- 写博客/做分享

职业成长:
- 建立个人品牌
- 拓展人脉网络
- 寻找导师

个人成长:
- 保持学习热情
- 平衡工作生活
- 关注身心健康
```
