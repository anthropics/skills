# AI Infra 完整学习路线图

> 按读者背景提供分层学习计划，包含推荐资源、动手项目和面试准备。

## 学习目标

完成本章节学习后，你将能够：

- 🎯 根据自身背景选择最适合的学习路径
- 📊 评估自己在 AI Infra 各领域的技能水平
- 📚 系统地利用推荐资源进行学习
- 🛠️ 通过动手项目巩固所学知识
- 💼 为 AI Infra 相关面试做充分准备
- 🌱 建立持续学习的习惯和方法
