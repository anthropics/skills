# AI 训练基础知识

> 理解神经网络训练的核心原理，这是所有 AI Infra 知识的前置基础。

## 📚 本章概述

在深入分布式训练、推理优化等 AI Infra 技术之前，你需要先理解神经网络是怎么训练的。本章从最基础的概念讲起，用大量生活类比帮你建立扎实的直觉。

**核心比喻**：AI 训练 = 学厨师。不断循环"试做→评分→反馈→调整手法"，最终做出满分菜肴。

### 学习目标

- 理解神经网络的基本组成：神经元、层、权重、偏置
- 掌握训练循环的完整流程：前向传播 → 损失计算 → 反向传播 → 参数更新
- 理解梯度的含义、链式法则的直觉
- 了解学习率、过拟合、正则化等关键训练概念
- 为后续学习分布式训练、推理优化打下基础

### 前置知识

- 基本的数学直觉（不需要精通，文档里有详细类比）
- Python 基础（代码示例部分）

## 💡 核心概念速览

### 训练循环五步

| 步骤 | 做什么 | 厨师类比 |
|------|--------|---------|
| ① 数据准备 | 取一批训练样本 | 准备一批食材 |
| ② 前向传播 | 数据流过网络，得到预测 | 按手法做菜 |
| ③ 计算损失 | 量化预测和真实值的差距 | 师傅打分 |
| ④ 反向传播 | 计算每个参数的梯度 | 师傅逐项反馈 |
| ⑤ 参数更新 | 沿负梯度方向调整参数 | 根据反馈改进手法 |

### 关键概念对照表

| 概念 | 定义 | 生活类比 |
|------|------|---------|
| 参数 (W, b) | 网络中需要学习的数值 | 厨师的手法（放盐量、火候） |
| 梯度 | 参数对 loss 的影响方向和大小 | "盐放多了3分" |
| 学习率 | 每步调整多大 | 每次少放多少盐 |
| 损失函数 | 衡量预测有多差 | 师傅的评分标准 |
| 前向传播 | 输入→预测 | 做菜 |
| 反向传播 | loss→梯度 | 师傅追溯问题 |
| 过拟合 | 记住训练数据但泛化差 | 背答案的学生 |
| 正则化 | 防止过拟合的技术 | 考试不只出原题 |
