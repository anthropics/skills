# Program Evaluation

Use this skill for evaluation plans, dashboards, logic models, or data interpretation.

## Steps

1. Identify program goals and success indicators.
2. Apply Kirkpatrick levels and logic models.
3. Analyze qualitative and quantitative data.
4. Produce dashboards and evaluation summaries.
5. Recommend improvements based on findings.

## Example Prompts

"Create a logic model for a leadership program."
"Summarize survey data and identify trends."
"Design a Kirkpatrick Level 3 evaluation plan."
