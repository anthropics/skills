# ESSA / Title I Reporting — Visualization Guide

## Core Reporting Requirements

| Requirement | Visualization Approach |
|---|---|
| Proficiency by subgroup | Grouped horizontal bar; one bar per subgroup, reference line at target |
| Growth (SGP or VAM) | Scatter plot (prior vs. current score) or percentile band chart |
| Chronic absenteeism | Line trend with threshold annotation at 10% |
| Progress toward targets (AMOs) | Bullet chart or progress bar with target marker |
| Participation rate (≥95%) | Simple gauge or flag if below threshold |

## Required Subgroups (ESSA §1111)

Always offer to disaggregate by:
- All students
- Economically disadvantaged
- Students with disabilities (IDEA)
- English learners
- Major racial/ethnic groups (as defined by state — check state plan)
- Foster youth / homeless (where applicable)

**Privacy rule**: Suppress cells where N < 10 (federal minimum; some states use higher threshold). Show suppressed cells as "n/a" with a footnote, never as zero.

## Evidence Tier Labels

When referencing an intervention's effectiveness in a visualization:
- **Tier 1** — Strong evidence (RCT with significant positive findings)
- **Tier 2** — Moderate evidence (quasi-experimental)
- **Tier 3** — Promising evidence (pre-post, correlational)
- **Tier 4** — Demonstrates a rationale (logic model + prior research)

Label the tier on any effectiveness chart or dashboard card.

## Title I Specific

- Show use-of-funds alignment to allowable activities
- Annual progress reports must include comparison to state average
- School Improvement Plans (CSI/TSI) should show a 3-year trend minimum
