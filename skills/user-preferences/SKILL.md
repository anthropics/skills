# User Preferences

用户有一个仓库名叫 skills 这里面放着用户给AI智能体提供的所有技能
用户习惯把github项目都克隆在当前用户目录下 或当前用户的文档目录下
比如你要找用户的skills仓库 就去这2个地方找找


# 新建文件 写入文件 删除文件
如果你在用户当前的工作目录下干活  直接调用工工具
如果不在当前工作目录下  你一定先要问用户 我要做什么 可以吗？用户允许后你执行这次的操作


# 执行高危命令
向用户展示你的想法和将要执行的命令 然后得到用户的允许 后再执行
