# Keyword Research — Deep Dive Reference

## Keyword Types & Intent

### By Search Intent
| Intent | Example | Content Type |
|--------|---------|--------------|
| Informational | "what is SEO" | Blog post, guide |
| Navigational | "semrush login" | Landing page |
| Commercial | "best SEO tools" | Comparison, review |
| Transactional | "buy SEO software" | Product page |

### By Funnel Stage
- **TOFU** (Top of Funnel): High volume, low intent → educational content
- **MOFU** (Middle): Research-phase → comparison content
- **BOFU** (Bottom): High intent, lower volume → product/service pages

## Step-by-Step Keyword Research Workflow

### Step 1: Seed Keywords
- Start with 5–10 core terms describing your product/service
- Enter into **Keyword Magic Tool**
- Export all variations (can be 10k+)

### Step 2: Filter & Prioritize
Filter by:
- **KD (Keyword Difficulty)**: Start with 0–40 for new sites, 40–70 for established
- **Volume**: Minimum 100/month (or lower for niche B2B)
- **Intent**: Match to your content type
- **SERP Features**: Prioritize keywords with featured snippets you could win

### Step 3: Competitive Gap Analysis
- **Keyword Gap Tool**: Enter your domain + up to 4 competitors
- Filter: "Competitors rank, you don't"
- These are your fastest wins — someone is already getting traffic, you can too

### Step 4: Cluster Keywords
- Use **Keyword Strategy Builder** to auto-group into topic clusters
- Each cluster = one content piece (or page)
- Group by: parent topic + subtopics

### Step 5: Map to Pages
- One primary keyword per URL
- Supporting keywords in same cluster go on same page
- Track in a keyword map spreadsheet

## Keyword Metrics Explained

| Metric | What It Means | Good Value |
|--------|--------------|------------|
| Volume | Monthly searches | Depends on niche |
| KD | 0–100 difficulty score | <40 for new sites |
| CPC | Cost per click (ad value signal) | Higher = more commercial |
| Competitive Density | Ad competition | Signals commercial intent |
| Trend | 12-month direction | Rising = opportunity |
| SERP Features | What's on the results page | Snippets = chance to steal traffic |

## Long-Tail Keyword Strategy

Long-tail (3+ words) keywords are:
- Lower competition
- Higher conversion rates
- Easier to rank for
- Combined volume can exceed head terms

**How to find them in Semrush:**
1. Keyword Magic Tool → Filter: Word count ≥ 3
2. Questions tab → find "how to", "what is", "why does" keywords
3. Related keywords tab → semantic variations

---

## Seasonal & Trend Keywords

- Use **Keyword Overview** → Trend graph to spot seasonality
- Cross-reference with **Exploding Topics** (App Center) for emerging terms
- Plan content 2–3 months before seasonal peak

---

## Local Keyword Modifiers

For local businesses, append to seed keywords:
- "[city name]"
- "near me"
- "[neighborhood]"
- "[state/region]"

Use **Position Tracking** with location set to your target city.
