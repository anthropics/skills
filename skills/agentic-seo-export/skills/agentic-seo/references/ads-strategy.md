# Ads Strategy — PPC Campaign Setup & Optimization

## PPC Research Phase

### Competitor Intelligence (Before You Spend)
1. **Advertising Research** → enter competitor domain
   - See their top keywords (by ad spend estimate)
   - View actual ad copy (headline + description)
   - Identify: what messaging works in your market

2. **Ads History** → see how long they've run ads on each keyword
   - Long-running ad = proven profitable (they wouldn't keep running it otherwise)
   - Use this to validate your keyword targets

3. **AdClarity** (App Center) → deeper intelligence
   - Estimated ad spend per competitor
   - Creative performance data
   - Ad impressions and engagement

## Keyword Planning for PPC

### PPC Keyword Tool Workflow
1. Enter seed keywords
2. Add competitor domains to pull their keywords
3. Organize into: **Campaign → Ad Groups → Keywords**
4. **Auto-generate negatives**: prevents wasted spend on irrelevant searches

### Match Type Strategy
| Match Type | When to Use | Example |
|-----------|-------------|---------|
| Exact [keyword] | High intent, known converter | [buy running shoes] |
| Phrase "keyword" | Moderate control | "running shoes" |
| Broad +keyword | Discovery, Scale | +running +shoes |

### Quality Score Optimization
Higher Quality Score = lower CPC + better position

Improve by:
- Match ad copy to keyword intent
- Landing page matches ad promise
- Improve CTR (click-through rate)
- Reduce bounce rate on landing page

## Campaign Structure Best Practices

### Google Ads Structure
```
Account
├── Campaign 1 (Product A)
│   ├── Ad Group 1 (Feature-focused)
│   │   ├── Keywords: [buy product a], [product a price]
│   │   └── Ads: 3 RSAs per ad group
│   └── Ad Group 2 (Problem-solution)
│       ├── Keywords: [how to solve X], [X solution]
│       └── Ads: 3 RSAs per ad group
└── Campaign 2 (Product B)
    └── ...
```

### SKAG Approach (Single Keyword Ad Groups)
- Semrush PPC Tool supports SKAG setup
- Tightest keyword-to-ad relevance
- Highest Quality Scores
- More work to manage, but better performance

## AI Ad Creation (Ads Launch Assistant)

### What It Generates
- **Headlines**: 5–15 variations per asset
- **Descriptions**: Brand-aligned, benefit-focused
- **Images**: AI-generated, brand color/logo matched
- **Videos**: Short-form from templates
- **Display banners**: 500+ templates, customizable

### A/B Testing Setup
1. Create 2–3 campaign "scenarios" with different:
   - Ad copy angles (feature vs benefit vs social proof)
   - Creative styles
   - CTA variations
2. Launch all simultaneously with equal budget split
3. After 2 weeks: pause lowest CTR/conversion rate version
4. Scale budget to winner

## Bidding Strategies

| Strategy | When to Use | Goal |
|---------|-------------|------|
| Target CPA | Enough conversion data (50+/month) | Control cost per lead |
| Target ROAS | E-commerce with revenue data | Maximize revenue |
| Maximize Conversions | New campaigns, limited data | Get conversions fast |
| Manual CPC | Full control needed | Test phase |
| Enhanced CPC | Hybrid control + automation | Transitioning to smart bidding |

## Optimization Checklist (Weekly)

**Semrush Ads AI Agent** automates most of this, but review:

- [ ] Search term report: add converting terms as keywords, add irrelevant terms as negatives
- [ ] Quality Score: fix any keywords scoring < 5
- [ ] Bid adjustments: increase for high-converting devices/locations/times
- [ ] Ad rotation: pause underperforming ads (CTR or conversion rate)
- [ ] Budget pacing: check if budget is limiting delivery (increase or redistribute)
- [ ] Landing page: test new variations for conversion rate improvement

## Meta Ads Integration

**Launch from Semrush:**
- Connect Facebook Ads Manager
- Use same AI-generated creatives from Google campaign
- Target: custom audiences, lookalikes, interest-based
- Formats: single image, carousel, video, stories

**Retargeting setup:**
1. Install Meta Pixel on website
2. Create custom audience: website visitors (last 30 days)
3. Create lookalike: 1–3% similarity to converters
4. Separate campaigns for cold vs warm audiences
