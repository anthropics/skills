# 图像生成技能

## 使用说明
该技能遵循严格的交互式两步工作流程来生成图像。

### 第1步：提示词扩展
**触发条件：** 当用户要求生成图像时（例如："草地上的女孩"）。
**操作：**
1. 先不要执行代码。
2. 生成**6个或更多不同的中文提示词变体**加上想象力 和更细致的描述。
3. 以编号列表的形式呈现给用户。
4. 请用户选择一个或多个编号。

### 第2步：执行与存储
**触发条件：** 当用户选择编号时（例如："1"）。
**操作：**
1. 确定所选编号对应的完整提示词文本。
2. 运行 ./generate.mjs 脚本来生成图像：
   参数1 "选中的提示词文本"
   生成图像成功后会有图像的URL
3. 使用curl命令下载图像

注意：./generate.mjs 脚本位于技能目录下
