# Cursor 设计系统灵感参考

## 1. 视觉主题与氛围

Cursor 的网站是一场暖色调极简主义与代码编辑器优雅感的碰撞研究。整体体验建立在暖米白画布（`#f2f1ed`）之上，搭配深暖棕色文字（`#26251e`）——非纯黑，非中性灰，而是带有黄褐色调的深暖近黑，令人联想到旧纸张、墨水与匠艺。这种暖意渗透到每一个表面：背景偏向奶油色（`#e6e5e0`、`#ebeae5`），边框消融于使用 `oklab` 色彩空间的透明暖色覆层，甚至错误状态（`#cf2d56`）也带有暖意而非临床感的红色。整体效果更像一本优质印刷出版物，而非科技网站。

定制的 CursorGothic 字体是排版上的签名标识——一种哥特式无衬线字体，在展示尺寸下（72px 时 -2.16px）使用激进的负字间距，创造出被压缩的、工程化的质感。作为第二种声音，jjannon 衬线字体（带 OpenType `"cswh"` 上下文花押字替换特性）为正文和编辑性段落提供文学性的对位。等宽字体声音来自 berkeleyMono——一种精致的编码字体，将营销站点与 Cursor 作为代码编辑器的核心身份相连。这一三字体系统（哥特展示体、衬线正文体、等宽代码体）赋予了 Cursor 在开发工具领域最为丰富的排版调色板之一。

边框系统尤为独特——Cursor 使用 `oklab()` 色彩空间来设置边框颜色，以不同 alpha 值（0.1、0.2、0.55）将暖棕色应用于边框，使边框感觉有机而非机械。标志性边框颜色 `oklab(0.263084 -0.00230259 0.0124794 / 0.1)` 并非简单的 rgba 值，而是一种感知均匀的颜色，在不同背景下保持一致的视觉效果。

**核心特征：**
- CursorGothic 配激进负字间距（72px 时 -2.16px，36px 时 -0.72px），用于压缩展示标题
- jjannon 衬线字体用于正文，带 OpenType `"cswh"`（上下文花押字替换）
- berkeleyMono 用于代码和技术标签
- 暖米白背景（`#f2f1ed`）替代纯白——整个系统向暖色偏移
- 主文字颜色 `#26251e`（带黄色调的暖近黑）
- 强调橙色 `#f54e00` 用于品牌高亮和链接
- oklab 色彩空间边框，在不同 alpha 值下实现感知均匀的边缘处理
- 胶囊形元素使用极大圆角（33.5M px，实际为完全胶囊）
- 8px 基础间距系统，带精细的亚 8px 增量（1.5px、2px、2.5px、3px、4px、5px、6px）

## 2. 色彩体系与角色分工

### 主色
- **Cursor 深色** (`#26251e`)：主文字、标题、深色 UI 表面。带明显黄棕色调的暖近黑——系统的定义色。
- **Cursor 奶油色** (`#f2f1ed`)：页面背景、主表面。非白色，而是设定整体暖色调的暖奶油色。
- **Cursor 浅色** (`#e6e5e0`)：次级表面、按钮背景、卡片填充。略偏暖、略深的奶油色。
- **纯白** (`#ffffff`)：稀少使用，用于最高对比度元素和特定表面高亮。
- **真黑** (`#000000`)：极少使用，特定代码/终端场景。

### 强调色
- **Cursor 橙色** (`#f54e00`)：品牌强调色，`--color-accent`。鲜艳的红橙色，用于主要 CTA、活跃链接和品牌时刻。暖而紧迫。
- **金色** (`#c08532`)：次级强调色，暖金色用于高级或高亮场景。

### 语义色
- **错误** (`#cf2d56`)：`--color-error`。暖深红玫瑰色，而非冷红色。
- **成功** (`#1f8a65`)：`--color-success`。偏暖的柔和蓝绿色。

### 时间轴 / 功能色
- **思考中** (`#dfa88f`)：暖桃色，用于 AI 时间轴的"思考"状态。
- **搜索** (`#9fc9a2`)：柔和鼠尾草绿，用于搜索/grep 操作。
- **读取** (`#9fbbe0`)：柔和蓝色，用于文件读取操作。
- **编辑** (`#c0a8dd`)：柔和薰衣草色，用于编辑操作。

### 表面色阶
- **表面 100** (`#f7f7f4`)：最浅的按钮/卡片表面，几乎无色调。
- **表面 200** (`#f2f1ed`)：主页面背景。
- **表面 300** (`#ebeae5`)：按钮默认背景，轻微强调。
- **表面 400** (`#e6e5e0`)：卡片背景，次级表面。
- **表面 500** (`#e1e0db`)：第三级按钮背景，更深的强调。

### 边框颜色
- **主边框** (`oklab(0.263084 -0.00230259 0.0124794 / 0.1)`)：标准边框，oklab 色彩空间中 10% 暖棕。
- **中等边框** (`oklab(0.263084 -0.00230259 0.0124794 / 0.2)`)：强调边框，20% 暖棕。
- **强边框** (`rgba(38, 37, 30, 0.55)`)：强边框，表格线。
- **实色边框** (`#26251e`)：全不透明深色边框，最高对比度。
- **浅色边框** (`#f2f1ed`)：与页面背景匹配的浅色边框。

### 阴影与层深
- **卡片阴影** (`rgba(0,0,0,0.14) 0px 28px 70px, rgba(0,0,0,0.1) 0px 14px 32px, oklab(0.263084 -0.00230259 0.0124794 / 0.1) 0px 0px 0px 1px`)：带暖 oklab 边框光环的重度浮起卡片。
- **环境阴影** (`rgba(0,0,0,0.02) 0px 0px 16px, rgba(0,0,0,0.008) 0px 0px 8px`)：浮动元素的微妙环境光晕。

## 3. 排版规则

### 字体家族
- **展示/标题**：`CursorGothic`，备用字体：`CursorGothic Fallback, system-ui, Helvetica Neue, Helvetica, Arial`
- **正文/编辑**：`jjannon`，备用字体：`Iowan Old Style, Palatino Linotype, URW Palladio L, P052, ui-serif, Georgia, Cambria, Times New Roman, Times`
- **代码/技术**：`berkeleyMono`，备用字体：`ui-monospace, SFMono-Regular, Menlo, Monaco, Consolas, Liberation Mono, Courier New`
- **UI/系统**：`system-ui`，备用字体：`-apple-system, Segoe UI, Helvetica Neue, Arial`
- **图标**：`CursorIcons16`（图标字体，14px 和 12px）
- **OpenType 特性**：jjannon 正文使用 `"cswh"`，CursorGothic 按钮/说明文字使用 `"ss09"`

### 排版原则
- **哥特压缩的冲击力**：CursorGothic 在展示尺寸下 72px 时字间距 -2.16px，逐渐放松：36px 时 -0.72px，26px 时 -0.325px，22px 时 -0.11px，16px 及以下恢复正常。字间距创造出精确工程的质感。
- **衬线字体赋予灵魂**：jjannon 提供文学性的暖意。`"cswh"` 特性增加上下文花押字替换，赋予正文书法般的质量。
- **三种排版声音**：哥特体（展示/UI）、衬线体（编辑/正文）、等宽体（代码/技术）。每种声音服务于不同的沟通目的。
- **字重克制**：CursorGothic 几乎完全使用字重 400，依赖尺寸和字间距建立层级而非字重。系统 UI 组件使用 500-700 进行功能强调。

## 5. 布局原则

### 间距系统
- 基础单位：8px
- 精细比例：1.5px、2px、2.5px、3px、4px、5px、6px（亚 8px 微调整）
- 标准比例：8px、10px、12px、14px（从提取中推导）
- 扩展比例（推断）：16px、24px、32px、48px、64px、96px
- 值得注意的是：精细的亚 8px 增量对于图标/文字微对齐至关重要

### 网格与容器
- 内容最大宽度：约 1200px
- 主图：居中单列，宽松顶部内边距（80-120px）
- 功能章节：2-3 列网格用于卡片和功能
- 全宽章节，暖奶油或稍深的背景
- 文档和设置页面的侧边栏布局

### 留白哲学
- **暖色负空间**：奶油色背景意味着留白有温度和质感，与冷白极简主义不同。大面积空白区域感觉温馨而非临床。
- **压缩文字，开阔布局**：CursorGothic 标题上激进的负字间距，被周围宽裕的边距所平衡。文字密集，周围的空间呼吸畅通。
- **章节变化**：交替的表面色调（奶油 → 较浅的奶油 → 奶油）创造细微的章节区分，无强硬分界。

### 圆角比例
- 微型（1.5px）：精细细节元素
- 小（2px）：行内元素、代码 span
- 中等（3px）：小型容器、行内徽章
- 标准（4px）：卡片、图片、紧凑按钮
- 舒适（8px）：主按钮、卡片、菜单
- 特色（10px）：较大容器、特色卡片
- 完全胶囊（33.5M px / 9999px）：胶囊按钮、标签、徽章

## 6. 层深与高度

| 级别 | 处理方式 | 用途 |
|-------|-----------|-----|
| 平面（第 0 级） | 无阴影 | 页面背景、文字块 |
| 边框光环（第 1 级） | `oklab(0.263 / 0.1) 0px 0px 0px 1px` | 标准卡片/容器边框（暖 oklab） |
| 中等边框（第 1b 级） | `oklab(0.263 / 0.2) 0px 0px 0px 1px` | 强调边框、活跃状态 |
| 环境（第 2 级） | `rgba(0,0,0,0.02) 0px 0px 16px, rgba(0,0,0,0.008) 0px 0px 8px` | 浮动元素、微妙光晕 |
| 浮起卡片（第 3 级） | `rgba(0,0,0,0.14) 0px 28px 70px, rgba(0,0,0,0.1) 0px 14px 32px, oklab 光环` | 弹窗、气泡、浮起卡片 |
| 焦点 | 按钮焦点时 `rgba(0,0,0,0.1) 0px 4px 12px` | 交互焦点反馈 |

**阴影哲学**：Cursor 的层深系统建立在两个理念之上。首先，边框使用感知均匀的 oklab 色彩空间而非 rgba，确保暖棕色边框在不同背景色调下呈现一致。其次，高度阴影使用极大的模糊值（28px、70px）配合适中不透明度（0.14、0.1），创造出漫射的、大气的浮起感，而非硬边投影。卡片不像漂浮在页面上——它们感觉像是页面轻柔地为它们开辟了一处空间。

### 装饰性层深
- 暖奶油表面变化无需阴影即可创造微妙的色调深度
- 10% 和 20% 的 oklab 边框创造边缘定义的光谱
- 无强硬分隔线——通过背景色调变化和间距实现章节分隔

## 7. 交互与动效

### 悬停状态
- 按钮：悬停时文字颜色切换至 `--color-error`（`#cf2d56`）——这种独特的暖深红信号交互性
- 链接：颜色切换至强调橙（`#f54e00`）或添加下划线装饰，带 `rgba(38, 37, 30, 0.4)`
- 卡片：悬停时阴影加深（环境 → 浮起）

### 焦点状态
- 基于阴影的焦点：`rgba(0,0,0,0.1) 0px 4px 12px`，用于基于深度的焦点指示
- 边框焦点：`oklab(0.263 / 0.2)`（20% 边框），用于输入框/表单焦点
- 所有焦点状态保持暖色调——无冷蓝色焦点环

### 过渡
- 颜色过渡：150ms ease，用于文字/背景颜色变化
- 阴影过渡：200ms ease，用于高度变化
- 变形：微妙的缩放或位移，用于交互反馈

## 8. 响应式行为

### 断点
| 名称 | 宽度 | 主要变化 |
|------|-------|-------------|
| 移动端 | <600px | 单列，减小内边距，堆叠导航 |
| 小型平板 | 600-768px | 开始 2 列网格 |
| 平板 | 768-900px | 扩展卡片网格，侧边栏出现 |
| 小型桌面端 | 900-1279px | 完整布局成形 |
| 桌面端 | >1279px | 完整布局，最大内容宽度 |

### 触摸目标
- 按钮使用舒适内边距（垂直 6px-14px，水平 8px-14px）
- 胶囊按钮以 3px-10px 内边距维持触摸友好尺寸
- 导航链接 14px 配适当触摸间距

### 折叠策略
- 主图：CursorGothic 72px → 36px → 26px，在较小屏幕上保持比例字间距
- 导航：水平链接 → 移动端汉堡菜单
- 功能卡片：3 列 → 2 列 → 单列堆叠
- 代码编辑器截图：保持宽高比，可缩小但保持边框处理
- 时间轴可视化：水平 → 垂直堆叠
