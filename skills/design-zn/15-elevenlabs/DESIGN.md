# ElevenLabs 设计系统灵感

## 1. 视觉主题与氛围

ElevenLabs 的网站是一份克制优雅的研究——近白色画布（`#ffffff`、`#f5f5f5`），排版和细腻阴影承担所有视觉重量。设计感觉像一本高端音频产品手册：干净、宽敞，足够自信让内容自己说话（从字面意义上说，因为 ElevenLabs 做的是语音 AI）。留白策略有几分像苹果，但更温暖——偶尔出现的温暖石灰色调（`#f5f2ef`、`#777169`）使这种纯净感不至于显得临床化。

排版系统建立在一种迷人的二元性上：Waldenburg 以 300 字重（细体）用于展示标题，创造出空灵、如耳语般纤细的标题——精致、精确，在大尺寸下出人意料地震撼。这种细体展示方式是设计的标志性特征——当大多数网站用粗体标题抓人眼球时，ElevenLabs 用纤细来制造引力。Inter 以务实可靠的方式处理所有正文和 UI 文字，使用微妙的正字间距（0.14px–0.18px），赋予正文一种通透、间距充足的阅读质感。WaldenburgFH 出现为特定按钮标签的粗体全大写变体。

ElevenLabs 的独特之处在于其多层阴影系统。元素使用复杂的阴影叠加：内嵌边框阴影（`rgba(0,0,0,0.075) 0px 0px 0px 0.5px inset`）、轮廓阴影（`rgba(0,0,0,0.06) 0px 0px 0px 1px`）和柔和抬升阴影（`rgba(0,0,0,0.04) 0px 4px 4px`）——所有这些都处于极低的透明度。结果是一种界面中的界面几乎不存在的设计，以最轻柔的触感漂浮于页面之上。胶囊形按钮（9999px），带温暖石灰底色背景（`rgba(245,242,239,0.8)`）和温暖阴影（`rgba(78,50,23,0.04)`），增添了一种触感十足的物理质感。

**核心特征：**
- 带温暖底调的近白色画布（`#f5f5f5`、`#f5f2ef`）
- Waldenburg 300 字重（细体）用于展示——空灵、如耳语般纤细的标题
- Inter 配正字间距（0.14–0.18px）用于正文——通透的可读性
- 低于 0.1 透明度的多层阴影叠加——界面几乎不存在
- 胶囊形按钮（9999px），带温暖石灰色背景
- WaldenburgFH 粗体全大写，用于特定行动召唤标签
- 温暖阴影色调：`rgba(78, 50, 23, 0.04)` — 阴影有颜色，不只是暗度
- Geist Mono / ui-monospace 用于代码片段

## 2. 色彩方案与角色

### 主色
- **纯白** (`#ffffff`)：主背景，卡片界面，按钮背景
- **浅灰** (`#f5f5f5`)：次要界面，微妙的区段差异化
- **温暖石灰** (`#f5f2ef`)：按钮背景（80% 透明度）——温暖的品牌标志
- **黑色** (`#000000`)：主要文字，标题，深色按钮

### 中性色系
- **深灰** (`#4e4e4e`)：次要文字，描述
- **温暖灰** (`#777169`)：第三级文字，弱化链接，装饰性下划线
- **近白** (`#f6f6f6`)：替代浅色界面

### 交互色
- **栅格青色** (`#7fffff`)：`--grid-column-bg`，25% 透明度——装饰性栅格叠加
- **环蓝** (`rgb(147 197 253 / 0.5)`)：`--tw-ring-color`，焦点环
- **浅边框** (`#e5e5e5`)：明确边框
- **微妙边框** (`rgba(0, 0, 0, 0.05)`)：超微妙底部边框

### 阴影
- **内嵌边框** (`rgba(0,0,0,0.075) 0px 0px 0px 0.5px inset`)：内部边缘定义
- **深色内嵌** (`rgba(0,0,0,0.1) 0px 0px 0px 0.5px inset`)：较强的内嵌变体
- **轮廓环** (`rgba(0,0,0,0.06) 0px 0px 0px 1px`)：阴影即边框
- **柔和抬升** (`rgba(0,0,0,0.04) 0px 4px 4px`)：轻柔的漂浮感
- **卡片阴影** (`rgba(0,0,0,0.4) 0px 0px 1px, rgba(0,0,0,0.04) 0px 4px 4px`)：按钮/卡片抬升
- **温暖阴影** (`rgba(78,50,23,0.04) 0px 6px 16px`)：温暖色调的按钮阴影
- **边缘阴影** (`rgba(0,0,0,0.08) 0px 0px 0px 0.5px`)：微妙边缘定义
- **内嵌环** (`rgba(0,0,0,0.1) 0px 0px 0px 1px inset`)：强内嵌边框

## 3. 排版规则

### 字体系列
- **展示字体**：`Waldenburg`，备用字体：`Waldenburg Fallback`
- **粗体展示**：`WaldenburgFH`，备用字体：`WaldenburgFH Fallback`
- **正文/UI**：`Inter`，备用字体：`Inter Fallback`
- **等宽字体**：`Geist Mono` 或 `ui-monospace, SFMono-Regular, Menlo, Monaco, Consolas`

### 排版原则
- **细体作为主角字重**：Waldenburg 300 是定义性的排版选择。当其他设计系统用粗体制造冲击时，ElevenLabs 用纤细创造引力——细线条感觉像音频波形，通过克制制造吸引力。
- **正文正字间距**：Inter 在正文文字上使用 +0.14px 至 +0.18px 字距，创造通透、间距充足的阅读节奏，与展示文字的紧凑字距（-0.96px）形成对比。
- **WaldenburgFH 用于强调**：Waldenburg 的一种粗体（700）全大写变体仅出现在特定行动召唤按钮标签中，配以 0.7px 字间距——这是整个排版系统中唯一发声响亮的地方。
- **等宽字体作为氛围**：Geist Mono 以宽松行高（1.85）用于代码块，感觉不急不缓，易于阅读。

## 5. 布局原则

### 间距系统
- 基础单位：8px
- 比例：1px、3px、4px、8px、9px、10px、11px、12px、16px、18px、20px、24px、28px、32px、40px

### 栅格与容器
- 慷慨最大宽度的居中内容
- 单列主视觉区，扩展至功能栅格
- 产品展示的全宽渐变区段
- 浅灰背景上的白色卡片栅格

### 留白哲学
- **苹果式慷慨**：区段之间的大量垂直间距创造出高端、从容的节奏。每个区段都是一个展览。
- **温暖的空白**：留白不是冷漠的——温暖石灰底调和温暖阴影赋予空白空间一种触感、物理质感。
- **排版引导的节奏**：细体 Waldenburg 标题创造视觉"耳语"，引导眼睛穿越大片白色空间。

### 圆角比例
- 极小（2px）：小链接，内联元素
- 微妙（4px）：导航项，选项卡面板，标签
- 标准（8px）：小容器
- 舒适（10px–12px）：中型卡片，下拉框
- 卡片（16px）：标准卡片，文章
- 大（18px–20px）：特色卡片，代码面板
- 区段（24px）：大型面板，区段容器
- 温暖按钮（30px）：温暖石灰行动召唤按钮
- 胶囊（9999px）：主要按钮，导航胶囊

## 6. 深度与层级

| 层级 | 处理方式 | 用途 |
|-------|-----------|-----|
| 平面（层级 0） | 无阴影 | 页面背景，文字块 |
| 内嵌边缘（层级 0.5） | `rgba(0,0,0,0.075) 0px 0px 0px 0.5px inset, #fff 0px 0px 0px 0px inset` | 内部边框定义 |
| 轮廓环（层级 1） | `rgba(0,0,0,0.06) 0px 0px 0px 1px` + `rgba(0,0,0,0.04) 0px 1px 2px` + `rgba(0,0,0,0.04) 0px 2px 4px` | 阴影即边框，用于卡片 |
| 卡片（层级 2） | `rgba(0,0,0,0.4) 0px 0px 1px, rgba(0,0,0,0.04) 0px 4px 4px` | 按钮抬升，突出卡片 |
| 温暖抬升（层级 3） | `rgba(78,50,23,0.04) 0px 6px 16px` | 特色行动召唤按钮——温暖色调 |
| 焦点（无障碍） | `var(--tw-ring-offset-shadow)` 蓝色环 | 键盘焦点 |

**阴影哲学**：ElevenLabs 使用所有已分析设计系统中最精致的阴影系统。每一层阴影的透明度都低于 0.1，许多同时包含向外投射和向内内嵌组件，而特色行动召唤按钮的阴影使用实际的温暖颜色（`rgba(78,50,23,...)`）而非中性黑色。半像素内嵌边框（`0px 0px 0px 0.5px inset`）创造出如此微妙的边缘，以至于被感知而非被看到——界面以最轻柔的触感定义自身。

## 7. 应做与禁止

### 应做
- 所有展示标题使用 Waldenburg 字重 300——纤细就是品牌本身
- 以低于 0.1 透明度应用多层阴影（内嵌 + 轮廓 + 抬升）
- 对特色元素使用温暖石灰色调（`#f5f2ef`、`rgba(245,242,239,0.8)`）
- 在 Inter 正文文字上应用正字间距（+0.14px 至 +0.18px）
- 主要按钮使用 9999px 圆角——胶囊形是标准
- 对特色行动召唤按钮使用温暖色调阴影（`rgba(78,50,23,0.04)`）
- 页面主体保持白色，以微妙灰色区段差异化
- WaldenburgFH 粗体全大写仅用于特定行动召唤按钮标签

### 禁止
- 不要对标题使用粗体（700）Waldenburg——300 字重不可妥协
- 不要使用厚重阴影（>0.1 透明度）——空灵质感需要耳语级别的深度
- 不要使用冷灰边框——整个系统自始至终带有温暖底调
- 不要跳过内嵌阴影组件——半像素内嵌边框定义边缘
- 不要对正文应用负字间距——Inter 使用正字距
- 不要在卡片上使用尖角（<8px）——慷慨的圆角是结构性的
- 不要引入品牌颜色——调色板刻意是无彩色配温暖底调
- 不要让按钮不透明而厚重——温暖半透明石灰处理是标志性特征

## 8. 响应式行为

### 断点
| 名称 | 宽度 | 主要变化 |
|------|-------|-------------|
| 移动端 | <1024px | 单列，汉堡导航，堆叠区段 |
| 桌面端 | >1024px | 完整布局，水平导航，多列栅格 |

### 触控目标
- 胶囊形按钮配慷慨内边距（12px–20px）
- 导航链接 15px 配足够间距
- 下拉选择框保持舒适尺寸

### 折叠策略
- 导航：水平 → 1024px 时汉堡
- 功能栅格：多列 → 堆叠
- 主视觉：保持居中布局，字体按比例缩放
- 渐变区段：保持全宽，内容堆叠
- 间距按比例压缩

### 图像行为
- 产品截图响应式缩放
- 渐变背景在移动端简化
- 音频波形预览保持宽高比
- 圆角在各断点保持不变
