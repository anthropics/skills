# Design System Inspiration of Together AI

## 1. 视觉主题与氛围

Together AI 的界面是一个为企业级 AI 基础设施而构建的粉彩渐变梦境——一种设计风格，竟能让 GPU 集群和模型推理感觉轻盈、通透、充满乐观。英雄区绽放着柔和的粉蓝紫薰衣草渐变和抽象的、画家风格的插图，唤起云朵与飞翔的感觉，为"AI 原生云"的主张建立视觉隐喻。在这种柔软感的对比下，排版以精准切入："The Future"展示字体在 64px 下搭配激进的负字母间距（-1.92px），创造出密集、权威的标题块。

设计在两个世界之间游走：明亮的白色画布浅色面，粉彩渐变和统计卡片创造出平易近人的平台概述；以及深海军蓝宇宙（`#010120`——不是灰黑色，而是深邃的午夜蓝），研究论文和技术内容在此驻留。这种双重世界方法优雅地将"商业"信息（浅色、友好、数据驱动）与"研究"信息（深色、严肃、学术）区分开来。

Together AI 的独特之处在于其字体系统。"The Future"处理所有展示和正文文字，呈现几何现代主义美学，而"PP Neue Montreal Mono"提供经过精心字母间距的大写标签——共同创造出"有品位的技术基础设施公司"的个性。品牌强调色——洋红（`#ef2cc1`）和橙色（`#fc4c02`）——在渐变和插图中出现，绝不污染干净的 UI。

**核心特征：**
- 纯白画布上的柔和粉彩渐变（粉色、蓝色、薰衣草色）
- 深色/研究区块使用深午夜蓝（`#010120`）——而非灰黑色
- 定制"The Future"字体，全局使用激进的负字母间距
- PP Neue Montreal Mono 用于大写技术标签
- 锐利几何（4px、8px 圆角）——不圆润，不胶囊形
- 洋红（#ef2cc1）+ 橙色（#fc4c02）品牌强调色仅用于插图
- 薰衣草色（#bdbbff）作为柔和的辅助强调色
- 企业级统计数据突出显示（2x、60%、90%）
- 深蓝色调阴影（rgba(1, 1, 32, 0.1)）

## 2. 色彩体系与角色

### 主色
- **品牌洋红** (`#ef2cc1`)：主要品牌强调色——用于渐变插图和最高信号品牌时刻的鲜艳粉洋红色。绝不用于 UI 框架。
- **品牌橙** (`#fc4c02`)：次要品牌强调色——用于渐变端点和温暖强调时刻的鲜艳橙色。
- **深蓝** (`#010120`)：主要深色表面——用于研究区块、页脚和深色容器的深午夜蓝黑色。不是灰色，不是黑色——明确是蓝色。

### 辅助色与强调色
- **柔和薰衣草色** (`#bdbbff`)：温和的蓝紫色，用于微妙强调、次要指示器和柔和 UI 高亮。
- **黑色 40** (`#00000066`)：去强调叠层和次要文字的半透明黑色。

### 表面与背景
- **纯白** (`#ffffff`)：主要浅色区块页面背景。
- **深蓝** (`#010120`)：深色区块背景——研究、页脚、技术内容。
- **磨砂浅色** (`rgba(255, 255, 255, 0.12)`)：深色区块上按钮的磨砂玻璃背景。
- **磨砂深色** (`rgba(0, 0, 0, 0.08)`)：浅色区块上的微妙色调表面。

### 中性色与文字
- **纯黑** (`#000000`)：浅色表面上的主要文字。
- **纯白** (`#ffffff`)：深色表面上的主要文字。
- **黑色 8%** (`rgba(0, 0, 0, 0.08)`)：浅色表面上的边框和微妙包含感。
- **白色 12%** (`rgba(255, 255, 255, 0.12)`)：深色表面上的边框和包含感。

### 渐变系统
- **粉彩云朵渐变**：英雄插图中从柔和粉色 → 薰衣草色 → 柔和蓝色的渐变。以抽象的、画家风格形式呈现——云朵、羽毛、流动形状——创造视觉温暖而无字面意义。
- **英雄渐变**：英雄背景使用叠加在白色上的柔和粉彩色调，创造出黎明般的大气效果。

## 3. 排版规则

### 字体家族
- **主字体**：`The Future`，备选字体：`Arial`
- **等宽字体 / 标签**：`PP Neue Montreal Mono`，备选字体：`Georgia`

### 层次体系

| 角色 | 字体 | 尺寸 | 字重 | 行高 | 字母间距 | 备注 |
|------|------|------|--------|-------------|----------------|-------|
| 展示 / 英雄 | The Future | 64px (4rem) | 400–500 | 1.00–1.10（紧凑） | -1.92px | 最大冲击力，密集块 |
| 区块标题 | The Future | 40px (2.5rem) | 500 | 1.20（紧凑） | -0.8px | 功能区块标题 |
| 副标题 | The Future | 28px (1.75rem) | 500 | 1.15（紧凑） | -0.42px | 卡片标题 |
| 功能标题 | The Future | 22px (1.38rem) | 500 | 1.15（紧凑） | -0.22px | 小型功能标题 |
| 大正文 | The Future | 18px (1.13rem) | 400–500 | 1.30（紧凑） | -0.18px | 描述、区块 |
| 正文 / 按钮 | The Future | 16px (1rem) | 400–500 | 1.25–1.30 | -0.16px | 标准正文、导航、按钮 |
| 说明文字 | The Future | 14px (0.88rem) | 400–500 | 1.40 | normal | 元数据、描述 |
| 等宽标签 | PP Neue Montreal Mono | 16px (1rem) | 500 | 1.00（紧凑） | 0.08px | 大写区块标签 |
| 等宽小字 | PP Neue Montreal Mono | 11px (0.69rem) | 500 | 1.00–1.40 | 0.055–0.08px | 小型大写标签 |
| 等宽微型 | PP Neue Montreal Mono | 10px (0.63rem) | 400 | 1.40 | 0.05px | 最小大写标签 |

### 排版原则
- **全局负字母间距**："The Future"的每个尺寸都使用负字母间距（-0.16px 至 -1.92px），创造出一贯紧凑、现代的文字效果。
- **等宽字体用于结构**：PP Neue Montreal Mono 大写搭配正字母间距创造技术"标签"时刻，在不与展示文字竞争的情况下构建页面结构。
- **字重 500 作为强调**：系统使用 400（常规）和 500（中等）——没有粗体。中等字重标记标题和强调。
- **全局紧凑行高**：即使正文也使用 1.25–1.30 行高——比典型值更紧，创造出密集的、信息丰富的感觉。

## 4. 组件样式

### 按钮

**深色表面上的玻璃效果**
- 背景：`rgba(255, 255, 255, 0.12)`（磨砂玻璃）
- 文字：纯白 (`#ffffff`)
- 圆角：锐利（4px）
- 不透明度：0.5
- 悬停：透明深色叠层
- 用于深色区块——微妙、玻璃质感

**深色实心**
- 背景：深蓝 (`#010120`) 或纯黑
- 文字：纯白
- 圆角：锐利（4px）
- 浅色表面上的主要行动号召

**描边浅色**
- 边框：`1px solid rgba(0, 0, 0, 0.08)`
- 背景：透明或微妙玻璃
- 文字：纯黑
- 圆角：锐利（4px）
- 浅色表面上的次要操作

### 卡片与容器
- 背景：纯白或微妙玻璃色调
- 边框：浅色表面 `1px solid rgba(0, 0, 0, 0.08)`；深色表面 `1px solid rgba(255, 255, 255, 0.12)`
- 圆角：徽章和小元素锐利（4px）；大容器舒适（8px）
- 阴影：深蓝色调（`rgba(1, 1, 32, 0.1) 0px 4px 10px`）——温暖而微妙
- 统计卡片突出显示大数字

### 徽章 / 标签
- 背景：浅色 `rgba(0, 0, 0, 0.04)` 或深色 `rgba(255, 255, 255, 0.12)`
- 文字：浅色时黑色，深色时白色
- 内边距：2px 8px（紧凑）
- 圆角：锐利（4px）
- 边框：`1px solid rgba(0, 0, 0, 0.08)`
- PP Neue Montreal Mono，大写，16px

### 导航
- 白色/透明背景上的简洁横向导航
- Logo：Together AI 字标
- 链接：The Future 16px，字重 400
- 行动号召：深色实心按钮
- 悬停：无文字装饰

### 图片处理
- 抽象粉彩渐变插图（云朵/羽毛形态）
- 深色/浅色表面上的产品 UI 截图
- 编辑风格的团队照片
- 带深色背景的研究论文卡片

### 特色组件

**统计栏**
- 大型性能指标（2x、60%、90%）
- 粗体展示数字
- 下方简短描述说明
- 简洁的横向布局

**等宽区块标签**
- PP Neue Montreal Mono，大写，11px，字母间距 0.055px
- 在整个页面中用作导航标识
- 技术性、结构化的感觉

**研究区块**
- 深蓝（#010120）背景
- 白色文字，研究论文缩略图
- 创造出独特的"学术"区域

**大型页脚 Logo**
- "together"字标在深色页脚以巨大尺寸呈现
- 创造出品牌宣言式的结尾时刻

## 5. 布局原则

### 间距系统
- 基础单位：8px
- 刻度：1px、2px、4px、8px、10px、12px、16px、20px、24px、32px、44px、48px、80px、100px、120px
- 按钮/徽章内边距：2px 8px（紧凑）
- 卡片内部内边距：约 24–32px
- 区块垂直间距：充裕（80–120px）

### 网格与容器
- 最大容器宽度：约 1200px，居中
- 英雄区：居中，粉彩渐变背景
- 功能区块：多列卡片网格
- 统计数据：横向排列的指标卡片
- 研究：深色全宽区块

### 留白理念
- **乐观的呼吸空间**：区块之间的充裕间距创造出开放、邀请的感觉，使企业级 AI 基础设施感觉平易近人。
- **双重氛围**：浅色区块在留白中呼吸；深色区块内容密度更高。
- **统计数据作为视觉锚点**：大数字搭配小说明文字创造出自然的焦点。

### 圆角刻度
- 锐利（4px）：按钮、徽章、标签、小型交互元素——主要圆角
- 舒适（8px）：大容器、功能卡片

*这是一个刻意克制的圆角系统——没有胶囊形，没有大圆角。锐利几何与柔和粉彩渐变形成对比。*

## 6. 深度与层次

| 层级 | 处理方式 | 用途 |
|-------|-----------|-----|
| 平面（0 级） | 无阴影，无边框 | 页面背景，文字块 |
| 包含（1 级） | 浅色 `1px solid rgba(0,0,0,0.08)` 或深色 `rgba(255,255,255,0.12)` | 卡片、徽章、容器 |
| 悬浮（2 级） | `rgba(1, 1, 32, 0.1) 0px 4px 10px` | 功能卡片、悬停状态 |
| 深色区（3 级） | 深蓝（#010120）全宽背景 | 研究、页脚、技术区块 |

**阴影理念**：Together AI 使用单一的独特阴影——用深蓝（`rgba(1, 1, 32, 0.1)`）而非通用黑色进行着色。这赋予悬浮元素一种微妙的蓝色调，将其与品牌的午夜蓝暗黑模式联系起来。阴影柔和（10px 模糊，4px 偏移）且始终向下——营造出温和的纸张悬浮层次。

## 7. 应做与禁止

### 应做
- 英雄插图和装饰背景使用粉彩渐变（粉/蓝/薰衣草）
- 深色区块使用深蓝（#010120）——绝不使用通用灰黑色
- "The Future"的所有文字应用负字母间距（按尺寸缩放）
- 区块标签和技术标记使用 PP Neue Montreal Mono 大写
- 徽章和交互元素保持锐利圆角（4px）
- 使用深蓝色调阴影表示层次
- 保持浅色/深色区块的双重性——商业（浅色）vs 研究（深色）
- 用大型展示数字突出显示企业级统计数据

### 禁止
- 不要将品牌洋红（#ef2cc1）或品牌橙（#fc4c02）用作 UI 颜色——它们仅用于插图
- 不要使用胶囊形或大圆角——几何是锐利的
- 不要使用通用灰黑色作为深色区块——始终使用深蓝（#010120）
- 不要在"The Future"上使用正字母间距——始终为负值
- 不要使用粗体（700+）字重——400–500 是完整范围
- 不要使用暖色调阴影——始终使用深蓝色调
- 不要将区块间距缩小至 48px 以下——开放感是核心
- 不要混入其他字体——"The Future" + PP Neue Montreal Mono 是固定搭配

## 8. 响应式行为

### 断点
| 名称 | 宽度 | 关键变化 |
|------|-------|-------------|
| 移动端 | <479px | 紧凑布局，一切堆叠 |
| 大型移动端 | 479–767px | 单列，汉堡导航 |
| 平板 | 768–991px | 2 列网格开始 |
| 桌面 | 992px+ | 完整多列布局 |

### 触控目标
- 按钮具有足够的内边距
- 卡片表面作为触控目标
- 导航链接舒适的 16px

### 折叠策略
- **导航**：移动端折叠为汉堡菜单
- **英雄文字**：64px → 40px → 28px 渐进缩放
- **统计栏**：横向 → 垂直堆叠
- **功能网格**：多列 → 单列
- **研究区块**：卡片垂直堆叠

### 图片行为
- 粉彩插图按比例缩放
- 产品截图保持宽高比
- 团队照片在容器内缩放
