# Design System Inspiration of Airbnb

## 1. 视觉主题与氛围

Airbnb 的网站是一个温暖的、以摄影为先导的市场平台，感觉就像翻阅一本旅行杂志，每一页都在邀请你预订。设计建立在纯白（`#ffffff`）画布之上，标志性的 Rausch 红（`#ff385c`）——以 Airbnb 第一条街道地址命名——作为唯一的品牌强调色。结果是一块简洁、通透的画布，其中房源摄影、类别图标和红色行动号召按钮是唯一的颜色来源。

排版使用 Airbnb Cereal VF——一款温暖亲切的定制可变字体，带有圆润的末端，呼应品牌"随处都是归宿"的哲学。字体在紧凑的字重范围内运作：大多数 UI 使用 500（中等），强调使用 600（半粗体），主要标题使用 700（粗体）。标题上轻微的负字母间距（-0.18px 至 -0.44px）创造出一种温馨、亲密的阅读体验，而非科技公司的压缩效率感。

Airbnb 的独特之处在于其基于调色板的令牌系统（`--palette-*`）和多层阴影方式。主要卡片阴影使用三层叠加（`rgba(0,0,0,0.02) 0px 0px 0px 1px, rgba(0,0,0,0.04) 0px 2px 6px, rgba(0,0,0,0.1) 0px 4px 8px`），创造出微妙的、温暖的抬起感。结合充裕的圆角（8px–32px）、圆形导航控件（50%）和带横向滚动的类别药丸形栏，界面感觉触觉性强且引人入胜——为浏览而设计，而非命令。

**核心特征：**
- 纯白画布，Rausch 红（`#ff385c`）作为唯一品牌强调色
- Airbnb Cereal VF——带有温暖、圆润末端的定制可变字体
- 基于调色板的令牌系统（`--palette-*`）用于系统性颜色管理
- 三层卡片阴影：边框环 + 柔和模糊 + 较强模糊
- 充裕的圆角：8px 按钮，14px 徽章，20px 卡片，32px 大元素
- 圆形导航控件（50% 圆角）
- 摄影优先的房源卡片——图片是英雄内容
- 近黑色文字（`#222222`）——温暖，不冷漠
- Luxe 紫（`#460479`）和 Plus 洋红（`#92174d`）用于高端级别

## 2. 色彩体系与角色

### 主品牌色
- **Rausch 红** (`#ff385c`)：`--palette-bg-primary-core`，主要行动号召，品牌强调，激活状态
- **深 Rausch** (`#e00b41`)：`--palette-bg-tertiary-core`，品牌红的按下/深色变体
- **错误红** (`#c13515`)：`--palette-text-primary-error`，浅色上的错误文字
- **错误深色** (`#b32505`)：`--palette-text-secondary-error-hover`，错误悬停

### 高端级别
- **Luxe 紫** (`#460479`)：`--palette-bg-primary-luxe`，Airbnb Luxe 级别品牌
- **Plus 洋红** (`#92174d`)：`--palette-bg-primary-plus`，Airbnb Plus 级别品牌

### 文字刻度
- **近黑** (`#222222`)：`--palette-text-primary`，主要文字——温暖，不冷漠
- **聚焦灰** (`#3f3f3f`)：`--palette-text-focused`，聚焦状态文字
- **次要灰** (`#6a6a6a`)：次要文字、描述
- **禁用** (`rgba(0,0,0,0.24)`)：`--palette-text-material-disabled`，禁用状态
- **链接禁用** (`#929292`)：`--palette-text-link-disabled`，禁用链接

### 交互色
- **法律蓝** (`#428bff`)：`--palette-text-legal`，法律链接、信息类
- **边框灰** (`#c1c1c1`)：卡片和分隔线的边框颜色
- **浅表面** (`#f2f2f2`)：圆形导航按钮，次要表面

### 表面与阴影
- **纯白** (`#ffffff`)：页面背景，卡片表面
- **卡片阴影** (`rgba(0,0,0,0.02) 0px 0px 0px 1px, rgba(0,0,0,0.04) 0px 2px 6px, rgba(0,0,0,0.1) 0px 4px 8px`)：三层温暖抬起
- **悬停阴影** (`rgba(0,0,0,0.08) 0px 4px 12px`)：按钮悬停层次

## 3. 排版规则

### 字体家族
- **主字体**：`Airbnb Cereal VF`，备选字体：`Circular, -apple-system, system-ui, Roboto, Helvetica Neue`
- **OpenType 特性**：特定说明文字元素上的 `"salt"`（样式交替）

### 排版原则
- **温暖字重范围**：500–700 为主导。标题没有字重 300 或 400——Airbnb 的字体始终至少是中等字重，创造温暖、自信的声音。
- **标题上的负字母间距**：展示文字上 -0.18px 至 -0.44px 的字母间距创造亲密、温馨的标题，而非冷漠、压缩的效果。
- **"salt" OpenType 特性**：特定 UI 元素（徽章、说明文字）上的样式交替创造出增添视觉趣味的微妙字形变化。
- **可变字体精度**：Cereal VF 实现连续字重插值，尽管设计系统使用 500、600 和 700 的离散节点。

## 4. 组件样式

### 按钮

**主要深色**
- 背景：`#222222`（近黑，非纯黑）
- 文字：`#ffffff`
- 内边距：0px 24px
- 圆角：8px
- 悬停：通过 `var(--accent-bg-error)` 过渡至错误/品牌强调色
- 焦点：`0 0 0 2px var(--palette-grey1000)` 环 + scale(0.92)

**圆形导航**
- 背景：`#f2f2f2`
- 文字：`#222222`
- 圆角：50%（圆形）
- 悬停：阴影 `rgba(0,0,0,0.08) 0px 4px 12px` + translateX(50%)
- 激活：4px 白色边框环 + 焦点阴影
- 焦点：scale(0.92) 收缩动画

### 卡片与容器
- 背景：`#ffffff`
- 圆角：14px（徽章），20px（卡片/按钮），32px（大型）
- 阴影：`rgba(0,0,0,0.02) 0px 0px 0px 1px, rgba(0,0,0,0.04) 0px 2px 6px, rgba(0,0,0,0.1) 0px 4px 8px`（三层）
- 房源卡片：顶部全宽摄影，下方详情
- 轮播控件：圆形 50% 按钮

### 输入框
- 搜索：`#222222` 文字
- 焦点：`var(--palette-bg-primary-error)` 背景色调 + `0 0 0 2px` 环
- 圆角：根据上下文而定（搜索栏使用近似药丸形圆角）

### 导航
- 白色粘性头部，居中搜索栏
- Airbnb Logo（Rausch 红）左对齐
- 类别筛选药丸形：搜索栏下方横向滚动
- 圆形导航控件用于轮播导航
- "成为房东"文字链接，头像/菜单右对齐

### 图片处理
- 房源摄影以充裕高度填充卡片顶部
- 带圆点指示器的图片轮播
- 图片上的心形/收藏图标叠加
- 包含图片上的 8px–14px 圆角

## 5. 布局原则

### 间距系统
- 基础单位：8px
- 刻度：2px、3px、4px、6px、8px、10px、11px、12px、15px、16px、22px、24px、32px

### 网格与容器
- 居中搜索的全宽头部
- 类别药丸形栏：横向可滚动行
- 房源网格：响应式多列（桌面端 3–5 列）
- 带链接列的全宽页脚

### 留白理念
- **旅行杂志间距**：区块之间充裕的垂直内边距创造出悠闲的浏览节奏——你应该慢慢滚动，像浏览杂志一样。
- **摄影密度**：房源卡片排列相对紧凑，但每张图片足够大以让人有沉浸感。
- **搜索栏突出**：搜索栏在头部获得最大垂直空间——找到目的地是首要操作。

### 圆角刻度
- 微妙（4px）：小链接
- 标准（8px）：按钮、标签、搜索元素
- 徽章（14px）：状态徽章、标签
- 卡片（20px）：功能卡片、大型按钮
- 大型（32px）：大容器、英雄元素
- 圆形（50%）：导航控件、头像、图标

## 6. 深度与层次

| 层级 | 处理方式 | 用途 |
|-------|-----------|-----|
| 平面（0 级） | 无阴影 | 页面背景，文字块 |
| 卡片（1 级） | `rgba(0,0,0,0.02) 0px 0px 0px 1px, rgba(0,0,0,0.04) 0px 2px 6px, rgba(0,0,0,0.1) 0px 4px 8px` | 房源卡片、搜索栏 |
| 悬停（2 级） | `rgba(0,0,0,0.08) 0px 4px 12px` | 按钮悬停、交互抬起 |
| 激活焦点（3 级） | `rgb(255,255,255) 0px 0px 0px 4px` + 焦点环 | 激活/聚焦元素 |

**阴影理念**：Airbnb 的三层阴影系统创造出温暖、自然的抬起感。第 1 层（`0px 0px 0px 1px`，不透明度 0.02）是超微妙的边框。第 2 层（`0px 2px 6px`，不透明度 0.04）提供柔和的环境光阴影。第 3 层（`0px 4px 8px`，不透明度 0.1）增添主要抬起感。这种分级方式创造出感觉像自然光而非 CSS 效果的阴影。

## 7. 应做与禁止

### 应做
- 文字使用 `#222222`（温暖近黑）——绝不用纯 `#000000`
- 仅在主要行动号召和品牌时刻应用 Rausch 红（`#ff385c`）——它是唯一强调色
- 使用 Airbnb Cereal VF 字重 500–700——温暖字重范围是有意为之的
- 对所有悬浮表面应用三层卡片阴影
- 使用充裕的圆角：8px 按钮，20px 卡片，50% 控件
- 使用摄影作为主要视觉内容——房源以图片为先
- 标题应用负字母间距（-0.18px 至 -0.44px）以增添亲密感
- 轮播/导航控件使用圆形（50%）按钮

### 禁止
- 文字不要使用纯黑（`#000000`）——始终用 `#222222`（温暖）
- 不要将 Rausch 红应用于背景或大面积表面——它仅是强调色
- 标题不要使用细字体字重（300、400）——最少 500
- 不要使用重阴影（主要层不透明度 > 0.1）——保持温暖和分级
- 卡片不要使用尖角（0–4px）——充裕的圆角（20px+）是核心
- 不要在 Rausch/Luxe/Plus 系统之外引入额外品牌颜色
- 不要覆盖调色板令牌系统——一致使用 `--palette-*` 变量

## 8. 响应式行为

### 断点
| 名称 | 宽度 | 关键变化 |
|------|-------|-------------|
| 小型移动端 | <375px | 单列，紧凑搜索 |
| 移动端 | 375–550px | 标准移动端房源网格 |
| 小平板 | 550–744px | 2 列房源 |
| 平板 | 744–950px | 搜索栏扩展 |
| 小桌面 | 950–1128px | 3 列房源 |
| 桌面 | 1128–1440px | 4 列网格，完整头部 |
| 大桌面 | 1440–1920px | 5 列网格 |
| 超宽屏 | >1920px | 最大网格宽度 |

*注：Airbnb 检测到 61 个断点——是观察到的最精细的响应式系统之一，反映了他们对每种可能屏幕尺寸下布局的执着追求。*

### 触控目标
- 圆形导航按钮：足够的 50% 圆角尺寸
- 房源卡片：移动端全卡片触控目标
- 搜索栏：为拇指交互突出显示尺寸
- 类别药丸形：横向可滚动，充裕的内边距

### 折叠策略
- 房源网格：5 → 4 → 3 → 2 → 1 列
- 搜索：扩展栏 → 紧凑栏 → 叠层/切换
- 类别药丸形：所有尺寸横向滚动
- 导航：完整头部 → 简化移动端
- 地图：侧面板 → 叠层/切换

### 图片行为
- 房源照片：移动端带滑动手势的轮播
- 保持宽高比的响应式图片尺寸
- 心形叠加在所有尺寸一致定位
- 根据视口调整照片质量
