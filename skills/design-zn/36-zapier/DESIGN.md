# Zapier 设计系统灵感

## 1. 视觉主题与氛围

Zapier 的网站散发着温暖、亲和的专业感。它拒绝了开发者工具冷漠的单色极简主义，转而采用奶油色调的画布（`#fffefb`），感觉像未漂白的纸张——数字时代的精心整理的笔记本。近黑色（`#201515`）文字带有淡淡的红棕色温度，营造出比机械更人性化的氛围。这是一款让自动化感觉毫不费力而非技术复杂的产品。

排版系统是两种截然不同个性的刻意交织。**Degular Display**——一种几何、宽字间的展示字体——以中等字重（500）和异常紧凑的行高（0.90）处理 56-80px 的英雄级标题，创造出垂直压缩的标题块，感觉像堆叠的积木。**Inter** 作为其他一切的主力，从区域标题到正文和导航，备用字体为 Helvetica 和 Arial。**GT Alpina**——一种优雅的细字重衬线字体，带激进的负字间距（-1.6px 至 -1.92px）——偶尔出现在更柔和的编辑时刻。这个三字体系统赋予 Zapier 变换语气的能力——从大胆有力（Degular）到干净实用（Inter）再到精致文学（GT Alpina）。

品牌标志性橙色（`#ff4f00`）无可辨认——一种鲜艳的、饱和的红橙色，精确处于交通锥的紧迫感和日落温暖之间。它使用克制但果断：主要行动按钮、激活状态下划线和强调边框。在温暖奶油色背景上，这种橙色创造了充满活力但不激进的色彩关系。

**核心特征：**
- 温暖奶油色画布（`#fffefb`）而非纯白——有机的纸质温度
- 带红色底调的近黑色（`#201515`）——文字有呼吸感而不主导
- Degular Display 0.90 行高英雄标题——压缩、有冲击力、现代
- Inter 作为所有功能性排版的通用 UI 字体
- GT Alpina 用于编辑强调——细字重衬线字体，带极端负字间距
- Zapier 橙色（`#ff4f00`）作为单一强调——鲜艳、温暖、克制使用
- 温暖中性调色板：边框（`#c5c0b1`）、弱化文字（`#939084`）、表面着色（`#eceae3`）
- 基础 8px 间距系统，行动按钮宽松内边距（20px 24px）
- 边框优先设计：`1px solid` 温暖灰边框定义结构而非阴影

## 2. 色彩系统与角色

### 主色
- **Zapier 黑** (`#201515`)：主文字、标题、深色按钮背景。带红色底调的温暖近黑色——永远不冷。
- **奶油白** (`#fffefb`)：页面背景、卡片表面、浅色按钮填充。非纯白；淡黄色温度是有意为之的。
- **乳白** (`#fffdf9`)：次级背景表面，细微的替代着色。与奶油白几乎无法区分，但创造了深度。

### 品牌强调
- **Zapier 橙色** (`#ff4f00`)：主要行动按钮、激活下划线指示器、强调边框。标志性颜色——鲜艳而温暖。

### 中性色系
- **深炭灰** (`#36342e`)：次级文字、页脚文字、强分隔线边框颜色。带 70% 不透明度变体的温暖深灰棕色。
- **暖灰** (`#939084`)：三级文字、弱化标签、时间戳风格内容。带绿色温暖底调的中间范围。
- **沙色** (`#c5c0b1`)：主要边框颜色、悬停状态背景、分隔线。Zapier 结构性元素的骨架。
- **浅沙** (`#eceae3`)：次级按钮背景、浅色边框、细微卡片表面。
- **中暖** (`#b5b2aa`)：备用边框调，用于特定 span 元素。

### 交互色
- **橙色行动按钮** (`#ff4f00`)：主要操作按钮和激活标签下划线。
- **深色行动按钮** (`#201515`)：配沙色悬停的次级深色按钮。
- **浅色行动按钮** (`#eceae3`)：配沙色悬停的三级/幽灵按钮。
- **链接默认** (`#201515`)：标准链接颜色，与正文匹配。
- **悬停下划线**：链接悬停时移除 `text-decoration: underline`（反向模式）。

### 遮罩与表面
- **半透明深色** (`rgba(45, 45, 46, 0.5)`)：遮罩按钮变体、类背景元素。
- **胶囊表面** (`#fffefb`)：带沙色边框的白色胶囊按钮。

### 阴影与深度
- **内嵌下划线** (`rgb(255, 79, 0) 0px -4px 0px 0px inset`)：激活标签指示器——使用内嵌盒阴影的橙色下划线。
- **悬停下划线** (`rgb(197, 192, 177) 0px -4px 0px 0px inset`)：非激活标签悬停——沙色下划线。

## 3. 字体规则

### 字体系列
- **展示**：`Degular Display` ——宽字间几何展示字体，用于英雄标题

- **编辑**：`GT Alpina` ——细字重衬线字体，用于编辑时刻

### 原则
- **三字体系统，明确角色**：Degular Display 仅在英雄比例上引起注意。Inter 处理所有功能性内容。GT Alpina 稀疏地添加编辑温度。
- **压缩展示**：Degular 在 0.90 行高创造垂直压缩的标题块，感觉现代而建筑感。
- **字重作为层级信号**：Inter 使用 400（阅读）、500（导航/强调）、600（标题/行动按钮）。Degular 使用 500（展示）和 600（按钮）。
- **大写用于标签**：区域标签（如"01 / 颜色"）和小型分类使用 `text-transform: uppercase` 配 0.5px 字间距。
- **负字间距体现优雅**：GT Alpina 使用 -1.6px 至 -1.92px 字间距，用于其细字重编辑标题。

## 5. 布局原则

### 间距系统
- 基础单位：8px
- 尺度：1px、4px、6px、8px、10px、12px、16px、20px、24px、32px、40px、48px、56px、64px、72px
- 行动按钮使用宽松内边距：大型 20px 24px，标准 8px 16px
- 区域内边距：64px-80px 垂直

### 网格与容器
- 最大内容宽度：约 1200px
- 英雄区：居中单列，带大型顶部内边距
- 功能区域：集成卡片的 2-3 列网格
- 全宽沙色边框分隔区域
- 页脚：多列深色背景（`#201515`）

### 空白哲学
- **温暖呼吸空间**：区域之间宽松的垂直间距（64px-80px），但内容区域相对密集——Zapier 在其奶油色画布中高效地打包信息。
- **建筑压缩**：Degular Display 标题在 0.90 行高垂直压缩，与周围开放的间距形成对比。
- **区域节奏**：整体奶油色背景，区域之间通过沙色边框而非背景颜色变化来分隔。

### 圆角尺度
- 紧凑（3px）：小型行内 span
- 标准（4px）：按钮（橙色行动按钮）、标签、小型元素
- 内容（5px）：卡片、链接、通用容器
- 舒适（8px）：特色卡片、大型按钮、标签
- 社交（14px）：社交图标按钮、胶囊状元素
- 胶囊（20px）：播放按钮、大型胶囊按钮、浮动操作

## 6. 深度与层次

| 层级 | 处理方式 | 用途 |
|------|----------|------|
| 平面（0 级） | 无阴影 | 页面背景、文字块 |
| 边框（1 级） | `1px solid #c5c0b1` | 标准卡片、容器、输入框 |
| 强边框（1b 级） | `1px solid #36342e` | 深色分隔线、强调区域 |
| 激活标签（2 级） | `rgb(255, 79, 0) 0px -4px 0px 0px inset` | 激活标签下划线（橙色） |
| 悬停标签（2b 级） | `rgb(197, 192, 177) 0px -4px 0px 0px inset` | 悬停标签下划线（沙色） |
| 焦点（无障碍） | `1px solid #ff4f00` 轮廓 | 交互元素的焦点环 |

**阴影哲学**：Zapier 刻意避免传统的基于阴影的层次。结构几乎完全通过边框定义——标准包容使用温暖沙色（`#c5c0b1`）边框，强调使用深炭灰（`#36342e`）边框。唯一类阴影的技术是用于标签下划线的内嵌盒阴影，其中 `0px -4px 0px 0px inset` 阴影创造底部栏指示器。这种边框优先的方法让设计保持接地气而非漂浮感。

### 装饰性深度
- 激活标签上的橙色内嵌下划线在元素底部创造视觉"重量"
- 沙色悬停下划线提供预览状态，不造成布局偏移
- 主内容中无背景渐变——奶油色画布保持一致
- 页脚使用完整深色背景（`#201515`）进行对比反转

## 7. 规范与禁忌

### 应该做
- 仅将 Degular Display 用于英雄级标题（40px+），配 0.90 行高实现压缩冲击力
- 所有功能性 UI 使用 Inter——导航、正文、按钮、标签
- 使用温暖奶油色（`#fffefb`）作为背景，永远不要使用纯白
- 文字使用 `#201515`，永远不要使用纯黑——红色温度很重要
- 将 Zapier 橙色（`#ff4f00`）保留给主要行动按钮和激活状态指示器
- 使用沙色（`#c5c0b1`）边框作为主要结构性元素而非阴影
- 大型行动按钮应用宽松按钮内边距（20px 24px），匹配 Zapier 宽阔的按钮风格
- 标签导航使用内嵌盒阴影下划线而非 border-bottom
- 区域标签和微型分类应用大写配 0.5px 字间距

### 不应该做
- 不要将 Degular Display 用于正文或 UI 元素——它仅用于展示
- 不要使用纯白（`#ffffff`）或纯黑（`#000000`）——Zapier 的调色板是暖色偏移的
- 不要给卡片应用盒阴影层次——使用边框代替
- 不要将 Zapier 橙色分散到 UI 各处——它保留给行动按钮和激活状态
- 不要在大型行动按钮上使用紧凑内边距——Zapier 的按钮是刻意宽阔的
- 不要忽视温暖中性系统——边框应该是 `#c5c0b1`，而非灰色
- 不要将 GT Alpina 用于功能性 UI——它仅作为细字重的编辑强调
- 不要给 GT Alpina 应用正字间距——它使用激进的负字间距（-1.6px 至 -1.92px）
- 不要对主要按钮使用圆形胶囊（9999px）——胶囊用于标签和社交图标

### 迭代指南
1. 始终使用温暖奶油色（`#fffefb`）背景，永远不要用纯白——温度定义了 Zapier
2. 边框（`1px solid #c5c0b1`）是结构骨架——避免阴影层次
3. Zapier 橙色（`#ff4f00`）是唯一的强调色；其他所有都是温暖中性色
4. 三种字体，严格角色：Degular Display（英雄）、Inter（UI）、GT Alpina（编辑）
5. 大型行动按钮需要宽松内边距（20px 24px）——Zapier 按钮感觉宽阔
6. 标签导航使用内嵌盒阴影下划线，而非 border-bottom
7. 文字始终是温暖的：`#201515` 用于深色，`#36342e` 用于正文，`#939084` 用于弱化
8. 区域分类使用 12-14px 大写标签，配 0.5px 字间距
