# 兰博基尼设计系统灵感

## 1. 视觉主题与氛围

兰博基尼的网站是一座黑暗的大教堂——一个数字舞台，纯黑色的表面无限延伸，每一个元素都像聚光灯下的机器一样从虚空中浮现。整个页面几乎完全是黑色的。不是深灰色，不是近黑色——而是真实的、不妥协的黑色（`#000000`），充满整个视口，拒绝让步。在这片深渊中，白色文字和兰博基尼金色（`#FFC000`）以外科手术般的精准度部署，创造出一种视觉语言，仿佛走进一场夜间赛车活动，每个表面都吸收光线，除了那些真正重要的东西。

英雄区是全视口视频——黑暗、电影感、沉浸式——展示赛事录像或车型发布，兰博基尼公牛标志如梦似幻地漂浮其上。导航极简主义：居中的公牛标志，左侧的"MENU"汉堡按钮，右侧的搜索/书签图标，全部以白色呈现在黑色画布上。没有边框，没有可见的导航容器，页眉没有背景色——只有白色标记漂浮在黑暗中。整体氛围是夜间奢华：独家、戏剧化，刻意令人敬畏。每个章节过渡都是在黑暗中滚动，进入下一个启示。

排版是这片黑暗的声音。LamboType——由 Character Type 和设计机构 Strichpunkt 为兰博基尼 2024 品牌焕新共同创作的定制新格罗特斯克字体——用于从 120px 大写展示标题到 10px 微型标签的所有内容。其独特的 12° 倾斜终端灵感来自兰博基尼超级跑车的空气动力学线条，比例从 Normal 到 Ultracompressed 宽度不等。标题以极大的比例全大写呼喊，行高紧凑（120px 时为 0.92），形成密集的文字块，感觉像是从钢铁上压印出来的。该字体携带六边形几何 DNA——由六边形、三臂星和圆形构成——这在整个界面中的六边形暂停按钮和 UI 图标中得到呼应。底层构建于 Bootstrap 网格，搭配 68 个 Element Plus/UI 组件，戏剧性表面之下是相当可观的技术基础设施。

**核心特征：**
- 真正的黑色（`#000000`）主导表面，白色和金色是唯一的缓解色彩
- LamboType 定制新格罗特斯克字体，带有受空气动力学车身线条启发的 12° 倾斜终端
- 兰博基尼金色（`#FFC000`）作为唯一强调色——专门用于主要 CTA 按钮
- 极大比例的全大写展示排版（120px、80px、54px），行高紧凑
- 全视口视频英雄区，呈现电影感的赛事/车型内容
- 按钮零圆角——锐利、有棱角、不妥协的矩形
- UI 元素中的六边形主题（暂停按钮、图标系统），呼应品牌几何形状
- Bootstrap 网格系统 + Element Plus/UI 68 个组件底层支撑
- 半透明幽灵按钮，白色边框透明度 50%，作为次要 CTA 模式

## 2. 色彩搭配与角色定义

### 主色
- **兰博基尼金色**（`#FFC000`）：标志性强调色——一种温暖、饱和的琥珀金色（rgb 255, 192, 0），专门用于主要操作按钮（"探索更多"、"购票"、"开始配置"）。整个界面中唯一的有色颜色，在黑色画布上点燃如同夜晚中切割的车灯
- **纯白**（`#FFFFFF`）：深色表面上的主要文字颜色、标志渲染、导航元素以及浅色模式按钮填充——从黑暗中说话的声音

### 次色与强调色
- **深金**（`#917300`）：金色按钮的悬停/按下状态——深琥珀色（rgb 145, 115, 0），加深金色以传达交互
- **金文本**（`#FFCE3E`）：略亮的金色变体（rgb 255, 206, 62），用于内联文字强调和高亮标签
- **青色脉冲**（`#29ABE2`）：电蓝青色（rgb 41, 171, 226），作为信息性强调和互动元素高亮出现
- **链接蓝**（`#3860BE`）：中蓝色（rgb 56, 96, 190），普遍用于所有文字颜色的链接悬停状态

### 表面与背景
- **绝对黑**（`#000000`）：主导表面颜色——用于页面背景、英雄区、页眉、页脚和大多数容器
- **木炭**（`#202020`）：提升的深色表面（rgb 32, 32, 32）——卡片、面板和文字容器的主要"深灰色"，位于黑色画布之上
- **深铁**（`#181818`）：微妙的表面变体（rgb 24, 24, 24）——几乎无法与黑色区分，用于页脚和深层部分
- **覆盖黑**（`rgba(0,0,0,0.7)`）：用于模态框和视频变暗的半透明覆盖层
- **近白**（`#F8F8F8`）：白色模式内容块的稀有浅色表面（rgb 248, 248, 248）
- **薄雾**（`#E6E6E6`）：次要浅色模式容器的浅灰色表面

### 中性色与文字
- **纯白**（`#FFFFFF`）：深色背景上的主要文字——标题、正文、导航标签
- **烟雾**（`#F5F5F5`）：深色表面上的次要文字——比纯白略柔和
- **石墨**（`#494949`）：浅色表面上的深灰色文字（rgb 73, 73, 73）
- **灰烬**（`#7D7D7D`）：柔和文字、时间戳和元数据的中间灰色（rgb 125, 125, 125）
- **钢铁**（`#969696`）：禁用文字和细微标签的较浅灰色（rgb 150, 150, 150）
- **板岩**（`#666666`）：次要内容的替代中灰色
- **铁灰**（`#555555`）：正文文字变体的深中灰
- **阴影**（`#313131`）：深色表面上白色过强时使用的非常深的灰色文字

### 语义色与强调色
- **青色脉冲**（`#29ABE2`）：用于信息性高亮和互动反馈
- **链接蓝**（`#3860BE`）：所有超链接的通用悬停状态
- **青色动作**（`#1EAEDB`）：透明/幽灵变体的按钮悬停背景（rgb 30, 174, 219）

### 渐变系统
- 色彩搭配中没有明确的渐变——深到浅的渐进通过表面层叠实现：`#000000` → `#181818` → `#202020` → `#494949` → `#7D7D7D`
- 视频英雄区使用内容本身的自然大气渐变
- 页面顶部渐变：全出血图像边缘的微妙深到更深的淡化

## 3. 排版规则

### 字体家族
- **展示与 UI**：`LamboType`、Roboto、Helvetica Neue、Arial——Character Type 为兰博基尼 2024 品牌焕新定制的新格罗特斯克字体。宽度从 Normal 到 Ultracompressed，字重从 Light（300）到 Black。具有受空气动力学车身几何形状启发的 12° 倾斜终端、六边形构造逻辑，以及对 200+ 种语言的支持，包括拉丁文、西里尔文和希腊文
- **回退/UI**：`Open Sans`——用于某些按钮/表单环境作为系统回退
- **无斜体变体**：营销网站上未观察到——品牌声音始终是直立的

### 原则
- **全大写是默认声音**：展示和特性标题普遍大写。这创造了一种呼喊、命令式的语调，与品牌的攻击性相匹配
- **极端的尺寸范围**：从 120px 英雄到 10px 微型标签——12:1 的比例，创造戏剧性的视觉层级
- **大尺寸紧凑行高**：展示尺寸使用 0.92-1.19 行高，创造密集、压缩的文字块，感觉像被压印而非排版
- **字重 400 主导**：与许多使用粗体强调的设计系统不同，兰博基尼的常规字重承担标题——字体本身如此独特，不需要字重变化
- **说明文字负字间距**：14px 说明文字的 -0.42px 字间距创造压缩、技术性的美感
- **微型文字正字间距**：10px 时 +0.225px 确保最小尺寸的可读性
- **单一字体纪律**：LamboType 处理所有内容——12° 倾斜终端和六边形几何形状提供跨所有尺寸的视觉一致性

## 4. 组件样式

### 按钮
所有按钮使用**零圆角**——锐利、有棱角的矩形，呼应兰博基尼车辆的攻击性线条。

**金色强调 CTA** — 主要操作：
- 默认：bg `#FFC000`（兰博基尼金色），文字 `#000000`，内边距 24px，字号 16px，字重 400，圆角 0px，无边框
- 悬停：bg `#917300`（深金），显著变暗
- 类名：`btn-accent btn-large`
- 用于："探索更多"、"购票"、"开始配置"

**透明幽灵** — 深色背景上的次要操作：
- 默认：bg 透明，文字 `#FFFFFF`，边框 1px solid `#FFFFFF`，内边距 16px，透明度 0.5
- 悬停：bg `#1EAEDB`（青色动作），文字白色，透明度 0.7
- 聚焦：bg `#1EAEDB`，边框 1px solid `#000000`，轮廓 2px solid `#000000`
- 用于：英雄区和深色面板上的次要 CTA

**白色填充** — 浅色模式主要：
- 默认：bg `#FFFFFF`，文字 `#202020`，无边框
- 用于：金色不适合的深色区域 CTA

**黑色填充** — 深色填充变体：
- 默认：bg `#000000`，文字 `#202020`
- 用于：浅色区域上的反转 CTA

**灰色中性** — 细微操作：
- 默认：bg `#969696`，文字 `#202020`
- 用于：次要/三级操作，徽章式按钮

### 卡片与容器
- 背景：黑色画布上的 `#202020`（木炭），或较浅区域上的 `#000000`
- 边框：`0px 1px solid #202020` 底部边框用于区域分隔线
- 圆角：0px（完全锐利的角）
- 阴影：最小化，使用覆盖透明度实现深度
- 内容：全出血摄影 + 白色叠加文字

### 输入框与表单
- 营销网站上表单存在感极低
- 开关元素：圆角 20px（唯一的圆形元素），边框 1px solid `#DDDDDD`
- Cookie 横幅输入样式：黑色上的白色文字，`#7D7D7D` 边框

### 导航
- **桌面端**：居中公牛标志，左侧带图标的"MENU"汉堡，右侧搜索图标 + 书签图标
- **背景**：透明（继承黑色页面背景）
- **固定**：固定在顶部，悬浮于内容之上
- **无可见边框或阴影**——元素漂浮在黑暗中
- **"MENU"标签**：14px 字重 400 的白色文字，大写，伴随汉堡图标
- **六边形主题**：英雄区的暂停按钮使用六边形轮廓形状

### 图像处理
- **英雄区**：全视口视频区域（100vh），包含电影感的赛事/车型录像
- **赛事摄影**：兰博基尼竞技场活动的全出血航拍镜头
- **车辆图像**：深色背景上的高对比度棚拍，全宽
- **宽高比**：主要为 16:9 及更宽，以实现电影感
- **深色渐变叠加**：视频顶部/底部边缘的微妙变暗，确保文字可读性

### 独特组件
- **六边形暂停按钮**：视频控件使用六边形轮廓（与字体的品牌几何 DNA 相匹配），位于英雄区右下角
- **进度条**：英雄区底部的细白线，指示视频/幻灯片进度
- **徽章/标签**：bg `#969696`，白色文字，内边距 8px，字号 10px，圆角 2px——微小的金属质感药丸

## 5. 布局原则

### 间距系统
- **基本单位**：8px
- **完整刻度**：2px、4px、5px、8px、10px、12px、15px、16px、20px、24px、32px、40px、48px、56px
- **按钮内边距**：16px（幽灵），24px（金色强调）
- **区域内边距**：垂直 48–56px，水平 40px
- **小间距**：2–5px 用于精细调整（徽章内边距、边框间距）

### 网格与容器
- **框架**：Bootstrap 网格系统（container + row + col）
- **最大宽度**：1440px（最大断点）
- **列数**：标准 12 列 Bootstrap 网格
- **全出血**：英雄区突破网格，填充视口边缘
- **内容区域**：在 1200px 最大宽度容器内居中

### 空白哲学
兰博基尼用黑暗作为空白。内容块之间宽阔的黑色空间与浅色设计中的空白具有相同的功能——创造呼吸空间，将每个元素提升到展品的地位。一个模型名称漂浮在黑色视口中央，具有与白墙上的画廊作品相同的视觉重量。颜色的缺席就是设计本身。

### 圆角刻度
| 数值 | 环境 |
|-------|---------|
| 0px | 所有元素的默认——按钮、卡片、容器、图像 |
| 1px | 微妙的 span 元素 |
| 2px | 徽章、关闭按钮、Cookie 元素——几乎无感知 |
| 20px | 仅限开关——唯一的圆形元素 |

## 6. 深度与层次

| 层级 | 处理方式 | 用途 |
|-------|-----------|-----|
| 第 0 层（深渊） | `#000000` 平面 | 页面背景，最深层 |
| 第 1 层（表面） | `#181818` 或 `#202020` | 卡片、内容面板、提升区域 |
| 第 2 层（覆盖） | `rgba(0,0,0,0.7)` | 模态框背景，视频变暗 |
| 第 3 层（雾） | `rgba(0,0,0,0.5)` | 较轻的覆盖层，悬停状态 |
| 第 4 层（薄雾） | `rgba(0,0,0,0.25)` | 微妙的深度暗示 |

### 阴影哲学
兰博基尼通过表面颜色层叠而非阴影来实现深度。在黑色画布上，传统的投影是不可见的——相反，系统通过从绝对黑到逐渐变浅的深灰色来创造层次：`#000000` → `#181818` → `#202020` → `#494949`。这种"黑暗渐变"方法意味着提升的元素在字面上比其周围环境更亮，颠覆了传统的阴影模型。

### 装饰性深度
- 全出血视频通过电影照明提供大气深度
- 六边形暂停按钮以细白轮廓描边漂浮
- 英雄区底部的进度条创造微妙的地平线
- UI 元素上无渐变、发光或模糊效果——摄影提供所有视觉丰富性

## 7. 注意事项

### 应该做
- 使用绝对黑色（`#000000`）作为主要背景——不要用深灰色代替
- 将兰博基尼金色（`#FFC000`）专门用于主要 CTA 按钮——不用于装饰目的
- 将所有展示标题设置为大写 LamboType——品牌声音始终在喊叫
- 在按钮和卡片上使用零圆角——锐利的角度是不可妥协的
- 在展示文字上保持紧凑的行高（0.92–1.19），以创造密集的建筑式文字块
- 在深色背景上使用透明幽灵按钮（白色边框，50% 透明度）作为次要 CTA
- 让全视口视频/摄影承载情感重量——UI 是基础设施，不是装饰
- 将六边形几何形状保留给 UI 图标和视频控制按钮
- 在标题中使用字重 400（常规）——字体本身足够独特，不需要粗体强调
- 保持灰色调无彩色——所有中性色都是纯灰色，没有颜色色调

### 不应该做
- 不要在金色之外引入额外的强调色——单色加金色系统是神圣的
- 不要在按钮或卡片上应用圆角——曲线边缘与有棱角的车辆美学相矛盾
- 不要以斜体或装饰风格使用 LamboType——品牌始终是直立和直接的
- 不要在按钮或表面上添加渐变——深度来自表面层叠，而非混合
- 不要将浅色背景作为主要画布——黑暗是默认状态，光明是例外
- 不要在展示标题中混入小写——大写惯例传达权威和力量
- 不要添加带有缩放或平移的悬停动画——交互应该只有颜色变化（背景/透明度变化）
- 不要将 Open Sans 用于展示文字——LamboType 必须处理所有可见排版
- 不要创建包含许多小元素的拥挤布局——兰博基尼的设计是关于单一、大胆的陈述
- 不要对元素应用阴影——在黑色画布上，阴影毫无意义；使用表面颜色变化代替

| 大桌面 | 1280-1440px | 完整布局，英雄 120px 展示，最大宽度容器 |
| 宽屏 | >1440px | 内容居中，边距扩大，英雄填充视口 |

### 触摸目标
- 金色 CTA 按钮：24px 内边距时最小高度 48px+（超过 WCAG 44×44px）
- 幽灵按钮：16px 内边距时 48px+
- 汉堡菜单：大触摸目标（约 48px 正方形）
- 六边形暂停按钮：约 48px 直径

### 折叠策略
- **导航**：始终基于汉堡（"MENU" + 图标）——在任何断点都不展开水平导航
- **英雄视频**：在所有断点保持全视口高度，调整 object-fit
- **展示文字**：从 120px（桌面）→ 80px（平板）→ 54px/40px（手机）缩放
- **按钮布局**：桌面并排，手机垂直堆叠
- **网格列**：3 列 → 2 列 → 1 列渐进
- **区域间距**：垂直内边距从 56px → 40px → 24px 缩减

### 图像行为
- 英雄视频使用 `object-fit: cover` 在所有尺寸保持电影感构图
- 车辆图像在其容器内保持宽高比缩放
- 赛事摄影在窄屏幕上裁剪至视口宽度
- 背景图像在边缘变暗，以在所有视口上保持文字对比度
