# Design System Inspiration of Warp

## 1. 视觉主题与氛围

Warp 的网站给人一种在深林中篝火旁静坐的感觉——温暖、深沉，充满着静谧的自信。与大多数开发者工具偏好的冷蓝色调黑不同，Warp 将一切包裹在温暖的近黑色中，感觉像是炭化的木材或深邃的大地。文字也不是纯白——而是暖羊皮纸色（`#faf9f6`），这种几乎无感知的奶油色柔化了每一个标题，让深色画布感觉温馨而非严峻。

排版是秘密武器：Matter，一种具有独特个性的几何无衬线字体，在几乎所有文字中都以常规字重部署。这种字体选择对于开发者工具来说是不寻常的——Matter 带有一种柔软性和人文气息，传递着"这个终端是为所有人而建，不只是老手"的信号。结合标题上的紧凑行高和受控的负字母间距，效果同时兼具精炼感和亲和力。自然摄影被编织在终端截图之间，创造出一种视觉语言，传递着：这个工具让你更接近心流，更接近平静的工作状态。

整体设计理念是通过温暖实现克制。最少的色彩（几乎单色的暖灰色）、最少的装饰，以及专注于在电影般的深色景观中展示产品。这是一家用生活方式品牌的方式营销的终端公司。

**核心特征：**
- 温暖的深色背景——不是冷黑，而是带有暖灰色底调的土质近黑色
- 暖羊皮纸色（`#faf9f6`）文字而非纯白——微妙的奶油色温暖感
- Matter 字体家族（常规字重）——几何但平易近人，不像典型的开发者工具字体
- 自然摄影与产品截图交错呈现——生活方式遇见开发者工具
- 几乎单色的暖灰色调色板——没有大胆的强调色
- 大写标签搭配宽字母间距（2.4px）用于分类——编辑风格的标记
- 胶囊形深色按钮（`#353534`，50px 圆角）——克制、静音的行动号召

## 2. 色彩体系与角色

### 主色
- **暖羊皮纸色** (`#faf9f6`)：主要文字颜色——几乎带有奶油感的米白色，柔化每一个表面
- **大地灰** (`#353534`)：按钮背景、深色交互表面——温暖，不冷漠
- **深邃虚空**（近黑色，页面背景）：温暖深色画布，来自页面背景色

### 辅助色与强调色
- **石灰色** (`#868584`)：次要文字、静音描述——温暖的中灰色
- **灰烬灰** (`#afaeac`)：正文、按钮文字——主要阅读颜色
- **紫调灰** (`#666469`)：链接文字，带有微妙紫色底调——内容中带下划线的链接

### 表面与背景
- **霜雾幕** (`rgba(255, 255, 255, 0.04)`)：用于表面差异化的超微妙白色叠层
- **薄雾边框** (`rgba(226, 226, 226, 0.35)` / `rgba(227, 227, 227, 0.337)`)：卡片包含的半透明边框
- **半透明羊皮纸色** (`rgba(250, 249, 246, 0.9)`)：略微透明的主要表面，允许深度透出

### 中性色与文字
- **暖羊皮纸色** (`#faf9f6`)：标题、高强调文字
- **灰烬灰** (`#afaeac`)：正文段落、描述
- **石灰色** (`#868584`)：次要标签、低调信息
- **静音紫** (`#666469`)：带下划线的链接、三级内容
- **深炭灰** (`#454545` / `#353534`)：边框、按钮背景

### 语义色与强调色
- Warp 作为几乎单色的系统运作——没有大胆的强调色
- 交互状态通过不透明度变化和下划线装饰传达，而非颜色变化
- 任何强调色都会破坏温暖、克制的调色板

### 渐变系统
- 营销网站上没有明确的渐变
- 深度通过分层的半透明表面和摄影创造，而非色彩渐变

## 3. 排版规则

### 字体家族
- **展示与正文**：`Matter Regular`——带有柔和个性的几何无衬线字体。备选字体：`Matter Regular Placeholder`，系统无衬线字体
- **中等字重**：`Matter Medium`——字重 500 变体，用于强调。备选字体：`Matter Medium Placeholder`
- **方形变体**：`Matter SQ Regular`——用于特定展示上下文的方形变体。备选字体：`Matter SQ Regular Placeholder`
- **UI 补充**：`Inter`——用于特定 UI 元素。备选字体：`Inter Placeholder`
- **等宽展示**：`Geist Mono`——用于代码/终端展示标题
- **等宽正文**：`Matter Mono Regular`——定制等宽配套字体。备选字体：`Matter Mono Regular Placeholder`

### 排版原则
- **常规字重主导**：几乎所有文字使用字重 400（常规）——甚至标题也是如此。Matter Medium（500）仅出现在卡片标题和按钮等强调场景。这创造出一种异常平静、均匀的排版质感
- **大写作为编辑标记**：小标签和类别使用大写 transform 搭配宽字母间距（1.4px–2.4px），创造出杂志编辑式的分类系统
- **温暖可读性**：Matter 的几何柔软感 + 暖色调文字（#faf9f6）+ 受控的负字母间距三者结合，在深色表面上创造出轻松自然的人文阅读体验
- **无粗体展示**：任何地方都不使用粗体（700+）字重——克制是理念

## 4. 组件样式

### 按钮
- **深色胶囊形**：`#353534` 背景，灰烬灰（`#afaeac`）文字，胶囊形（50px 圆角），`10px` 内边距。主要行动号召——温暖、低调、含蓄
- **磨砂标签**：`rgba(255, 255, 255, 0.16)` 背景，黑色文字（`rgb(0, 0, 0)`），矩形（6px 圆角），`1px 6px` 内边距。小型内联类标签按钮
- **幽灵形**：无可见背景，纯文字，悬停时下划线装饰
- **悬停**：微妙的不透明度或亮度变化——没有戏剧性的颜色变化

### 卡片与容器
- **摄影卡片**：全出血自然图片搭配叠加文字，8px–12px 圆角
- **终端截图卡片**：产品 UI 嵌入深色容器，圆角（8px–12px）
- **带边框卡片**：半透明边框（`rgba(226, 226, 226, 0.35)`）用于包含，12px–14px 圆角
- **悬停**：极少——内容卡片在悬停时不会剧烈变化，保持平静美学

### 输入框与表单
- 营销网站上表单存在感较弱
- 带暖灰色文字的深色背景输入框
- 焦点：边框亮度提升，无彩色环（与单色调色板一致）

### 导航
- **顶部导航**：深色背景，暖羊皮纸色品牌文字，链接使用 16px Matter Regular
- **链接颜色**：静音导航用石灰色（`#868584`），激活/悬停用暖羊皮纸色
- **行动号召按钮**：导航末端的深色胶囊形（#353534）——克制，不吸引眼球
- **移动端**：折叠为简化导航
- **粘性**：导航在滚动时保持固定

### 图片处理
- **自然摄影**：景观、森林、黄金时刻场景——开发者工具中完全独特
- **终端截图**：产品 UI 在真实终端窗口框架中展示
- **混合构图**：自然图片和终端截图交错排列，创造出生活方式遇见工具的叙事
- **全出血**：图片通常跨越容器全宽，8px 圆角
- **视频**：视频元素具有 10px 圆角

### 推荐语区块
- 社会证明区域（"不必只听我们的"）搭配引用语
- 静音风格与整体克制感一致

## 5. 布局原则

### 间距系统
- **基础单位**：8px
- **刻度**：1px、4px、5px、8px、10px、12px、14px、15px、16px、18px、24px、26px、30px、32px、36px
- **区块内边距**：主要区块之间 80px–120px 垂直间距
- **卡片内边距**：16px–32px 内部间距
- **组件间隙**：相关元素之间 8px–16px

### 网格与容器
- **最大宽度**：约 1500px 容器（断点在 1500px），居中
- **列模式**：全宽英雄，带摄影的 2 列功能区块，单列推荐语
- **电影般的布局**：宽容器让摄影自由呼吸

### 留白理念
- **广阔而温暖**：区块之间充裕的间距——深色背景创造出一种温暖的虚空，感觉是沉思性的而非空旷的
- **摄影作为留白**：自然图片作为密集产品信息之间的视觉喘息空间
- **编辑节奏**：布局像杂志一样阅读——每个区块都是刻意的翻页时刻

### 圆角刻度
- **4px**：小型交互元素——按钮、标签
- **5px–6px**：标准组件——链接、小容器
- **8px**：图片、视频容器、标准卡片
- **10px**：视频元素、中等容器
- **12px**：功能卡片、大图片
- **14px**：大容器、突出卡片
- **40px**：大圆角区块
- **50px**：胶囊形按钮——主要行动号召
- **200px**：进度条——完整胶囊形

## 6. 深度与层次

| 层级 | 处理方式 | 用途 |
|-------|-----------|-----|
| 0 级（平面） | 无阴影，深色背景 | 页面画布，大多数表面 |
| 1 级（薄雾幕） | `rgba(255, 255, 255, 0.04)` 叠层 | 微妙的表面差异化 |
| 2 级（边框） | `rgba(226, 226, 226, 0.35) 1px` 边框 | 卡片包含、区块分隔 |
| 3 级（环境光） | `rgba(0, 0, 0, 0.2) 0px 5px 15px`（从设计推断） | 图片容器、浮动元素 |

### 阴影理念
Warp 的层次系统异常平面——营销网站上几乎零阴影使用。深度通过以下方式传达：
- **半透明边框**代替阴影——35% 不透明度的边框创造出幽灵般的包含感
- **摄影分层**——图片创造出自然的深度，无需人工阴影
- **表面不透明度变化**——`rgba(255, 255, 255, 0.04)` 叠层创造出几乎难以察觉的层次差异
- 效果是平静而踏实的——没有什么在浮动，一切都在栖息

### 装饰性深度
- **摄影作为深度**：自然图片创造出阴影无法匹敌的大气深度
- **无玻璃或虚化效果**：设计完全避免流行的玻璃拟态效果
- **温暖环境光**：任何光晕都来自摄影的自然照明，而非人工 CSS

## 7. 应做与禁止

### 应做
- 使用暖米白（`#faf9f6`）而非纯白作为文字——奶油底调至关重要
- 保持按钮克制低调——深色填充（#353534）搭配静音文字（#afaeac），无明亮行动号召
- 几乎所有内容应用 Matter Regular（字重 400）——甚至标题也是如此。仅强调时使用 Medium（500）
- 使用宽字母间距（1.4px–2.4px）的大写标签进行分类
- 自然摄影与产品截图交错排列——这是品牌身份的核心
- 保持几乎单色的暖灰色调色板——无大胆强调色
- 卡片包含使用半透明边框（`rgba(226, 226, 226, 0.35)`）而非阴影
- 标题保持负字母间距（-0.4px 至 -2.4px），用于 Matter 的压缩展示处理

### 禁止
- 文字不要使用纯白（#ffffff）——始终使用暖羊皮纸色（#faf9f6）
- 不要添加大胆强调色（蓝色、红色、绿色）——该系统有意为单色暖灰色
- 不要对任何文字应用粗体字重（700+）——Warp 从不超过 Medium（500）
- 不要使用重投影阴影——深度来自边框、摄影和不透明度变化
- 不要创建冷色调或蓝色调的深色背景——温暖感至关重要
- 不要添加装饰性渐变或发光效果——摄影提供所有视觉趣味
- 不要使用紧凑压缩的布局——编辑间距是充裕而沉思性的
- 除 Matter 字体家族 + Inter 补充外不要混入其他字体

## 8. 响应式行为

### 断点
| 名称 | 宽度 | 关键变化 |
|------|-------|-------------|
| 移动端 | <810px | 单列，堆叠区块，英雄文字缩至约 48px，汉堡导航 |
| 平板 | 810px–1500px | 2 列功能开始显现，摄影缩放，导航链接部分可见 |
| 桌面 | >1500px | 完整电影般布局，80px 英雄展示，并排摄影+文字 |

### 触控目标
- 胶囊形按钮：50px 圆角搭配 10px 内边距——舒适的触控目标
- 导航链接：16px 文字搭配周围内边距以满足无障碍性
- 移动端行动号召：移动端全宽胶囊形，便于拇指触达

### 折叠策略
- **导航**：完整横向导航 → 简化移动端导航
- **英雄文字**：80px 展示 → 56px → 48px 跨断点变化
- **功能区块**：并排摄影+文字 → 垂直堆叠
- **摄影**：在容器内缩放，保持电影般的宽高比
- **区块间距**：按比例缩小——宽敞桌面 → 紧凑移动端

### 图片行为
- 自然摄影响应式缩放，保持宽电影般的宽高比
- 终端截图在响应式容器内保持宽高比
- 视频元素随 10px 圆角一同缩放
- 无艺术指导变化——跨断点使用相同的构图

## 9. 智能助手提示指南

### 快速颜色参考
- 主要文字：暖羊皮纸色（`#faf9f6`）
- 次要文字：灰烬灰（`#afaeac`）
- 三级文字：石灰色（`#868584`）
- 按钮背景：大地灰（`#353534`）
- 边框：薄雾边框（`rgba(226, 226, 226, 0.35)`）
- 背景：深暖近黑色（页面背景）

### 组件提示示例
- 「在温暖深色背景上创建英雄区，使用 80px Matter Regular 标题，暖羊皮纸色（#faf9f6），行高 1.0，字母间距 -2.4px，以及深色胶囊形按钮（#353534，50px 圆角，#afaeac 文字）」
- 「设计带半透明边框（rgba(226,226,226,0.35)）的功能卡片，12px 圆角，暖深色背景，24px Matter Regular 标题，灰烬灰（#afaeac）18px 正文」
- 「使用 Matter Regular 12px、大写 transform、字母间距 2.4px、石灰色（#868584）颜色构建分类标签——编辑杂志风格」
- 「创建带暖羊皮纸色引用的推荐语区块，Matter Regular 24px，来源使用石灰色（#868584），深色背景，极少装饰」
- 「设计带暖深色背景的导航栏，16px Matter Regular 链接，石灰色（#868584），悬停变为暖羊皮纸色（#faf9f6），右侧深色胶囊形行动号召按钮（#353534）」

### 迭代指南
精炼使用此设计系统生成的现有界面时：
1. 验证文字颜色为暖羊皮纸色（#faf9f6）而非纯白——温暖感微妙但至关重要
2. 确保所有按钮使用克制的深色调色板（#353534）——无明亮或彩色行动号召
3. 检查 Matter Regular（400）是默认字重——Medium（500）仅用于强调
4. 确认大写标签具有宽字母间距（1.4px–2.4px）——紧凑大写感觉不对
5. 整体基调应该感觉温暖而平静，像一本设计精良的杂志——不激进、不科技炫耀
