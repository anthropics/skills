# 设计系统：Wise

本项目使用 **Wise** 风格设计系统。

> **必须**：编写任何 UI 代码前，请先完整阅读 `DESIGN.md`。

## 快速使用

```bash
# 将此目录复制到项目根目录：
cp DESIGN.md /your-project/DESIGN.md
```

或重新安装：

```bash
npx getdesign@latest add wise
```

来源：https://getdesign.md/wise/design-md

## AI 编程助手规则

当项目根目录存在 `DESIGN.md` 时：

1. **颜色** — 仅使用颜色调色板中的 hex 值，禁止直接写颜色字面量。
2. **字体** — 仅使用字体规范中定义的字体系列和字号。
3. **组件** — 按照文档中的按钮、卡片、表单规范严格执行。
4. **间距** — 使用间距规范中的值，基础单位为 8px（另有说明除外）。
5. **圆角** — 按照圆角规范中各组件类型的对应值执行。
6. **阴影** — 按照阴影层级规范应用对应深度的阴影。
7. **一致性** — 每个 UI 元素均需可追溯至 `DESIGN.md` 中的某个设计令牌。

## 使用方式

告知 AI 助手：
> "请使用 DESIGN.md 中定义的设计系统构建此界面"

AI 将读取 `DESIGN.md` 并自动生成风格一致的代码。
