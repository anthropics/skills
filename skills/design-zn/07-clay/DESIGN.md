# Clay 设计系统灵感参考

## 1. 视觉主题与氛围

Clay 的网站是一场温暖的、充满趣味的色彩庆典，将 B2B 数据丰富视为工艺而非企业苦差。设计语言建立在暖奶油背景（`#faf9f7`）和燕麦色调边框（`#dad4c8`、`#eee9df`）的基础上，赋予每一个表面手工纸张般的触感品质。在这片工匠画布之上，一系列鲜艳的色板以个性迸发——抹茶绿、沙冰青、柠檬金、山芋紫、石榴粉、蓝莓藏青和火龙果洋红——每一种都像果汁吧的口味名称，而非企业 UI 套件中的颜色。

排版由 Roobert 主导，这是一种充满个性的几何无衬线字体，搭载了一套丰富的 OpenType 风格集（`"ss01"`、`"ss03"`、`"ss10"`、`"ss11"`、`"ss12"`），赋予文字与众不同的、略带怪异的个性。在展示尺寸（80px，字重 600）下，Roobert 使用激进的负字间距（-3.2px），将标题压缩成简洁的、广告牌式的表达。Space Mono 作为等宽配套字体用于代码和技术标签，完成工艺遇技术的二重性。

Clay 真正独特的地方在于其悬停微动画：按钮在悬停时微微旋转（`rotateZ(-8deg)`），向上位移（`translateY(-80%)`），背景变为对比色板颜色，并投下硬偏移阴影（`rgb(0,0,0) -7px 7px`）。这种俏皮的悬停行为——按钮字面上在交互时倾斜跳跃——创造出一种实体愉悦感，在 B2B 软件中极为罕见。配合慷慨圆润的容器（24px–40px 圆角）、与实线边框并存的虚线边框，以及包含内嵌高亮的多层阴影系统，Clay 感觉像是由真正热爱制作的人设计出的设计系统。

**核心特征：**
- 暖奶油画布（`#faf9f7`）配燕麦色调边框（`#dad4c8`）——工匠风，非临床风
- 命名色板调色板：抹茶、沙冰、柠檬、山芋、石榴、蓝莓、火龙果
- Roobert 字体，带 5 个 OpenType 风格集——怪趣的几何个性
- 俏皮悬停动画：rotateZ(-8deg) + translateY(-80%) + 硬偏移阴影
- Space Mono 用于代码和技术标签
- 慷慨圆角：24px 卡片，40px 章节，1584px 胶囊
- 混合边框样式：同一界面中实线 + 虚线并存
- 带内嵌高亮的多层阴影：`0px 1px 1px` + `-1px inset` + `-0.5px`

## 2. 色彩体系与角色分工

- **Clay 黑** (`#000000`)：文字、标题、定价卡片文字，`--_theme--pricing-cards---text`

- **暖奶油** (`#faf9f7`)：页面背景——温暖的、纸张般的画布

### 阴影
- **Clay 阴影** (`rgba(0,0,0,0.1) 0px 1px 1px, rgba(0,0,0,0.04) 0px -1px 1px inset, rgba(0,0,0,0.05) 0px -0.5px 1px`)：带内嵌高亮的多层——标志性用法
- **硬偏移** (`rgb(0,0,0) -7px 7px`)：悬停状态——俏皮的硬阴影

- **主字体**：`Roobert`，备用字体：`Arial`

### 排版原则
- **五个风格集作为身份**：`"ss01"`、`"ss03"`、`"ss10"`、`"ss11"`、`"ss12"` 的组合在 Roobert 上创造独特的排版个性。`ss01` 保留给标题和强调——正文省略它，通过字形变化创造微妙层级。
- **激进展示压缩**：80px 时 -3.2px，60px 时 -2.4px——与最宽裕的正文间距（1.60 行高）形成最鲜明的对比，创造戏剧性反差。
- **字重 600 用于标题，500 用于 UI，400 用于正文**：干净的三层系统，每种字重有严格的角色。
- **大写标签配正字间距**：12px 大写文字 1.08px 字间距创造系统化的寻路图案。

## 4. 组件样式

### 按钮

**主要（透明配悬停动画）**
- 背景：透明（`rgba(239, 241, 243, 0)`）
- 文字：`#000000`
- 内边距：6.4px 12.8px
- 边框：无（或描边变体使用 `1px solid #717989`）
- 悬停：背景切换至色板颜色（如 `#434346`），文字变白，`rotateZ(-8deg)`，`translateY(-80%)`，硬阴影 `rgb(0,0,0) -7px 7px`
- 焦点：`rgb(20, 110, 245) solid 2px` 轮廓

**白色实色**
- 背景：`#ffffff`
- 文字：`#000000`
- 内边距：6.4px
- 悬停：燕麦 200 色板颜色，旋转 + 阴影动画
- 用途：有色章节上的主 CTA

**幽灵描边**
- 背景：透明
- 文字：`#000000`
- 内边距：8px
- 边框：`1px solid #717989`
- 圆角：4px
- 悬停：火龙果色板颜色，白色文字，旋转动画

### 卡片与容器
- 背景：奶油画布上的 `#ffffff`
- 边框：`1px solid #dad4c8`（暖燕麦）或 `1px dashed #dad4c8`
- 圆角：12px（标准卡片），24px（功能卡片/图片），40px（章节容器/页脚）
- 阴影：`rgba(0,0,0,0.1) 0px 1px 1px, rgba(0,0,0,0.04) 0px -1px 1px inset, rgba(0,0,0,0.05) 0px -0.5px 1px`
- 彩色章节背景使用色板调色板（抹茶、沙冰、山芋、柠檬）

### 输入框与表单
- 文字：`#000000`
- 边框：`1px solid #717989`
- 圆角：4px
- 焦点：`rgb(20, 110, 245) solid 2px` 轮廓

### 导航
- 奶油背景上的粘性顶部导航
- Roobert 15px 字重 500 用于导航链接
- Clay 标志左对齐
- CTA 按钮右对齐，带胶囊圆角
- 底部边框：`1px solid #dad4c8`
- 移动端：767px 处汉堡折叠

### 图片处理
- 带燕麦边框的白色卡片内的产品截图
- 带色板背景色的彩色插图章节
- 图片 8px–24px 圆角
- 全宽彩色章节背景

### 特色组件

**色板色彩章节**
- 全宽章节，带色板颜色背景（抹茶绿、沙冰青、山芋紫、柠檬金）
- 深色色板上白色文字，浅色色板上黑色文字
- 每个章节通过颜色讲述不同的产品故事

**俏皮悬停按钮**
- 悬停时旋转 -8deg + 向上位移
- 硬偏移阴影（`-7px 7px`）而非柔和模糊
- 背景过渡至对比色板颜色
- 创造出实体的、玩具般的交互质量

**虚线边框元素**
- 虚线边框（`1px dashed #dad4c8`）与实线边框并存
- 用于次级容器和装饰元素
- 增添手绘的、工艺般的质量

## 5. 布局原则

### 间距系统
- 基础单位：8px
- 比例：1px、2px、4px、6.4px、8px、12px、12.8px、16px、18px、20px、24px

### 网格与容器
- 居中最大内容宽度
- 功能章节在白色卡片和彩色色板背景之间交替
- 卡片网格：桌面端 2–3 列
- 打破网格的全宽彩色章节
- 带慷慨 40px 圆角容器的页脚

### 留白哲学
- **温暖、宽裕的呼吸感**：奶油背景在内容块之间提供温暖的休息。间距宽裕但非严苛——感觉像一张摆放精良的餐桌，充满欢迎感。
- **颜色作为空间节奏**：交替的色板彩色章节通过色相而非仅仅留白创造视觉节奏。每个彩色章节都是其自己的"房间"。
- **卡片内部的工艺密度**：在卡片内，内容紧凑而组织有序，与外部宽裕的间距形成对比。

### 圆角比例
- 锐利（4px）：幽灵按钮、输入框
- 标准（8px）：小型卡片、图片、链接
- 徽章（11px）：标签徽章
- 卡片（12px）：标准卡片、按钮
- 功能（24px）：功能卡片、图片、面板
- 章节（40px）：大型章节、页脚、容器
- 胶囊（1584px）：CTA、胶囊形按钮

## 6. 层深与高度

| 级别 | 处理方式 | 用途 |
|-------|-----------|-----|
| 平面（第 0 级） | 无阴影，奶油画布 | 页面背景 |
| Clay 阴影（第 1 级） | `rgba(0,0,0,0.1) 0px 1px 1px, rgba(0,0,0,0.04) 0px -1px inset, rgba(0,0,0,0.05) 0px -0.5px` | 卡片、按钮——带内嵌高亮的多层 |
| 悬停硬（第 2 级） | `rgb(0,0,0) -7px 7px` | 悬停状态——俏皮的硬偏移阴影 |
| 焦点（第 3 级） | `rgb(20, 110, 245) solid 2px` | 键盘焦点环 |

**阴影哲学**：Clay 的阴影系统独特地分为三层：向下投射（`0px 1px 1px`）、向上内嵌高亮（`0px -1px 1px inset`）和微妙边缘（`0px -0.5px 1px`）。这创造出一种"被压入黏土"的质量，元素既像是凸起的又像是嵌入的——就像黏土板上被印章压入的内容。悬停硬阴影（`-7px 7px`）是刻意的复古图形风格，呼应印刷时代的投影，增添实体的趣味性。

### 装饰性层深
- 全宽色板彩色章节通过色彩对比创造戏剧性深度
- 虚线边框在实线边框旁边增添视觉质感
- 带暖色、有机艺术风格的产品插图

## 7. 应做与禁止

### 应做
- 使用暖奶油（`#faf9f7`）作为页面背景——暖意即是身份
- 在 Roobert 标题上应用全部 5 个 OpenType 风格集：`"ss01", "ss03", "ss10", "ss11", "ss12"`
- 使用命名色板调色板（抹茶、沙冰、柠檬、山芋、石榴、蓝莓）作为章节背景
- 应用俏皮悬停动画：`rotateZ(-8deg)`、`translateY(-80%)`、硬阴影 `-7px 7px`
- 使用暖燕麦边框（`#dad4c8`）——非中性灰
- 混合实线和虚线边框增加视觉多样性
- 使用慷慨圆角：卡片 24px，章节 40px
- 标题严格使用字重 600，UI 使用 500，正文使用 400

### 禁止
- 不要使用冷灰背景——暖奶油（`#faf9f7`）是不可妥协的
- 不要使用中性灰边框（`#ccc`、`#ddd`）——始终使用暖燕麦色调
- 同一章节中不要混用超过 2 种色板颜色
- 不要跳过 OpenType 风格集——它们定义了 Roobert 的个性
- 不要使用微妙的悬停效果——旋转 + 硬阴影是标志性交互
- 功能卡片上不要使用小圆角（<12px）——慷慨的圆润是结构性的
- 不要使用标准阴影（模糊型）——Clay 使用硬偏移和多层内嵌
- 不要忘记带 1.08px 字间距的大写标签——它们是寻路系统

## 8. 响应式行为

### 断点
| 名称 | 宽度 | 主要变化 |
|------|-------|-------------|
| 小型移动端 | <479px | 单列，紧凑内边距 |
| 移动端 | 479–767px | 标准移动端，堆叠布局 |
| 平板 | 768–991px | 2 列网格，精简导航 |
| 桌面端 | 992px+ | 完整布局，3 列网格，扩展章节 |

### 触摸目标
- 按钮：最低 6.4px + 12.8px 内边距，确保足够触摸面积
- 导航链接：15px 字体配宽裕间距
- 移动端：全宽按钮，便于轻触

### 折叠策略
- 主图：80px → 60px → 更小展示文字
- 导航：水平 → 767px 处汉堡折叠
- 功能章节：多列 → 堆叠
- 彩色章节：保持全宽但压缩内边距
- 卡片网格：3 列 → 2 列 → 单列

### 图片行为
- 产品截图按比例缩放
- 彩色章节插图适应视口宽度
- 圆角在断点间保持

### 迭代指南
1. 从暖奶油（#faf9f7）开始——绝不使用冷白
2. 色板颜色用于完整章节，而非小型强调——大胆使用抹茶、沙冰、山芋
3. 燕麦边框（#dad4c8）随处可见——虚线变体用于装饰
4. OpenType 风格集是必需的——它们使 Roobert 看起来像 Roobert
5. 悬停动画是标志性的——旋转 + 硬阴影，而非微妙渐变
6. 慷慨圆角：24px 卡片，40px 章节——没有任何锐利或企业感
7. 三种字重：600（标题）、500（UI）、400（正文）——严格角色分工
