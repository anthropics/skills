# Design System Inspiration of Figma

## 1. 视觉主题与氛围

Figma 的界面是一件自我设计的设计工具杰作——在排版精妙程度上堪称教科书级别。其定制可变字体（figmaSans）在极细（字重 320）到粗体（字重 700）之间流畅过渡，中间还停驻于大多数字体系统从未探索过的非常规节点（330、340、450、480、540）。这种对字重的精细把控赋予每个文字元素精准校准的视觉重量，通过微观差异建立层次感，而非借助"常规 vs 粗体"这种粗糙的手段。

该页面呈现出一种迷人的双重性：界面框架严格使用黑白两色（字面意义上仅检测到 `#000000` 和 `#ffffff`），而英雄区和产品展示区则迸发出充满活力的多色渐变——电光绿、明亮黄、深紫和热粉红。这种分离意味着设计系统本身是无色的，将产品的彩色输出视为英雄内容。Figma 的营销页面本质上是一面白色画廊墙，用于展示彩色艺术品。

除了可变字体之外，Figma 还有另一个独特之处：圆形与胶囊形几何。按钮使用 50px 圆角半径（胶囊形）或 50%（图标按钮的完美圆形），营造出一种有机的、工具调色板般的质感。虚线轮廓的焦点指示器（`dashed 2px`）是一种刻意的设计选择，呼应了 Figma 编辑器中的选择手柄——网站的 UI 语言引用了产品的 UI 语言。

**核心特征：**
- 定制可变字体（figmaSans），具有非常规字重节点：320、330、340、450、480、540、700
- 界面框架严格使用黑白两色——颜色仅存在于产品内容中
- figmaMono 用于大写技术标签，搭配宽字母间距
- 胶囊形（50px）和圆形（50%）按钮几何
- 虚线焦点轮廓，呼应 Figma 编辑器的选择手柄
- 充满活力的多色英雄渐变（绿、黄、紫、粉）
- 全局启用 OpenType `"kern"` 字距调整特性
- 全局使用负字母间距——即使正文也达 -0.14px 至 -0.26px

## 2. 色彩体系与角色

### 主色
- **纯黑** (`#000000`)：所有文字、所有实心按钮、所有边框。界面的唯一"颜色"。
- **纯白** (`#ffffff`)：所有背景、白色按钮、深色表面上的文字。二元体系的另一半。

*注：Figma 营销网站的界面层仅使用这两种颜色。所有鲜艳颜色仅出现在产品截图、英雄渐变和嵌入内容中。*

### 表面与背景
- **纯白** (`#ffffff`)：主页面背景和卡片表面。
- **磨砂黑** (`rgba(0, 0, 0, 0.08)`)：次要圆形按钮和玻璃效果的微妙深色叠层。
- **磨砂白** (`rgba(255, 255, 255, 0.16)`)：深色/彩色表面上按钮的磨砂玻璃叠层。

### 渐变系统
- **英雄渐变**：使用电光绿、明亮黄、深紫和热粉红的多色节点渐变。这一渐变是英雄区的视觉签名——代表了该工具的创意可能性。
- **产品区渐变**：各产品区域（设计、开发模式、原型制作）在其展示中可能使用不同的色彩主题。

## 3. 排版规则

### 字体家族
- **主字体**：`figmaSans`，备选字体：`figmaSans Fallback, SF Pro Display, system-ui, helvetica`
- **等宽字体 / 标签**：`figmaMono`，备选字体：`figmaMono Fallback, SF Mono, menlo`

### 层次体系

| 角色 | 字体 | 尺寸 | 字重 | 行高 | 字母间距 | 备注 |
|------|------|------|--------|-------------|----------------|-------|
| 展示 / 英雄 | figmaSans | 86px (5.38rem) | 400 | 1.00（紧凑） | -1.72px | 最大冲击力，极度紧缩 |
| 区块标题 | figmaSans | 64px (4rem) | 400 | 1.10（紧凑） | -0.96px | 功能区块标题 |
| 副标题 | figmaSans | 26px (1.63rem) | 540 | 1.35 | -0.26px | 强调的区块文字 |
| 副标题（细体） | figmaSans | 26px (1.63rem) | 340 | 1.35 | -0.26px | 轻字重区块文字 |
| 功能标题 | figmaSans | 24px (1.5rem) | 700 | 1.45 | normal | 粗体卡片标题 |
| 大正文 | figmaSans | 20px (1.25rem) | 330–450 | 1.30–1.40 | -0.1px 至 -0.14px | 描述、介绍 |
| 正文 / 按钮 | figmaSans | 16px (1rem) | 330–400 | 1.40–1.45 | -0.14px 至 normal | 标准正文、导航、按钮 |
| 细正文 | figmaSans | 18px (1.13rem) | 320 | 1.45 | -0.26px 至 normal | 轻字重正文 |
| 等宽标签 | figmaMono | 18px (1.13rem) | 400 | 1.30（紧凑） | 0.54px | 大写区块标签 |
| 等宽小字 | figmaMono | 12px (0.75rem) | 400 | 1.00（紧凑） | 0.6px | 大写微型标签 |

### 排版原则
- **可变字体精度**：figmaSans 使用大多数系统从未涉及的字重——320、330、340、450、480、540。通过细微的字重差异建立层次，而非大幅跳跃。330 和 340 之间的差异几乎难以察觉，但在结构上意义重大。
- **轻体为基础**：大多数正文使用 320–340（比典型的 400"常规"更轻），营造出一种空灵、通透的阅读体验，契合设计工具的美学。
- **全局字距调整**：每个文字元素都启用 OpenType `"kern"` 特性——字距调整不是可选项，而是结构性要求。
- **默认负字母间距**：即使正文也使用 -0.1px 至 -0.26px 的字母间距，营造出普遍紧凑的文字效果。展示文字进一步压缩至 -0.96px 和 -1.72px。
- **等宽字体用于结构**：figmaMono 大写字母搭配正字母间距（0.54px–0.6px）创造出技术标识标签。

## 4. 组件样式

### 按钮

**黑色实心（胶囊形）**
- 背景：纯黑 (`#000000`)
- 文字：纯白 (`#ffffff`)
- 圆角：圆形（50%），用于图标按钮
- 焦点：虚线 2px 轮廓
- 最高强调度

**白色胶囊形**
- 背景：纯白 (`#ffffff`)
- 文字：纯黑 (`#000000`)
- 内边距：8px 18px 10px（垂直方向非对称）
- 圆角：胶囊形（50px）
- 焦点：虚线 2px 轮廓
- 深色/彩色表面上的标准行动号召

**磨砂深色**
- 背景：`rgba(0, 0, 0, 0.08)`（微妙深色叠层）
- 文字：纯黑
- 圆角：圆形（50%）
- 焦点：虚线 2px 轮廓
- 浅色表面上的次要操作

**磨砂浅色**
- 背景：`rgba(255, 255, 255, 0.16)`（磨砂玻璃）
- 文字：纯白
- 圆角：圆形（50%）
- 焦点：虚线 2px 轮廓
- 深色/彩色表面上的次要操作

### 卡片与容器
- 背景：纯白
- 边框：无或极少
- 圆角：6px（小容器）、8px（图片、卡片、对话框）
- 阴影：从微妙到中等的层次效果
- 产品截图作为卡片内容

### 导航
- 白色背景上的简洁横向导航
- Logo：黑色 Figma 字标
- 产品标签：胶囊形（50px）标签导航
- 链接：黑色文字，1px 下划线装饰
- 行动号召：黑色胶囊形按钮
- 悬停：通过 CSS 变量改变文字颜色

### 特色组件

**产品标签栏**
- 横向胶囊形标签（50px 圆角）
- 每个标签代表一个 Figma 产品区域（设计、开发模式、原型制作等）
- 激活标签高亮显示

**英雄渐变区块**
- 全宽充满活力的多色渐变背景
- 白色文字叠加，86px 展示标题
- 产品截图浮动于渐变之中

**虚线焦点指示器**
- 所有交互元素在焦点时使用 `dashed 2px` 轮廓
- 引用 Figma 编辑器中的选择手柄
- 一种将网站与产品连接的元设计选择

## 5. 布局原则

### 间距系统
- 基础单位：8px
- 刻度：1px、2px、4px、4.5px、8px、10px、12px、16px、18px、24px、32px、40px、46px、48px、50px

### 网格与容器
- 最大容器宽度：最高 1920px
- 英雄区：全宽渐变，内容居中
- 产品区块：交替展示
- 页脚：深色全宽区块
- 响应式范围：559px 至 1920px

### 留白理念
- **画廊般的节奏感**：充裕的间距让每个产品区块都能作为独立展品呼吸。
- **色彩区块作为视觉喘息**：渐变英雄区和产品展示区在单色界面区块之间提供色彩缓冲。

### 圆角刻度
- 极小（2px）：小型链接元素
- 微妙（6px）：小容器、分隔线
- 舒适（8px）：卡片、图片、对话框
- 胶囊形（50px）：标签按钮、行动号召
- 圆形（50%）：图标按钮、圆形元素

## 6. 深度与层次

| 层级 | 处理方式 | 用途 |
|-------|-----------|-----|
| 平面（0 级） | 无阴影 | 页面背景、大多数文字 |
| 表面（1 级） | 渐变/深色区块上的白色卡片 | 卡片、产品展示 |
| 悬浮（2 级） | 微妙阴影 | 浮动卡片、悬停状态 |

**阴影理念**：Figma 谨慎使用阴影。主要的深度机制是**背景对比**（彩色/深色区块上的白色内容）以及产品截图本身固有的立体感。

## 7. 应做与禁止

### 应做
- 使用 figmaSans 并精确指定可变字重（320–540）——精细的字重控制就是设计本身
- 保持界面严格黑白——颜色仅来自产品内容
- 对所有交互元素使用胶囊形（50px）和圆形（50%）几何
- 应用虚线 2px 焦点轮廓——标志性的无障碍模式
- 在所有文字上启用 `"kern"` 特性
- figmaMono 大写搭配正字母间距用于标签
- 全局应用负字母间距（-0.1px 至 -1.72px）

### 禁止
- 不要添加界面颜色——单色调色板是绝对的
- 不要使用标准字重（400、500、600、700）——使用可变字体的独特节点（320、330、340、450、480、540）
- 不要在按钮上使用尖锐角——只用胶囊形和圆形几何
- 不要使用实线焦点轮廓——虚线是标志性选择
- 不要将正文字重增加到 450 以上——轻字重美学是核心
- 不要在正文上使用正字母间距——始终为负值

## 8. 响应式行为

### 断点
| 名称 | 宽度 | 关键变化 |
|------|-------|-------------|
| 小型移动端 | <560px | 紧凑布局，堆叠 |
| 平板 | 560–768px | 微调 |
| 小桌面 | 768–960px | 2 列布局 |
| 桌面 | 960–1280px | 标准布局 |
| 大桌面 | 1280–1440px | 扩展 |
| 超宽屏 | 1440–1920px | 最大宽度 |

### 折叠策略
- 英雄文字：86px → 64px → 48px
- 产品标签：移动端横向滚动
- 功能区块：堆叠单列
- 页脚：多列 → 堆叠

## 9. 智能助手提示指南

### 快速颜色参考
- 全部：「纯黑 (#000000)」和「纯白 (#ffffff)」
- 磨砂深色：「rgba(0, 0, 0, 0.08)」
- 磨砂浅色：「rgba(255, 255, 255, 0.16)」

### 组件提示示例
- 「在充满活力的多色渐变（绿、黄、紫、粉）背景上创建英雄区。标题使用 86px figmaSans 字重 400，行高 1.0，字母间距 -1.72px，白色文字。白色胶囊形行动号召按钮（50px 圆角，8px 18px 内边距）。」
- 「设计一个胶囊形按钮的产品标签栏（50px 圆角）。激活状态：黑色背景，白色文字。非激活：透明，黑色文字。figmaSans 20px 字重 480。」
- 「创建区块标签：figmaMono 18px，大写，字母间距 0.54px，黑色文字。启用字距调整。」
- 「创建 20px figmaSans 字重 330 的正文，行高 1.40，字母间距 -0.14px。白色背景上的纯黑色文字。」

### 迭代指南
1. 精确使用可变字重节点：320、330、340、450、480、540、700
2. 界面始终为黑白——绝不在框架中添加颜色
3. 虚线焦点轮廓，非实线
4. 正文字母间距始终为负，等宽标签字母间距始终为正
5. 按钮/标签使用胶囊形（50px），图标按钮使用圆形（50%）
