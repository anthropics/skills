# 五次长波详解

> 从水力纺织到信息互联网——240 年间五次技术革命驱动的经济长波

## 📚 长波的四季模型

每一轮康波都可以类比为"四季"：

| 阶段 | 别名 | 时长 | 经济特征 | 资产表现 |
|------|------|------|----------|----------|
| **春（回升期）** | 播种期 | ~15 年 | 新技术萌芽，经济温和复苏 | 股票↑ 商品→ |
| **夏（繁荣期）** | 收获期 | ~10 年 | 技术广泛应用，经济高速增长 | 股票↑↑ 商品↑ 房产↑ |
| **秋（衰退期）** | 收割期 | ~10 年 | 通胀高企，增长放缓 | 商品↑ 黄金↑ 股票↓ |
| **冬（萧条期）** | 蛰伏期 | ~15 年 | 通缩风险，经济低迷 | 现金↑ 债券↑ 黄金↑ |

---

## 🌊 第一波：纺织与水力革命（1780s-1845）

| 维度 | 内容 |
|------|------|
| **时间** | 上升期 ~1782-1815，下降期 1815-~1845 |
| **核心技术** | 飞梭、珍妮纺纱机、水力纺纱机、蒸汽机早期应用 |
| **主导产业** | 棉纺织业、铁器制造 |
| **主导国家** | 英国 |
| **上升期标志** | 英国工业革命启动，棉纺产量激增 |
| **转折事件** | 拿破仑战争结束（1815），战时需求消失 |
| **下降期特征** | 农业萧条、卢德运动（破坏机器）、社会动荡 |

---

## 🌊 第二波：蒸汽与铁路革命（1845-1892）

| 维度 | 内容 |
|------|------|
| **时间** | 上升期 ~1845-1873，下降期 1873-~1892 |
| **核心技术** | 蒸汽机改良、铁路建设、贝塞麦炼钢法 |
| **主导产业** | 铁路、钢铁、煤炭 |
| **主导国家** | 英国、德国、美国 |
| **上升期标志** | 铁路建设狂潮（"铁路狂热"），全球贸易网络形成 |
| **转折事件** | 1873 年大萧条（Long Depression）开始 |
| **下降期特征** | 持续通缩、企业倒闭潮、保护主义抬头 |

---

## 🌊 第三波：电气与化工革命（1892-1948）

| 维度 | 内容 |
|------|------|
| **时间** | 上升期 ~1892-1920，下降期 1920-~1948 |
| **核心技术** | 电力、内燃机、化学工业、电话 |
| **主导产业** | 电力、汽车、化工、石油 |
| **主导国家** | 美国、德国 |
| **上升期标志** | 爱迪生电力系统、福特 T 型车流水线（1908）、第一次世界大战 |
| **转折事件** | 1929 年华尔街崩盘 → 大萧条 |
| **下降期特征** | 1929-1933 大萧条、第二次世界大战 |

> 📌 第三波的下降期极为惨烈，因为多个周期同时下行共振（康波+朱格拉+基钦三重下行）

---

## 🌊 第四波：石化与航空革命（1948-1991）

| 维度 | 内容 |
|------|------|
| **时间** | 上升期 ~1948-1973，下降期 1973-~1991 |
| **核心技术** | 石油化工、航空航天、核能、半导体早期 |
| **主导产业** | 石油化工、航空、汽车（日本）、电子 |
| **主导国家** | 美国、日本、德国 |
| **上升期标志** | 战后重建、马歇尔计划、"黄金三十年"（1945-1975） |
| **转折事件** | 1973 年石油危机、布雷顿森林体系崩溃 |
| **下降期特征** | 滞胀（高通胀+高失业）、1979 第二次石油危机、日本泡沫 |

---

## 🌊 第五波：信息与互联网革命（1991-?）

| 维度 | 内容 |
|------|------|
| **时间** | 上升期 ~1991-2008，下降期 ~2008-~2040 |
| **核心技术** | 个人电脑、互联网、移动通信、AI |
| **主导产业** | 软件、互联网、电商、云计算、AI |
| **主导国家** | 美国、中国 |
| **上升期标志** | 互联网普及、全球化加速、中国崛起 |
| **转折事件** | 2008 年全球金融危机 |
| **下降期特征** | 低利率/零利率时代、量化宽松、去全球化、AI 颠覆 |

### 第六波的猜想

部分研究者认为第六波长波可能始于 2030s-2040s，潜在驱动技术包括：
- **人工智能与自动化**
- **量子计算**
- **可控核聚变**
- **生物技术/基因工程**
- **太空经济**
