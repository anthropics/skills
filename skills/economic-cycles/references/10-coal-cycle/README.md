# 煤炭与经济周期

> 产能周期与能源转型——中国煤炭周期的独特特征

## 📚 本章概述

煤炭是全球最古老的化石能源，也是中国能源结构的基石（占一次能源消费的 55% 以上）。煤炭价格的周期性波动，深刻反映了**工业化进程**、**产能投资周期**和**政策干预**的三重交织。

与石油不同，煤炭市场的核心特征是**供给驱动**——价格波动更多源于产能扩张/收缩的时滞效应，而非突发的需求冲击或地缘政治事件。

### 煤炭二十年复盘（2003-2025）

| 阶段 | 时间 | 煤价走势 | 核心驱动 |
|------|------|----------|----------|
| **黄金十年** | 2003-2011 | 264→785 元/吨 | 入WTO后经济高增长、固定资产投资强劲 |
| **产能过剩** | 2012-2015 | 785→350 元/吨 | 产能集中释放、需求增速放缓 |
| **供给侧改革** | 2016-2020 | 350→600 元/吨 | 淘汰落后产能、优化供给结构 |
| **能源危机** | 2021-至今 | 600→2000→700 元/吨 | 全球能源危机、保供稳价政策 |

### 中国煤炭周期的三个核心特征

1. **产能周期受政策强约束**：从供给侧改革到"双碳"政策，产能扩张被政策多方抑制
2. **从强周期向弱周期转变**：行业属性正从典型周期品向公用事业属性过渡
3. **能源安全的"压舱石"**：尽管碳中和是长期方向，但短期内煤炭仍是能源安全的保障
