# 中国煤炭周期特征

> 从"黄金十年"到"保供稳价"——中国煤炭产业的四次大转折

## 📚 中国煤炭消费的四个阶段

中国是全球最大的煤炭生产国和消费国，煤炭在中国经济中的角色经历了深刻变化：

### 第一阶段：下降期（1997-2000）

| 维度 | 详情 |
|------|------|
| **宏观背景** | 1997年亚洲金融危机，国内需求萎缩 |
| **煤炭消费** | 连续四年下降 |
| **煤价** | 低迷（政府定价时代，尚未完全市场化） |
| **行业状况** | 大量小煤矿关闭，乡镇煤矿整顿 |

### 第二阶段：黄金十年（2001-2013）

| 维度 | 详情 |
|------|------|
| **宏观背景** | 入WTO后经济高增长，重工业化、城镇化加速 |
| **煤炭消费** | 年均增速超过 10%，从 13 亿吨增至 42 亿吨 |
| **煤价** | 2002年实现市场化，价格从 264 元/吨升至 785 元/吨 |
| **行业状况** | 产量暴增，煤企利润丰厚，大规模投资建矿 |

**关键驱动力**：
- 🏗️ 固定资产投资年均增速 25%+
- 🏭 电力（火电占比 75%）、钢铁、水泥行业高速扩张
- 🏘️ 房地产开发热潮

### 第三阶段：产能过剩与供给侧改革（2014-2020）

| 维度 | 详情 |
|------|------|
| **宏观背景** | 经济增速换挡（"新常态"），"三期叠加" |
| **煤炭消费** | 2014-2016年连续下降，后缓慢回升 |
| **煤价** | 先跌至 350 元/吨，供改后回升至 600 元/吨 |
| **行业状况** | 淘汰落后产能 10 亿吨+，煤矿数量减半 |

**产能过剩的成因**：
1. 黄金十年期间过度投资——实际产能达到 57 亿吨，远超 39 亿吨的消费量
2. "四万亿"刺激计划（2009年）进一步催化了产能扩张
3. 需求增速从两位数降至个位数，但产能释放仍在继续

### 第四阶段：能源安全新时代（2021-至今）

| 维度 | 详情 |
|------|------|
| **宏观背景** | 疫后复苏+全球能源危机+碳中和政策 |
| **煤炭消费** | 维持高位，2023年超 45 亿吨 |
| **煤价** | 2021年一度突破 2000 元/吨，后限价于 675-1155 元/吨区间 |
| **行业状况** | "保供稳价"成为关键词，行业向公用事业属性转型 |

**2021年煤荒的教训**：
- 供给侧改革使供给弹性极低
- 极端天气+经济复苏导致需求超预期
- 进口端受限（中澳贸易摩擦、印尼出口禁令）
- 结果：煤价暴涨 3 倍以上，多地出现"电荒"

## 🏛️ 政策干预：从市场化到再管制

中国煤炭行业的一个显著特征是政策干预的深度和广度不断增加：

### 定价机制的演变

| 时期 | 定价方式 | 特点 |
|------|----------|------|
| 2002年前 | 政府定价 | 计划经济遗留 |
| 2002-2012 | 双轨制（市场+计划电煤） | 逐步市场化 |
| 2013-2020 | 完全市场化 | 供改后价格中枢上移 |
| 2021-至今 | 市场化+行政限价 | "长协+现货"双轨，长协占比 80%+ |

### 当前政策框架

- **中长期合同**：电煤长协基准价 675 元/吨（5500 大卡），签约率要求"三个 100%"
- **现货限价**：港口现货价格上限 1155 元/吨
- **保供任务**：主要煤企承担月度保供任务量
- **进口调节**：根据国内供需动态调整进口煤关税

> 💡 **煤炭的"公用事业化"**：这套政策体系使煤炭行业的价格波动大幅减小，利润趋于稳定，周期属性明显弱化。煤炭更像是中国的能源基础设施，而非传统意义上的周期品。

## 🌱 能源转型与煤炭的"终局"

### "双碳"目标下的煤炭定位

- **碳达峰（2030年前）**：煤炭消费可能在 2025-2028 年达峰
- **碳中和（2060年前）**：煤炭在一次能源中的占比需从 55% 降至 5% 以下

### 转型路径

```
当前（2025）: 煤炭占一次能源 ~55%
    ↓ 逐步替代
2030: 煤炭占比降至 ~45%，可再生能源占比 ~30%
    ↓ 加速替代
2040: 煤炭占比降至 ~25%，可再生能源占比 ~50%
    ↓ 深度转型
2060: 煤炭占比 < 5%，可再生能源占比 ~80%
```

### 转型期的矛盾

| 矛盾 | 表现 |
|------|------|
| **安全 vs 转型** | 过快退煤→能源安全风险（2021年教训） |
| **投资 vs 预期** | 远期需求下降预期→投资意愿不足→短期供给紧张 |
| **就业 vs 效率** | 煤炭行业直接从业人员 300 万+，社会稳定压力 |
| **区域 vs 全局** | 煤炭主产区经济高度依赖煤炭 |

> ⚠️ **"峰值煤炭"的投资悖论**：与"峰值石油需求"类似，如果市场预期煤炭需求将在近期达峰，企业将减少投资新产能 → 短期内因产能不足反而可能导致价格上涨。

## 📊 煤炭价格的季节性规律

中国煤炭市场存在明显的**季节性需求特征**：

| 月份 | 需求特征 | 煤价表现 |
|------|----------|----------|
| 1-2月 | 冬季供暖旺季 + 春节备货 | 偏强 |
| 3-4月 | 供暖结束，需求回落 | 偏弱（"淡季不淡"时除外） |
| 5-6月 | 迎峰度夏前备货 | 逐步回暖 |
| 7-8月 | 夏季用电高峰 | 全年最强 |
| 9-10月 | 酷暑结束，需求回落 | 短暂回调 |
| 11-12月 | 冬储备煤 | 再度走强 |
