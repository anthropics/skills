# 石油与经济周期

> 大宗商品之王——石油超级周期与全球经济的深度耦合

## 📚 本章概述

石油是现代工业文明的血液，其价格波动与全球经济周期深度耦合。作为最重要的大宗商品，原油价格不仅是经济运行的**温度计**，更是能够**反向影响**经济周期运行的关键变量——历史上三次石油危机均直接触发了全球经济衰退。

### 油价的双重周期性

石油价格受到两个层面的周期驱动：

| 周期层面 | 对应经济周期 | 时间跨度 | 核心驱动 |
|----------|-------------|----------|----------|
| **长周期** | 康波长波/超级周期 | 20-40 年 | 劳动生产率、发展中国家工业化、产能周期 |
| **中周期** | 朱格拉周期 | 7-11 年 | 固定资本投资、经济增长率、OPEC政策 |

### 原油价格历史概览（1973年至今）

| 时期 | 价格区间 | 核心事件 |
|------|----------|----------|
| **1973年前** | 1.5-3 美元/桶 | 西方石油公司垄断定价，低油价时代 |
| **1973-1986** | 3→41→10 美元/桶 | 两次石油危机 + OPEC市场份额争夺战 |
| **1986-2003** | 10-40 美元/桶 | 市场定价时代，低位震荡 |
| **2003-2008** | 28→147 美元/桶 | 中国工业化推动超级周期 |
| **2008-2020** | 147→26→-37 美元/桶 | 金融危机 + 页岩油革命 + 疫情 |
| **2020-至今** | -37→120→80 美元/桶 | 疫后复苏 + 俄乌冲突 + 能源转型 |

### 四次大宗商品超级周期

根据加拿大央行（Bank of Canada）的研究，自 1900 年以来经历了四次大宗商品价格超级周期，石油是其中波动最剧烈的品种：

| 超级周期 | 时间 | 驱动力 |
|----------|------|--------|
| 第一次 | 1890s-1930s | 美国工业化 |
| 第二次 | 1930s-1950s | 二战后重建 |
| 第三次 | 1960s-1980s | 欧日重建 + OPEC崛起 + 石油危机 |
| 第四次 | 2000s-2010s | 中国工业化与城镇化 |
