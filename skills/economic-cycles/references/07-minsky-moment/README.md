# 明斯基金融不稳定假说

> 金融体系的内生脆弱性——"稳定中内含着不稳定"

## 📚 本章概述

**海曼·明斯基**（Hyman Minsky, 1919-1996），美国经济学家，后凯恩斯主义代表人物。他提出的**金融不稳定假说**（Financial Instability Hypothesis, FIH）认为，资本主义金融体系具有**内在的不稳定性**，倾向于在经济繁荣期积累脆弱性，最终导致金融危机。

**核心命题**："稳定中内含着不稳定"——长时期的经济稳定会导致债务增加、杠杆比率上升，从内部滋生爆发金融危机的风险。

### 三类融资结构

明斯基将融资分为三种类型，反映了金融体系从健康到脆弱的演化：

| 融资类型 | 现金流特征 | 风险水平 | 典型案例 |
|----------|-----------|----------|----------|
| **对冲融资** | 现金流可覆盖**本息** | 低 | 传统银行贷款、优质企业 |
| **投机融资** | 现金流仅能覆盖**利息** | 中 | 短债长投、依赖再融资 |
| **庞氏融资** | 现金流**既不能还本也不能付息** | 极高 | 借新还旧、依赖资产增值 |
