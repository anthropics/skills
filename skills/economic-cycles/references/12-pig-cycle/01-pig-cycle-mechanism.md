# 猪周期机制详解

> 蛛网模型的活教材——从母猪补栏到肥猪出栏的生物钟

## 📚 核心理论

### 蛛网模型与猪周期

猪周期是经济学中**蛛网模型（Cobweb Model）**的最经典现实案例。蛛网模型描述了在供给调整存在时滞的市场中，价格如何围绕均衡点产生周期性波动。

**猪肉市场满足蛛网模型的所有条件**：

| 条件 | 猪肉市场的表现 |
|------|---------------|
| 需求相对稳定 | ✅ 猪肉是刚需食品，年消费量约 5,500 万吨，需求弹性低 |
| 供给调整有时滞 | ✅ 从补栏母猪到肥猪出栏需要约 16 个月 |
| 生产者基于当期价格决策 | ✅ 养殖户普遍根据当前猪价决定补栏/淘汰 |
| 无法储存 | ✅ 活猪不能长期储存（不同于粮食、金属） |

### 生物性时滞：猪周期的"生物钟"

```
母猪补栏 → 配种妊娠(4个月) → 仔猪出生 → 保育(2个月) → 育肥(4个月) → 出栏
        |←————————— 约 10 个月 ——————————→|
```

**关键时间节点**：
- **能繁母猪存栏量**变化 → 领先**生猪出栏量** 约 10 个月
- **生猪出栏量**变化 → **同步/微领先**猪价拐点

> 💡 **核心观察指标**：能繁母猪存栏量是判断猪周期拐点最可靠的领先指标。母猪产能去化深度和斜率，可以提前 10 个月预判猪价的拐点。

## 🔄 猪周期的四个阶段

### 阶段一：猪价上行期（利润修复→产能扩张）

| 特征 | 表现 |
|------|------|
| **猪价** | 持续上涨 |
| **养殖利润** | 丰厚（头均利润可达数百元） |
| **母猪存栏** | 养殖户补栏积极，母猪存栏上升 |
| **行为逻辑** | "猪价这么高，赶紧补栏/扩产" |

### 阶段二：猪价见顶→下行初期（产能释放）

| 特征 | 表现 |
|------|------|
| **猪价** | 见顶回落 |
| **养殖利润** | 开始收窄 |
| **母猪存栏** | 前期补栏的母猪产仔，生猪出栏量增加 |
| **行为逻辑** | "价格还行，再等等看" |

### 阶段三：猪价加速下行期（亏损→去产能）

| 特征 | 表现 |
|------|------|
| **猪价** | 加速下跌 |
| **养殖利润** | 深度亏损（头均亏损数百元） |
| **母猪存栏** | 养殖户被迫淘汰母猪，产能去化 |
| **行为逻辑** | "养一头亏一头，赶紧卖母猪" |

### 阶段四：猪价触底→回升（供给收缩→价格修复）

| 特征 | 表现 |
|------|------|
| **猪价** | 触底回升 |
| **养殖利润** | 从亏损转为盈亏平衡 |
| **母猪存栏** | 母猪去化接近尾声，存栏量见底 |
| **行为逻辑** | "母猪都卖了，猪不够了，价格要涨了" |

## 📊 两个关键分析要素

### 要素一：产能（决定供给方向）

**核心指标**：能繁母猪存栏量

| 母猪变化 | 含义 | 对猪价影响 |
|----------|------|-----------|
| 存栏持续下降 | 未来供给减少 | 利好（10个月后） |
| 存栏触底回升 | 供给拐点 | 猪价可能见顶（10个月后） |
| 存栏持续上升 | 未来供给增加 | 利空（10个月后） |

### 要素二：效率（决定供给弹性）

**核心指标**：MSY（每头母猪每年提供的断奶仔猪数）

| MSY 变化 | 含义 |
|----------|------|
| MSY 提高 | 相同母猪存栏下，出栏更多生猪→供给弹性增大 |
| MSY 降低 | 疫病或管理问题→供给效率下降 |

> 中国 MSY 从 2018 年的 ~18 头提升至 2025 年的 ~24 头，意味着同等母猪存栏下的生猪供给能力大幅提升。这是导致近年猪周期变异的重要原因之一。

## 🆚 猪周期与基钦周期的类比

猪周期与基钦库存周期存在高度的结构性相似：

| 维度 | 猪周期 | 基钦库存周期 |
|------|--------|-------------|
| 周期长度 | 3-4 年 | 3-5 年 |
| 核心机制 | 供给调整时滞 | 库存调整时滞 |
| 模型 | 蛛网模型 | 加速器模型 |
| 四阶段 | 上涨→见顶→下跌→触底 | 补库→被动补库→去库→被动去库 |
| 领先指标 | 能繁母猪存栏 | 新订单/PMI |
| 同步指标 | 生猪出栏量 | 产成品库存 |

**本质差异**：
- 基钦周期的库存可以快速调节（生产计划变更即可）
- 猪周期的"库存"（活猪）受生物规律约束，调节速度极慢

---

## ⚠️ 猪周期的"外生冲击"

外生冲击可以显著改变猪周期的运行节奏：

| 冲击类型 | 案例 | 对周期的影响 |
|----------|------|-------------|
| **重大疫病** | 非洲猪瘟（2018-2019） | 产能突然大幅去化→猪价暴涨→周期被重置 |
| **环保政策** | 禁养区清退（2015-2017） | 产能被动去化→母猪与猪价脱钩→非典型周期 |
| **政府收储/放储** | 中央储备冻肉收放 | 短期平抑波动，不改变周期方向 |
| **食品安全事件** | 瘦肉精（2011） | 短期冲击需求，不改变供给周期 |
| **产业政策** | 规模化补贴 | 长期改变供给弹性 |
