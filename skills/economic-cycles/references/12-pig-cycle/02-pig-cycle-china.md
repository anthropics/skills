# 中国猪周期历史回顾

> 从散养到规模化——五轮猪周期的演化与变异

## 📚 五轮猪周期详解

### 第一轮猪周期（2002-2006）

| 维度 | 详情 |
|------|------|
| **上涨期** | 2002.5-2004.9（27个月），涨幅 +71% |
| **下跌期** | 2004.9-2006.5（22个月），跌幅 -40% |
| **最高猪价** | 10.0 元/公斤 |
| **关键事件** | 2003年非典→宰杀母猪、补栏停滞；2004年禽流感→猪肉替代需求大增 |
| **特征** | 典型蛛网模型，供需错配导致价格波动 |

### 第二轮猪周期（2006-2009）

| 维度 | 详情 |
|------|------|
| **上涨期** | 2006.6-2008.3（20个月），涨幅 +158% |
| **下跌期** | 2008.3-2009.5（14个月），跌幅 -49% |
| **最高猪价** | 17.4 元/公斤 |
| **关键事件** | 2006年高致病性蓝耳病→大量母猪死亡→供给暴减 |
| **特征** | 疫病驱动的供给冲击，上涨幅度最大 |

### 第三轮猪周期（2010-2014）

| 维度 | 详情 |
|------|------|
| **上涨期** | 2010.4-2011.9（19个月），涨幅 +80% |
| **下跌期** | 2011.9-2014.4（32个月），跌幅 -47% |
| **最高猪价** | 19.6 元/公斤 |
| **关键事件** | 2011年瘦肉精事件冲击消费信心 |
| **特征** | 下跌期异常漫长，产能出清缓慢 |

### 第四轮猪周期（2014-2018）

| 维度 | 详情 |
|------|------|
| **上涨期** | 2014.4-2016.5（24个月），涨幅 +130% |
| **下跌期** | 2016.5-2018.5（25个月），跌幅 -55% |
| **最高猪价** | 21.2 元/公斤 |
| **关键事件** | 2015-2017年环保政策加严→禁养区产能强制清退 |
| **特征** | **非典型猪周期**——母猪存栏在猪价上涨时仍持续下行（环保去产能主导） |

> ⚠️ 第四轮是第一次出现"母猪存栏与猪价脱钩"的现象。环保政策取代了市场力量，成为产能去化的主要驱动力。

### 第五轮猪周期（2018-至今）

这是中国猪周期史上最剧烈、最复杂的一轮：

**阶段一：非洲猪瘟冲击（2018.8-2019.10）**

| 维度 | 详情 |
|------|------|
| **事件** | 2018年8月中国确认首例非洲猪瘟 |
| **产能冲击** | 能繁母猪存栏从 3,200 万头降至 ~2,000 万头（-40%） |
| **猪价** | 从 10 元/kg 飙升至 40+ 元/kg（+300%） |
| **生猪出栏** | 2019年出栏量下降 20% 以上 |

**阶段二：疯狂扩产（2019.10-2021.6）**

| 维度 | 详情 |
|------|------|
| **行业行为** | 猪价暴利吸引大量资本涌入，头部猪企（牧原、温氏、新希望等）疯狂扩产 |
| **产能** | 母猪存栏快速回升，2021年中恢复至 4,500+ 万头 |
| **养殖效率** | MSY 从 18 头提升至 22+ 头 |
| **结果** | 严重产能过剩 |

**阶段三：漫长磨底（2021.7-至今）**

| 维度 | 详情 |
|------|------|
| **猪价** | 从 30+ 元/kg 跌至 10 元/kg 以下 |
| **行业状况** | 深度亏损，头均亏损数百元 |
| **去产能** | 母猪持续去化，但MSY提升对冲了部分效果 |
| **2026年现状** | 猪价创 2019 年以来新低，行业"反内卷"政策推进 |

## 🔄 猪周期的结构性变化

### 从散养到规模化

| 指标 | 2010年 | 2020年 | 2025年 |
|------|--------|--------|--------|
| 年出栏 500 头以上规模化率 | ~35% | ~57% | ~70% |
| 前 10 大猪企市占率 | ~5% | ~15% | ~25% |
| MSY | ~16 头 | ~20 头 | ~24 头 |

### 规模化对猪周期的影响

| 影响维度 | 方向 | 逻辑 |
|----------|------|------|
| 周期振幅 | 可能减小 | 规模企业决策更理性，反应更快 |
| 周期长度 | 可能拉长 | 规模企业资金实力强，扛亏损能力强 |
| 价格底部 | 可能抬升 | 成本线趋于一致，行业集中度提升 |
| 外生冲击敏感度 | 降低 | 生物安全防控能力提升 |

### MSY 提升的"周期钝化"效应

MSY（母猪年提供断奶仔猪数）的持续提升，使得：
- 更少的母猪可以产出更多的生猪
- 母猪存栏量下降不再等同于产能下降
- 传统的"母猪→猪价"传导链条被打破
- 需要同时观察母猪存栏 × MSY 来判断真实产能

## 📊 猪周期与 CPI

猪肉价格在中国 CPI 中的权重约为 **2-3%**，但由于猪价波动剧烈，实际影响远大于权重本身：

| 猪价变动 | 对 CPI 的拉动 |
|----------|--------------|
| 猪价 +50% | CPI +1.0-1.5 个百分点 |
| 猪价 +100% | CPI +2.0-3.0 个百分点 |
| 猪价 -30% | CPI -0.6-0.9 个百分点 |

**政策影响**：
- 猪价大涨时→央行货币政策可能收紧（担忧通胀）
- 猪价大跌时→CPI 被拉低→为降息提供空间
- 因此，猪周期间接影响中国的利率周期和流动性环境

## 🔮 当前位置与展望（2026年）

### 周期位置判断

| 信号 | 现状 |
|------|------|
| 猪价水平 | 10 元/kg 以下，创 5 年新低 |
| 养殖利润 | 深度亏损，连续亏损 7 个月+ |
| 母猪去化 | 持续去化中，但速度偏慢 |
| MSY | 仍在提升（23.93头），对冲去化效果 |
| 政策 | "反内卷"政策推进，加速产能调控 |

> 💡 **判断**：当前大概率处于猪周期的**阶段三（深度亏损→去产能）**向**阶段四（触底回升）**的过渡区间。关键观察点是母猪去化深度是否达到 10-15%，以及 MSY 是否还在继续提升。
