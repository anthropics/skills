# 猪周期

> 中国最具特色的农产品周期——3-4 年的供给轮回

## 📚 本章概述

**猪周期**是中国经济中最典型、最具特色的农产品价格周期。由于猪肉在中国居民饮食中的核心地位（占肉类消费约 60%），猪价波动不仅影响 CPI 走势，还直接关系到货币政策取向和民生稳定。

猪周期是经济学中**蛛网模型（Cobweb Model）**的经典案例：猪肉需求相对稳定，价格波动主要由**供给侧**主导，而供给调整存在**生物性时滞**（约 10 个月），导致价格呈现 3-4 年的规律性波动。

### 猪周期的核心逻辑

```
猪价上涨 → 养殖利润丰厚 → 养殖户扩大产能（补栏母猪）
→ 10个月后仔猪出栏增加 → 6个月后肥猪供给增加
→ 供给过剩 → 猪价下跌 → 养殖亏损 → 淘汰母猪（去产能）
→ 10+6个月后供给减少 → 猪价上涨 → ...
```

### 中国五轮猪周期概览（2002-至今）

| 轮次 | 时间 | 上涨时长 | 涨幅 | 下跌时长 | 跌幅 | 关键外生冲击 |
|------|------|----------|------|----------|------|-------------|
| 第一轮 | 2002-2006 | 27个月 | +71% | 22个月 | -40% | 非典（2003）、禽流感（2004） |
| 第二轮 | 2006-2009 | 20个月 | +158% | 14个月 | -49% | 蓝耳病（2006）导致大量母猪死亡 |
| 第三轮 | 2010-2014 | 19个月 | +80% | 32个月 | -47% | 瘦肉精事件（2011） |
| 第四轮 | 2014-2018 | 24个月 | +130% | 25个月 | -55% | 环保限产、非洲猪瘟（2018） |
| 第五轮 | 2018-至今 | 变异中 | 变异中 | 变异中 | 变异中 | 非洲猪瘟、规模化加速 |
