# 铜铝周期详解

> "铜博士"与铝——工业金属的周期韵律

## 📚 铜："铜博士"的周期密码

### 为什么铜是经济周期的最佳指标？

铜被称为**"Doctor Copper"（铜博士）**，因为它比任何经济学家都更准确地诊断经济健康状况：

1. **无处不在的应用**：电力（48%）、建筑（16%）、家电（14%）、交通（10%）、工业机械（12%）
2. **需求弹性高**：经济扩张→几乎所有行业增加铜消费；经济收缩→需求快速下降
3. **金融化程度高**：LME（伦敦金属交易所）铜期货是全球交易最活跃的金属品种之一
4. **库存信号清晰**：全球铜显性库存变化是判断供需的直观指标

### 铜价的历史走势

| 时期 | LME铜价 | 核心驱动 |
|------|---------|----------|
| 1990s | 1,500-3,000 美元/吨 | 亚洲金融危机→低迷 |
| 2003-2006 | 1,800→8,800 美元/吨 | 中国工业化 |
| 2008 | 8,800→2,800 美元/吨 | 金融危机暴跌 |
| 2009-2011 | 2,800→10,190 美元/吨 | "四万亿"刺激+全球QE |
| 2011-2016 | 10,190→4,300 美元/吨 | 中国增速放缓 |
| 2020-2022 | 4,600→10,700 美元/吨 | 疫后复苏+新能源需求 |
| 2023-2025 | 7,800→9,500 美元/吨 | 新旧周期拉锯 |

## 🔄 铜的双周期驱动

### 基钦库存周期（3-5年）

铜的短期价格由**库存周期**驱动：

| 阶段 | 特征 | 铜价 |
|------|------|------|
| **被动去库** | 需求恢复，库存被动下降 | 开始上涨 |
| **主动补库** | 企业主动增加库存 | 加速上涨 |
| **被动补库** | 需求放缓，库存被动增加 | 见顶回落 |
| **主动去库** | 企业主动降低库存 | 加速下跌 |

**关键库存指标**：
- LME 铜库存（全球显性库存）
- SHFE（上期所）铜库存
- 保税区库存（反映贸易流向）

### 朱格拉投资周期（7-11年）

铜的中期价格由**固定资本投资周期**驱动：

- **扩张期**：企业增加设备投资、产能建设 → 铜需求增长 → 铜价上涨
- **收缩期**：企业削减投资 → 铜需求减少 → 铜价下跌

**历史上的朱格拉周期与铜价**：

| 朱格拉周期 | 时间 | 铜价表现 |
|-----------|------|----------|
| 第一轮上行 | 2001-2007 | 1,500→8,800 美元 |
| 第一轮下行 | 2008-2009 | 8,800→2,800 美元 |
| 第二轮上行 | 2009-2011 | 2,800→10,190 美元 |
| 第二轮下行 | 2012-2016 | 10,190→4,300 美元 |
| 第三轮上行 | 2020-2022 | 4,600→10,700 美元 |

## 🏭 铜的供给侧

### 全球铜矿供给格局

| 维度 | 数据 |
|------|------|
| 全球产量 | ~2,200 万吨/年 |
| 主要产国 | 智利（27%）、秘鲁（10%）、刚果（10%）、中国（8%） |
| 再生铜占比 | ~33% |
| 矿山产能利用率 | ~80-85% |

### TC/RC 指标（铜精矿加工费）

**TC（Treatment Charge，冶炼加工费）**是判断铜矿供需的重要指标：

| TC 水平 | 含义 |
|---------|------|
| TC > 80 美元/吨 | 矿石供给充裕，矿企议价权弱 |
| TC 40-80 美元/吨 | 供需大致平衡 |
| TC < 40 美元/吨 | 矿石供给偏紧，矿企议价权强 |
| TC < 10 美元/吨 | 极度紧缺（2024年一度出现） |

## 🔲 铝的周期特征

### 铝与铜的周期差异

| 维度 | 铜 | 铝 |
|------|------|------|
| 供给集中度 | 高（智利+秘鲁占37%） | 中（中国占57%） |
| 能源敏感度 | 低 | 极高（电解铝耗电量大） |
| 中国影响 | 需求主导 | 供需双主导 |
| 价格弹性 | 中等 | 较高（产能调节较快） |

### 铝的独特周期因素

1. **电价周期**：铝冶炼是高耗电行业（每吨铝耗电约 13,500 kWh），电价波动直接影响成本和产能开关
2. **中国产能"天花板"**：国家限制电解铝总产能不超过 4,500 万吨
3. **碳排放约束**：铝冶炼是碳排放大户，碳中和政策限制产能扩张

### 铝价走势

| 时期 | LME铝价 | 核心驱动 |
|------|---------|----------|
| 2004-2008 | 1,600→3,300 美元/吨 | 中国需求 |
| 2008-2009 | 3,300→1,300 美元/吨 | 金融危机 |
| 2010-2011 | 1,300→2,700 美元/吨 | 刺激政策 |
| 2015-2016 | 1,400 美元/吨 | 产能过剩 |
| 2021-2022 | 3,800 美元/吨 | 能源危机+限电 |

## 🔮 新旧超级周期之争

### 旧超级周期：中国基建/房地产（正在结束）

- 过去 20 年铜铝需求的核心驱动：中国房地产+基建投资
- 随着中国房地产长周期见顶，这一驱动力正在衰减
- 铜的传统需求（电线电缆、空调、建筑）面临增速放缓

### 新超级周期：能源转型（正在崛起）

能源转型正在创造巨大的新增金属需求：

| 应用 | 金属需求 | 增量估计 |
|------|----------|----------|
| **电动汽车** | 每辆 EV 用铜约 80kg（传统车仅 20kg） | 巨大 |
| **充电桩** | 铜为核心导体材料 | 中等 |
| **风电** | 单台 6MW 风机用铜约 30 吨 | 大 |
| **光伏** | 铜用于汇流条、逆变器、线缆 | 中等 |
| **电网升级** | 分布式能源需要更密集的电网 | 极大 |
| **储能** | 锂电池用铝箔集流体、铜箔 | 大 |

> 💡 **"铜被困在新旧超级周期之间"**（Reuters, 2023）：旧周期（中国房地产）正在结束，新周期（能源转型）正在崛起，铜市场处于两股力量的拉锯之中。

### 供给瓶颈

与需求的结构性增长相对的是供给端的困境：
- 全球铜矿品位持续下降
- 新发现大型矿床越来越少
- 从发现到投产平均需要 15-20 年
- ESG/社区阻力增加开发难度
- 铜精矿 TC 降至历史低位，反映矿石短缺

## 📊 铜金比：经济周期的风向标

**铜金比 = 铜价 / 金价**，是判断市场风险偏好和经济预期的重要指标：

| 铜金比走势 | 含义 |
|-----------|------|
| 上升 | 经济向好预期、风险偏好上升 |
| 下降 | 经济衰退预期、避险情绪升温 |

**历史极端值**：
- 2007年（高位）：全球经济繁荣，铜需求旺盛
- 2008年底（低位）：金融危机，铜暴跌而金价稳定
- 2020年3月（低位）：疫情冲击
- 2024-2025年（低位运行）：黄金超预期上涨压低铜金比
