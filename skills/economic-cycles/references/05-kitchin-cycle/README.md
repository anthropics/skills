# 基钦周期

> 3-5 年的库存周期——投资择时的实用工具

## 📚 本章概述

**约瑟夫·基钦**（Joseph Kitchin, 1861-1932），英国统计学家，1923 年根据英国和美国 1890-1922 年的利率、物价、生产、就业数据，发现了约 40 个月（3-4 年）内发生的有规则的经济波动，其中以**厂商存货的增减变化**最为显著。

基钦周期是四大经济周期中**最短、最易观测、最具投资实用性**的一个，其核心逻辑是企业库存调整的滞后效应。

### 四阶段速览

```
           需求↑              需求↑              需求↓              需求↓
           库存↓              库存↑              库存↑              库存↓
    ┌──────────────┬──────────────┬──────────────┬──────────────┐
    │  ① 被动去库存  │  ② 主动补库存  │  ③ 被动补库存  │  ④ 主动去库存  │
    │  (复苏)       │  (过热)       │  (滞胀)       │  (衰退)       │
    └──────────────┴──────────────┴──────────────┴──────────────┘
```
