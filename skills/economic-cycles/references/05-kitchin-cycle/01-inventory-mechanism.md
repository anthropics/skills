# 库存周期机制

> 需求与库存的"猫鼠游戏"——企业永远慢半拍的调整

## 📚 四阶段详解

库存周期的本质是**企业生产调整对市场需求变化的滞后反应**。由于企业无法即时调整产量，库存成为供需失衡的"缓冲器"。

### ① 被动去库存（复苏期）

| 维度 | 表现 |
|------|------|
| **需求** | ↑ 开始回升 |
| **库存** | ↓ 持续下降 |
| **价格** | ↑ 开始上涨 |
| **企业行为** | 还没反应过来，产量尚未增加 |
| **对应美林时钟** | 复苏期 |

> **类比**：超市打折促销后，货架上的商品被快速买走，但仓库还没来得及补货。

### ② 主动补库存（过热期）

| 维度 | 表现 |
|------|------|
| **需求** | ↑↑ 持续旺盛 |
| **库存** | ↑ 快速增加 |
| **价格** | ↑↑ 加速上涨 |
| **企业行为** | 确认需求回暖，积极扩产、加速采购 |
| **对应美林时钟** | 过热期 |

> **类比**：超市发现某商品卖得火爆，大量进货补满货架，甚至多订了一些。

### ③ 被动补库存（滞胀期）

| 维度 | 表现 |
|------|------|
| **需求** | ↓ 开始下降 |
| **库存** | ↑ 继续增加 |
| **价格** | → 高位震荡或开始回落 |
| **企业行为** | 还没反应过来，产量尚未减少，库存被动堆积 |
| **对应美林时钟** | 滞胀期 |

> **类比**：超市订了很多货但顾客突然不买了，仓库越堆越满。

### ④ 主动去库存（衰退期）

| 维度 | 表现 |
|------|------|
| **需求** | ↓↓ 持续萎缩 |
| **库存** | ↓ 主动减少 |
| **价格** | ↓↓ 加速下跌 |
| **企业行为** | 确认需求下行，减产、降价促销清库存 |
| **对应美林时钟** | 衰退期 |

> **类比**：超市开始大甩卖，减少进货，想办法把积压的库存清掉。

---

## 🔄 完整循环图

```
        价格↑                价格→               价格↓
        ╱    ╲              ╱    ╲              ╱
   ①被动去库  ②主动补库   ③被动补库  ④主动去库   ①被动去库...
   需求>供给   需求=供给↑   供给>需求   需求<供给↓
   库存触底    库存回升      库存顶峰    库存下降
```

整个周期约 **40 个月（3-4 年）**，其中上行期（①②）约 20 个月，下行期（③④）约 20 个月。
