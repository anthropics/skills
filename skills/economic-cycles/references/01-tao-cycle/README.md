# 涛动周期（周金涛）

> 康波周期的中国实践——"人生发财靠康波"

## 📚 本章概述

周金涛（1972-2016），中信建投首席经济学家，被业界尊称为"周期天王"。他将康德拉季耶夫长波理论（康波周期）与中国经济实际相结合，创建了独特的"涛动周期"分析框架，并以此成功预测了多次重大经济转折点。

**核心命题**："人生发财靠康波"——个人财富的积累，不是你多大本领，而是来源于经济周期运动的时间给你的机会。在 60 年的经济周期中，每个人理论上有三次大的财富机会，抓住一次就能步入中产阶级。

### 学习目标

- 理解周金涛四周期嵌套模型：60 年康波 + 20 年地产周期 + 10 年设备周期 + 3-4 年库存周期
- 掌握第五轮康波的时间划分和各阶段特征
- 了解周金涛关于资产配置的核心建议
- 认识涛动周期理论的局限性

### 第五轮康波时间划分

| 阶段 | 时间 | 特征 |
|------|------|------|
| **回升期** | ~1982-1991 | 信息技术萌芽，个人电脑革命 |
| **繁荣期** | 1991-2004 | 互联网红利，全球化加速 |
| **衰退期** | 2004-2015 | 2008 次贷危机，增长放缓 |
| **萧条期** | 2015-2025 | 全球动荡，旧秩序解体 |
| **回升期** | 2025-2030+ | 新技术（AI？）驱动新一轮康波 |
