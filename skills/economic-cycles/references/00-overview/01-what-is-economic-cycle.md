# 什么是经济周期

> 经济活动在扩张与收缩之间的周期性波动，是理解宏观经济运行的基石。

## 📚 核心概念

### 定义

**经济周期**（Business Cycle / Economic Cycle），也称商业周期、商业循环、景气循环，是指经济运行中周期性出现的**经济扩张与经济紧缩交替更迭、循环往复**的一种现象。

> **生活类比**：经济周期就像四季轮回——春天万物复苏（复苏期），夏天热火朝天（繁荣期），秋天开始降温（衰退期），冬天万物凋零（萧条期），然后再次迎来春天。

### 衡量标准

经济周期的基本衡量指标是**国民总产出（GDP）、总收入和总就业**的波动。美国国家经济研究局（NBER）负责官方认定美国经济周期的起止时间点。

---

## 🔄 四个阶段

经济周期通常被划分为四个阶段，每个阶段有不同的经济特征和投资含义：

### 1. 繁荣期（Prosperity / Expansion）

| 特征维度 | 表现 |
|----------|------|
| **产出** | GDP 高速增长，产出持续扩张 |
| **就业** | 失业率低，劳动力市场紧张 |
| **物价** | 通胀加速上升，需求旺盛 |
| **投资** | 企业积极扩产，信用扩张 |
| **信心** | 公众对未来乐观，消费意愿强 |
| **最佳资产** | 大宗商品、股票 |

> ⚠️ 繁荣的种子中往往埋藏着衰退的因子——过度投资、产能过剩、资产泡沫

### 2. 衰退期（Recession / Contraction）

| 特征维度 | 表现 |
|----------|------|
| **产出** | GDP 增速放缓，从高峰开始下降 |
| **就业** | 失业率开始上升 |
| **物价** | 价格增速放缓，通胀回落 |
| **投资** | 企业缩减投资，供大于求 |
| **信心** | 消费者和企业信心减弱 |
| **最佳资产** | 债券 |

> 📌 技术定义：连续两个季度 GDP 负增长通常被视为衰退的判定标准

### 3. 萧条期（Depression / Trough）

| 特征维度 | 表现 |
|----------|------|
| **产出** | GDP 处于低谷，经济活动低迷 |
| **就业** | 失业率高企，企业倒闭增加 |
| **物价** | 通缩风险，资产缩水 |
| **投资** | 投资极度萎缩 |
| **信心** | 经济前景迷茫，社会需求不足 |
| **最佳资产** | 现金 |

> 💡 萧条期是"播种期"——优质资产被低估，为下一轮复苏创造条件

### 4. 复苏期（Recovery / Revival）

| 特征维度 | 表现 |
|----------|------|
| **产出** | GDP 开始回升，经济逐步恢复 |
| **就业** | 就业机会增加，失业率下降 |
| **物价** | 通胀水平得到控制 |
| **投资** | 政策刺激见效，投资回暖 |
| **信心** | 信心回升，消费恢复 |
| **最佳资产** | 股票（对经济弹性最大） |

## 📜 历史认知演变

经济周期理论的发展经历了三个主要阶段：

### 第一阶段：经验观察期（19 世纪）

- **1862 年**：法国经济学家**克莱芒·朱格拉**（Clément Juglar）出版《法国、英国和美国的商业危机及其周期》，首次系统描述了 7-11 年的经济波动规律
- **1860s-1900s**：各国经济学家开始收集价格、产出、就业数据，试图发现规律性

### 第二阶段：理论构建期（20 世纪上半叶）

- **1913 年**：美国经济学家**韦斯利·密契尔**（Wesley Mitchell）发表《经济循环》，提出扩张→危机→萧条→复苏的四阶段模型
- **1923 年**：**约瑟夫·基钦**（Joseph Kitchin）发现 40 个月左右的短周期（库存周期）
- **1925 年**：苏联经济学家**尼古拉·康德拉季耶夫**（Nikolai Kondratieff）提出 50-60 年的长波理论
- **1930 年**：**西蒙·库兹涅茨**（Simon Kuznets）发现 15-25 年的建筑周期
- **1936 年**：**约瑟夫·熊彼特**（Joseph Schumpeter）提出三周期嵌套模型，将基钦、朱格拉、康德拉季耶夫三种周期统一在"创新"框架下

### 第三阶段：现代发展期（20 世纪下半叶至今）

- **1950s-1960s**：凯恩斯主义主导，强调政府干预可以"熨平"经济周期
- **1970s**：滞胀（高通胀+高失业）打破了凯恩斯主义的乐观预期
- **1980s**：**真实商业周期理论**（RBC Theory）兴起，Kydland 和 Prescott 认为技术冲击是周期的主要来源（获 2004 年诺贝尔经济学奖）
- **1986/1992 年**：**海曼·明斯基**提出金融不稳定假说，认为金融体系具有内生的周期性不稳定性
- **2008 年**：全球金融危机使明斯基理论重获关注，"明斯基时刻"成为金融分析的标准术语
- **2010s**：周金涛将康波理论与中国经济实践结合，提出"涛动周期"

## 🎯 经济周期的实践意义

### 对投资者

- **美林投资时钟**：基于经济周期四阶段配置不同资产（债券→股票→商品→现金）
- **周期择时**：判断当前所处周期阶段，调整仓位和资产配置
- **逆向投资**：在萧条期买入被低估的优质资产

### 对企业管理者

- **产能规划**：避免在繁荣期过度扩产，在萧条期保留核心能力
- **库存管理**：跟踪基钦周期，优化库存水平
- **融资决策**：在信用宽松期储备资金，在紧缩期控制杠杆

### 对政策制定者

- **逆周期调控**：繁荣期紧缩、萧条期刺激
- **宏观审慎监管**：关注金融体系的系统性风险积累（明斯基视角）
