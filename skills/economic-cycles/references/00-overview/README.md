# 经济周期总论

> 理解经济周期的整体框架，建立对各周期理论的全景认知。

## 📚 本章概述

经济周期（Business Cycle / Economic Cycle）是宏观经济学中最古老也最核心的研究课题之一。从 19 世纪法国经济学家朱格拉首次系统描述经济波动的规律性，到 20 世纪熊彼特将多种周期理论整合为"三周期嵌套模型"，再到当代明斯基从金融体系内生不稳定性的角度重新审视经济波动——经济周期理论经历了两百年的演化，形成了一个多层次、多维度的理论体系。

**核心比喻**：经济周期 = 大海的波浪。表面是 3-5 年的小浪（基钦周期），下面是 7-11 年的中浪（朱格拉周期），再深处是 15-25 年的涌浪（库兹涅茨周期），最底层是 50-60 年的洋流（康德拉季耶夫长波）。它们同时存在、相互叠加，共同塑造了经济运行的起伏节奏。

### 学习目标

- 理解经济周期的定义、四个阶段及其特征
- 掌握主流经济周期理论的分类框架（按时间尺度、按驱动因素）
- 建立各周期理论之间的嵌套和关联关系认知
- 了解经济周期研究的历史演变与现代发展

### 前置知识

- 基本的宏观经济学概念（GDP、通胀、就业）
- 对投资和资产价格有基本认知

### 按驱动因素分类

| 驱动维度 | 代表理论 | 核心机制 |
|----------|----------|----------|
| **供给侧/技术** | 熊彼特、康德拉季耶夫 | 创新爆发→扩散→饱和→新创新 |
| **需求侧/投资** | 朱格拉、卡莱茨基、卡尔多 | 利润→投资→产能→过剩→出清 |
| **库存调整** | 基钦 | 补库存→去库存的短期往复 |
| **基础设施/人口** | 库兹涅茨 | 人口→城市化→建设→饱和 |
| **金融内生** | 明斯基 | 对冲→投机→庞氏→崩溃 |
| **中国实践** | 周金涛（涛动周期） | 康波+中国国运的映射 |

### 嵌套关系（熊彼特三周期模型）

熊彼特（1936）提出了经典的多周期嵌套模型：

- **1 个康德拉季耶夫长波 ≈ 3 个库兹涅茨周期 ≈ 6 个朱格拉周期 ≈ 18 个基钦周期**
- 每一层周期都嵌套在更长的周期之中
- 当多个周期的下行期同时叠加（"周期共振"），往往引发严重的经济危机
