# 固定投资周期机制

> 设备投资的"繁荣→危机→清算"三阶段循环

## 📚 核心机制

### 设备更新周期

朱格拉周期的微观基础是**设备的物理寿命和技术更新周期**：

- 美国经济分析局（BEA）统计：工程机械平均寿命约 **8-10 年**
- 整体机械设备折旧率约 **13%**，对应平均更换周期 **7-8 年**
- 工程机械更换周期更长，一般在 **8-10 年**

### 三阶段模型

朱格拉周期通常经历**繁荣（Prosperity）→ 危机（Crisis）→ 清算（Liquidation）** 三个阶段：

```
繁荣期                  危机点                 清算期
  │                       │                      │
  │  需求↑ → 投资↑        │  产能过剩             │  投资↓↓
  │  信贷扩张             │  利润下降             │  去产能
  │  设备大量采购          │  信贷收紧             │  企业倒闭/并购
  │  就业增加             │  资产价格下跌          │  等待下一轮设备老化
  │                       │                      │
  ▼─────────────→ ▼──────────────→ ▼─────────→ 新一轮繁荣
      (~4-5年)         (~1-2年)        (~3-4年)
```

### 投资-利润反馈循环

```
经济复苏 → 企业盈利改善
    ↓
利润预期上升 → 增加资本开支（设备投资）
    ↓
固定资产投资拉动 GDP → 就业增加 → 消费增长
    ↓
需求持续旺盛 → 更多企业跟进投资
    ↓
产能逐渐过剩 → 供大于求
    ↓
产品价格下跌 → 利润率下降
    ↓
企业削减投资 → 信贷紧缩 → 部分企业破产
    ↓
产能出清 → 幸存企业市占率提升 → 等待下一轮复苏
```

## 💰 信贷与朱格拉周期

### 信贷扩张与收缩

信贷周期是朱格拉周期的重要放大器：

| 阶段 | 信贷状态 | 银行行为 | 企业行为 |
|------|----------|----------|----------|
| **复苏期** | 谨慎放贷 | 风险偏好回升 | 修复资产负债表 |
| **繁荣期** | 积极放贷 | 竞争性放贷 | 加杠杆扩产 |
| **危机期** | 信贷急冻 | 紧缩抽贷 | 被迫去杠杆 |
| **清算期** | 信贷低迷 | 坏账处置 | 破产重组 |

### 与明斯基假说的关系

朱格拉周期的信贷放大机制与明斯基的金融不稳定假说高度一致——在繁荣期，融资结构从对冲型逐步恶化为投机型甚至庞氏型，为危机埋下伏笔。
