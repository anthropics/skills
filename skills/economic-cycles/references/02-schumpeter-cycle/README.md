# 熊彼特经济周期

> 创造性破坏——技术创新驱动经济长波的内在机制

## 📚 本章概述

**约瑟夫·熊彼特**（Joseph Alois Schumpeter, 1883-1950），奥地利裔美国经济学家，是创新理论的鼻祖和经济周期理论的集大成者。他提出了"创造性破坏"（Creative Destruction）这一划时代的概念，并将康德拉季耶夫长波、朱格拉中波和基钦短波统一在"创新"这一驱动力下，构建了经典的三周期嵌套模型。

**核心命题**：经济发展不是平滑连续的过程，而是由企业家推动的"创新浪潮"间歇性地打破旧均衡、建立新均衡的"创造性破坏"过程。

### 学习目标

- 理解"创造性破坏"的核心含义和机制
- 掌握熊彼特三周期嵌套模型的结构和逻辑
- 了解五次技术革命浪潮与康波长波的对应关系
- 认识熊彼特理论对现代创新经济学的影响

## 🗺️ 熊彼特理论全景

### 创新的五种形式

熊彼特在《经济发展理论》（1912）中定义了五种创新：

1. **新产品**：引入消费者尚不了解的产品或产品的新质量
2. **新方法**：引入新的生产方法或技术
3. **新市场**：开辟新的销售市场
4. **新来源**：获得原材料或半成品的新供应来源
5. **新组织**：实现新的产业组织形式（如建立垄断地位或打破垄断）

### 三周期嵌套模型

```
1 个长波（康波，~54年）
├── 6 个中波（朱格拉，~9年）
│   ├── 3 个短波（基钦，~3年）
│   ├── 3 个短波
│   └── ...
├── 6 个中波
│   └── ...
└── ...

总计：1 康波 ≈ 6 朱格拉 ≈ 18 基钦
```
