# 库兹涅茨周期

> 15-25 年的建筑/房地产周期——人口结构与城市化驱动的中长周期

## 📚 本章概述

**西蒙·库兹涅茨**（Simon Kuznets, 1901-1985），美国经济学家（1971 年诺贝尔经济学奖），1930 年根据对美国建筑业大量数据的研究，发现经济中存在 15-25 年不等、平均约 20 年的长周期波动，其中建筑业表现尤为显著。

库兹涅茨周期的典型模式是**15 年上涨 + 5 年下跌**，核心驱动力是**人口变动、城市化进程和建筑/房地产投资**。

### 核心逻辑

```
人口增长/迁移 → 住房需求增加 → 房地产开发投资 → 建筑业繁荣
    → 城市扩张 → 基础设施建设 → 相关产业链繁荣
        → 人口增长放缓/城市化饱和 → 投资减少 → 下行期
```
