## 什么是经济周期

经济周期（Business Cycle / Economic Cycle）是指经济活动沿着长期增长趋势发生的周期性波动，通常表现为**繁荣→衰退→萧条→复苏**四个阶段的循环往复。

### 核心周期理论一览

| 周期名称 | 时长 | 驱动因素 | 代表人物 |
|----------|------|----------|----------|
| **基钦周期** | 3-5 年 | 库存波动 | Joseph Kitchin |
| **朱格拉周期** | 7-11 年 | 固定资本投资 | Clément Juglar |
| **库兹涅茨周期** | 15-25 年 | 建筑/房地产/人口 | Simon Kuznets |
| **康德拉季耶夫长波** | 50-60 年 | 技术革命 | Nikolai Kondratieff |
| **涛动周期** | 60 年 | 康波+中国实践 | 周金涛 |
| **熊彼特周期** | 多尺度嵌套 | 创新/创造性破坏 | Joseph Schumpeter |
| **明斯基假说** | 非固定 | 金融内生不稳定 | Hyman Minsky |

### 周期嵌套关系

```
┌──────────────────────────────────────────────────────────────────┐
│                    康德拉季耶夫长波（50-60年）                      │
│  ┌────────────────────────────────────────────────────────────┐  │
│  │              库兹涅茨周期（15-25年）                         │  │
│  │  ┌──────────────────────────────────────────────────────┐  │  │
│  │  │            朱格拉周期（7-11年）                        │  │  │
│  │  │  ┌────────────────────────────────────────────────┐  │  │  │
│  │  │  │          基钦周期（3-5年）                       │  │  │  │
│  │  │  │  繁荣 → 衰退 → 萧条 → 复苏 → 繁荣 → ...        │  │  │  │
│  │  │  └────────────────────────────────────────────────┘  │  │  │
│  │  └──────────────────────────────────────────────────────┘  │  │
│  └────────────────────────────────────────────────────────────┘  │
│                                                                  │
│  明斯基时刻 ⚡ ← 金融不稳定性在任何周期转折点都可能爆发            │
└──────────────────────────────────────────────────────────────────┘
```

## 经济周期全景图

经济周期理论描绘了经济活动在不同时间尺度上的波动模式。各周期之间存在**嵌套**关系——短周期嵌套在中周期之中，中周期又嵌套在长周期之内。当多个周期在同一时间点**共振**时（同时进入上行或下行阶段），会产生放大效应。

| 层级 | 周期 | 时长 | 对应资产/领域 |
|------|------|------|--------------|
| 长波 | 康波（涛动周期） | 50-60年 | 黄金（信用对冲）、技术革命 |
| 中长波 | 库兹涅茨周期 | 15-25年 | 房地产、城镇化 |
| 中波 | 朱格拉周期 | 7-11年 | 石油、铜、设备投资 |
| 短波 | 基钦周期 | 3-5年 | 库存、猪周期、工业金属 |
| 金融 | 明斯基假说 | 非固定 | 金融危机、杠杆周期 |

## Part I - 经济周期理论篇

### 0. 📚 经济周期总论

**为什么重要**：理解经济周期的整体框架是深入研究任何单一周期理论的前置基础。

📖 **详细文档**：[references/00-overview.md](references/00-overview.md)

---

### 1. 🌊 涛动周期（周金涛）

**为什么重要**：将康波理论与中国经济实践相结合，提出"人生发财靠康波"的投资框架。

📖 **详细文档**：[references/01-tao-cycle.md](references/01-tao-cycle.md)

---

### 2. 💡 熊彼特经济周期

**为什么重要**：创造性破坏理论揭示了技术创新驱动经济长期增长的内在机制。

📖 **详细文档**：[references/02-schumpeter-cycle.md](references/02-schumpeter-cycle.md)

---

### 3. 📈 康德拉季耶夫长波

**为什么重要**：50-60 年的超长周期是理解技术革命与经济范式转换的宏观框架。

📖 **详细文档**：[references/03-kondratieff-wave.md](references/03-kondratieff-wave.md)

---

### 4. 🏭 朱格拉周期

**为什么重要**：7-11 年的固定资本投资周期是宏观经济分析中最实用的中周期框架。

📖 **详细文档**：[references/04-juglar-cycle.md](references/04-juglar-cycle.md)

---

### 5. 📦 基钦周期

**为什么重要**：3-5 年的库存周期是最容易观测和预判的短周期，对行业研究和投资决策有直接参考价值。

📖 **详细文档**：[references/05-kitchin-cycle.md](references/05-kitchin-cycle.md)

---

### 6. 🏗️ 库兹涅茨周期

**为什么重要**：15-25 年的建筑/房地产周期与人口结构、城市化进程紧密相关，对理解房地产市场至关重要。

📖 **详细文档**：[references/06-kuznets-cycle.md](references/06-kuznets-cycle.md)

---

### 7. ⚡ 明斯基金融不稳定假说

**为什么重要**：揭示了金融体系从稳定走向崩溃的内生机制，"明斯基时刻"已成为金融危机分析的标准术语。

📖 **详细文档**：[references/07-minsky-moment.md](references/07-minsky-moment.md)

## Part II - 大宗商品与资产价格周期篇

### 8. 🥇 黄金与经济周期

**为什么重要**：黄金作为"终极避险资产"，其价格走势与康波、美元周期、通胀周期紧密关联。

📖 **详细文档**：[references/08-gold-cycle.md](references/08-gold-cycle.md)

---

### 9. 🛢️ 石油与经济周期

**为什么重要**：石油是"大宗商品之王"，其超级周期与全球经济周期深度耦合。

📖 **详细文档**：[references/09-oil-cycle.md](references/09-oil-cycle.md)

---

### 10. ⛏️ 煤炭与经济周期

**为什么重要**：煤炭产能周期与工业化进程、能源转型政策密切相关，尤其在中国具有独特特征。

📖 **详细文档**：[references/10-coal-cycle.md](references/10-coal-cycle.md)

---

### 11. 🔩 工业金属与经济周期

**为什么重要**：铜、铝、铁矿石等工业金属价格是经济周期的"温度计"，与基钦库存周期和朱格拉投资周期高度相关。

📖 **详细文档**：[references/11-industrial-metals.md](references/11-industrial-metals.md)

---

### 12. 🐷 猪周期

**为什么重要**：猪周期是中国最具特色的农产品周期，3-4 年的能繁母猪→生猪供给周期对 CPI 有显著影响。

📖 **详细文档**：[references/12-pig-cycle.md](references/12-pig-cycle.md)

---

### 13. 🏠 房价与经济周期

**为什么重要**：房价周期与库兹涅茨建筑周期、人口周期、信贷周期/明斯基时刻深度关联，是影响居民财富的核心变量。

📖 **详细文档**：[references/13-housing-cycle.md](references/13-housing-cycle.md)

## 学习路径指南

### 🛤️ 路径 A：宏观经济研究者

**建议路径**：

```
Step 1: 经济周期总论（建立全景认知）
    ↓
Step 2: 康波 + 熊彼特（理解长周期驱动力）
    ↓
Step 3: 朱格拉 + 基钦（掌握中短周期分析）
    ↓
Step 4: 库兹涅茨 + 明斯基（补充建筑周期和金融周期）
    ↓
Step 5: 涛动周期（中国视角的综合应用）
    ↓
Step 6: 大宗商品与资产价格周期（理论联系实际）
```

### 🛤️ 路径 B：投资分析师

**建议路径**：

```
Step 1: 经济周期总论 + 涛动周期（快速建立投资框架）
    ↓
Step 2: 基钦周期（库存周期择时）
    ↓
Step 3: 朱格拉周期（产能周期判断）
    ↓
Step 4: 黄金 + 石油 + 工业金属（大宗商品轮动）
    ↓
Step 5: 房价 + 猪周期（中国特色资产研究）
    ↓
Step 6: 明斯基（风险识别）
```
